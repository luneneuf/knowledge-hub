## 1-A. 한국어 소비자 언어

### 올리브영·네이버 리뷰 주요 표현

#### 호평 표현 (Positive)
| 표현 | 의미/맥락 |
|---|---|
| 광택감 / 윤기 | 고광택 마무리 — 리뷰에서 가장 빈도 높음 |
| 촉촉함 / 보습력 | 수분 공급 효과, 클린/케어 뷰티 트렌드와 연결 |
| 끈적임 없음 | 글로스 최대 불만 사항의 해소로 언급 |
| 발색 좋음 | 투명~컬러 글로스 모두에서 사용, 색상 재현도 |
| 부드러운 발림성 | 어플리케이터·텍스처 품질 표현 |
| 플럼핑 효과 / 볼륨감 | 입술 도톰 연출 기대값 |
| 가성비 | 특히 10,000원 이하 제품 리뷰에서 빈출 |
| 지속력 / 오래 가요 | 호평에서도 불만에서도 언급되는 이중적 키워드 |

#### 불만 표현 (Negative)
| 표현 | 구체 내용 |
|---|---|
| 끈적임 | 립글로스 카테고리 전체의 1위 불만 (해결 시 차별화 포인트) |
| 지속력이 약함 | 가장 빈번한 불만 — 음식 먹으면 즉시 제거 |
| 발색이 약함 | 여러 번 덧발라야 한다 |
| 머리카락 달라붙음 | 끈적임의 실생활 결과로 언급 |
| 강한 향 | 향수 성분이 부담된다 |
| 화끈거림 | 플럼퍼 성분(생강·캡사이신 계열) 사용 제품에서 발생 |
| 색상 빨리 지워짐 | 식사 후 유지력 없음 |
