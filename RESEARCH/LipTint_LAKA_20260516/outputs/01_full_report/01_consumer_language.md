## 1.1 한국어 소비자 언어

### 호평 표현 패턴

올리브영 및 네이버 리뷰에서 반복적으로 등장하는 긍정 키워드:

| 키워드 | 의미 | 사용 맥락 예시 |
|--------|------|--------------|
| **발색** | 색이 선명하게 표현되는 정도 | "발색이 강하고 선명해요", "발색이 맑고 깨끗해요" |
| **지속력** | 색의 지속 시간 | "지속력이 장시간 유지돼요", "식사 후에도 지속력 유지" |
| **착색** | 피부에 색이 물드는 정도 | "착 붙는 텍스쳐", "자연스럽게 물든 듯한 착색" |
| **수분감** | 입술의 촉촉함 | "수분감이 살아있어 건조하지 않음" |
| **밀착** | 제품이 입술에 붙는 느낌 | "밀착력이 좋아 번짐 없음" |
| **번짐 없음** | 색이 번지지 않는 것 | "마스크에 거의 묻어나지 않는 지속력" |
| **물든듯한** | 자연스러운 착색 연출 | "자연스럽게 물든 듯한" |

### 불만 표현 패턴

| 키워드 | 의미 | 사용 맥락 예시 |
|--------|------|--------------|
| **건조함** | 입술이 건조해지는 문제 | "건조할 수 있어 립밤과 함께 사용 추천" |
| **각질** | 각질이 일어나는 문제 | "각질이 많거나 주름 있을 경우 도드라짐" |
| **뭉침** | 제품이 고르게 발리지 않는 문제 | "입술 끝에 뭉침 발생" |
| **색상 차이** | 발색이 화면과 다른 문제 | "실제 색이 광고와 달라요" |
| **밀림** | 제형이 입술 위에서 밀리는 현상 | "시간 지나면 밀려요" |

## 1.2 영어 소비자 언어

### TikTok / Instagram 핵심 어휘

| 영어 표현 | 의미 | 사용 맥락 |
|----------|------|----------|
| **staining** | 착색 효과 | "it stains the lips for hours" — 지속력 강조 |
| **no-transfer** | 번짐 없음 | "transfer-proof", "no-transfer formula" — 마스크 문제 해결 소구 |
| **gradient lip** | 그라데이션 립 | Korean beauty technique, center-heavy application |
| **Douyin lip** | 중국 TikTok 스타일 | "bitten lips effect", center-saturated, edges soft |
| **bitten lip** | 물어뜯은 듯한 입술 | Douyin 영향의 서양 표현 |
| **MLBB (My Lips But Better)** | 자연스러운 입술 색 보정 | 누드·로즈 계열 소구에 사용 |
| **buildable** | 레이어링 가능 | 하나로 연하게/진하게 모두 가능 |
| **non-sticky** | 끈적임 없음 | K-beauty 공통 소구점 |

### 飲み会リップ vs. 한국 표현 비교

출처: @cosme 飲み会リップ 특집 (2025 겨울 에디션)

**飲み会リップ (노미카이 립)** — "술자리 립"의 의미:
- 정의: 식사·음주 중에도 색이 유지되는 립 제품
- 평가 기준 3가지: ① 컵 착색 방지 ② 식사 중 발색 유지 ③ 색의 선명도
- @cosme 편집부가 ★5 기준 3개 차원으로 테스트
- 대상 제품: RMK, CipiCipi, TIRTIR, BBIA, Laka 등

**한국 vs. 일본 표현 구조 차이:**

| 구분 | 한국 | 일본 |
|------|------|------|
| 지속력 표현 | "먹어도 안 지워져요", "마스크에 안 묻어요" | "食べてもにじまない", "飲み会リップ" |
| 문화적 맥락 | 마스크 착용 문화 기반 소구 (팬데믹 이후 지속) | 술자리(居酒屋) 문화 기반 소구 |
| 그라데이션 | "물든듯한", "그라데이션 립" | "グラデーションリップ" — 동일 기술 다른 명칭 |
| 검색 행동 | "립틴트 추천", "올리브영 틴트" | "ティント ランキング", "リップティント 落ちない" |
