## 1. 정의·제형 분류

### 1.1 립글로스란?

립글로스는 입술에 **광택(shine)**과 촉촉함을 부여하도록 설계된 액체~반고체형 립 메이크업. 핵심 구분 기준은 고광택 효과와 점도(viscosity)에 있다.

| 제품 유형 | 발색 | 광택 | 지속력 | 텍스처 |
|----------|------|------|--------|--------|
| 립글로스 | 얇음~중간 | **높음** (젖은 느낌) | 2–4시간 | 두껍고 끈적할 수 있음 |
| 립틴트 | 얇음~중간 | 없음~약간 | 6–12시간 | 가벼운 액체/젤 |
| 립오일 | 얇음 (쉬어) | 중간 (윤기) | 2–3시간 | 매우 가벼움, 비끈적 |
| 립스틱 | 높음 | 없음(매트)~중간 | 2–4시간 | 고체 크리미 |

- **립글로스 vs. 립오일**: 립오일은 피부에 흡수, 비끈적; 립글로스는 피막 형성 후 표면 잔류. 최근 경계가 희미해지는 "립 오일 글로스" 카테고리 등장
- **글로시 틴트 (K-뷰티 특화)**: 틴트의 지속력 + 글로스의 광택 → K-뷰티가 창출한 하이브리드 서브카테고리

### 1.2 서브 포맷 분류

| 서브포맷 | 특징 | 대표 제품 |
|---------|------|---------|
| **클리어 글로스** | 무색소, 광택만. 단독 또는 다른 립메이크업 위에 덧바름 | MAC Lipglass Clear |
| **틴티드 글로스** | 쉬어 컬러 + 광택. 가장 주류 | Fenty Gloss Bomb, 롬앤 글래스팅 컬러 글로스 |
| **글리터/심머 글로스** | 마이크로 글리터·펄 입자 함유; 홀리데이 수요 급증 | NYX Butter Gloss (shimmer) |
| **홀로그래픽 글로스** | 각도에 따라 색상 전환되는 홀로그래픽 피그먼트 | WONDERSKIN Top Gloss |
| **립 오일 글로스** | 오일 베이스 + 글로스 효과; 립오일과 경계 모호 | NYX Fat Oil Lip Drip, Dior Lip Glow Oil |
| **플럼핑 글로스** | 시나몬오일·캡사이신·멘틸락테이트로 혈류 촉진 → 볼륨 효과 | Too Faced Lip Injection, Dior Lip Maximizer |
| **글로시 틴트 (K-뷰티)** | 틴트 착색력 + 글로스 광택 결합 | hince Raw Glow Gel Tint, 롬앤 글래스팅 워터 글로스 |

### 1.3 핵심 성분

| 성분 | 역할 |
|------|------|
| **폴리부텐 (Polybutene)** | 강한 유리막 광택 제공, 필름 형성; 끈적임의 주원인 |
| **피마자유 (Castor Oil)** | 두꺼운 오일로 광택+지속력; 모이스처라이징 |
| **실리콘 (Silicones)** | 미끄러운 발림성, 비끈적 질감 |
| **시나몬오일/캡사이신** | 혈류 촉진 → 입술 볼륨 (플럼핑 글로스) |
| **히알루론산/펩타이드** | 수분공급; 최근 K-뷰티 글로스 스킨케어 성분으로 적극 채용 |
| **글리터 파티클** | 폴리에스터·마이카 기반 (EU 마이크로플라스틱 규제 대상) |

**텍스처 스펙트럼**: 씬·워터리(립 오일 글로스) → 미디엄(틴티드 글로스) → 씩·스티키(클래식 폴리부텐 글로스)

## 2. 역사

### 2.1 발명 (1930년대)
- **1930년**: Max Factor가 헐리우드 배우들의 흑백 필름 촬영을 위해 입술을 더욱 광택 있게 보이도록 최초 립글로스 개발
- **1932년**: 최초 상업용 립글로스 "X-Rated" 일반 소비자 출시. 1945년까지 연 매출 약 $230만 (전체 코스메틱의 18%)

### 2.2 1960–80년대: 디스코 시대 전성기
- 1970년대: 펄·심머가 디스코 문화와 맞물려 아이코닉 글로시 립 정착
- 1980년대: Bonne Bell 플레이버드 글로스가 대중화 → 10대 필수품

### 2.3 1990–2000년대: 글로시 립 절정기
- 2000년대: Y2K 미학 + 크리스티나 아길레라·제니퍼 로페스 등 팝스타들의 상징적 글로시 립

### 2.4 2010년대: 매트 트렌드와 하락기
- 카일리 제너로 대표되는 **매트 립스틱** 붐이 글로스 수요 대체. 셀피 문화에서 매트 마감 선호

### 2.5 2020년대: "글래스 립" 부활과 TikTok 르네상스

**2022–2024 부활 배경:**
- 마스크 해제 이후 억압된 립 메이크업 욕구 폭발
- TikTok #lipgloss 관련 콘텐츠 **2억 900만 개+**, 해시태그 **13억 뷰** (2024)
- "립글로스 택틱": 크리에이터들이 영상 초반에 립글로스 바르는 장면으로 시청자 참여율 후킹
- "Korean lip gloss" 검색지수 2024년 6월 52 → 2025년 1월 78 급상승

**K-뷰티 트렌드 수출:**

| 트렌드 | 설명 | 확산 시점 |
|--------|------|---------|
| **탕후루 립** | 중국 전통 간식처럼 투명하고 반짝이며 탱글탱글한 설탕 코팅 느낌 | 2023–2024 |
| **물광 글로스** | 스킨케어 결합, 촉촉하고 윤기있는 "물 먹은 입술" | 2023– |
| **글래스 립** | 유리처럼 투명하고 반짝이는 입술 | 2024–2025 1위권 |
| **거울광/디스코 립** | 더욱 강한 메탈릭 광택 | 2025→2026 전환 중 |

## 5. 소비자 언어 분석 (한·영·일)

### 5.1 한국어

**호평 표현**

| 키워드 | 설명 |
|--------|------|
| 끈적임 없음 | 가장 강력한 차별화 포인트 — 동시에 광택 유지되면 프리미엄 |
| 자연스러운 광택감 | 인위적이지 않은 촉촉한 윤기 |
| 수분감 | 촉촉함, 건조하지 않음 |
| 발색 | 틴티드 글로스에서 자연스러운 색 소구 |
| 볼륨감 | 플럼핑 효과 |
| 물먹은 듯한 | 물광 글로스 트렌드 핵심 표현 |
| 탕탕하다 / 탱탱하다 | 탕후루 립 소구 표현 |

**불만 표현**

| 키워드 | 설명 |
|--------|------|
| 끈적임 | 가장 빈번한 불만; 머리카락 달라붙음 언급 동반 |
| 지속력 약함 | 2시간 내 사라짐 |
| 색 빨리 지워짐 | 틴티드 글로스에서 주요 불만 |

**광택 강도 트렌드 어휘 (2023→2026 진화):**
탕후루립 → 물광 → 유리광 → **거울광/디스코 립** (2026 최신 트렌드)

**OliveYoung Sweet Spot 가격대:** KRW 10,000–15,000 (롬앤 글래스팅 워터 글로스 KRW 10,900이 1위)

### 5.2 영어 (TikTok·미국)

**핵심 어휘**

| 표현 | 의미 |
|------|------|
| Glass lip | 유리처럼 투명하고 반짝이는 입술 |
| Disco lips | 더 강한 메탈릭 광택 (2025→2026 전환 트렌드) |
| Mirror shine | 거울광 |
| Juicy lips | 즙이 많은 듯한 촉촉하고 반짝이는 느낌 |
| Sticky gloss comeback | 클래식 끈적임 글로스의 Y2K 복귀 |
| Lip combo | 라이너+글로스 레이어링; 검색 YoY +97.5% |
| Non-sticky | 끈적임 없는 글로스가 가장 큰 소구 포인트 |

**"Lip Combo" 트렌드 영향:** 립 라이너 카테고리 2022→2024 **+60% 성장**. 글로스가 레이어링 필수 구성품으로 위상 격상

**Amazon/Sephora 긍정 패턴:**
- "non-sticky formula", "lasts all day", "plumping effect"
- "buildable color", "moisturizing yet glossy", "Korean formula"

### 5.3 일본어

**핵심 어휘**

| 일본어 | 의미 |
|--------|------|
| ツヤ感 (つやかん) | 광택감 (최빈 키워드) |
| ぷるぷる / うるうる | 탱탱하고 촉촉한 느낌 |
| ガラスリップ | 글래스 립 |
| 透明感 (とうめいかん) | 투명감, 투명도 |
| うるツヤ | 촉촉하고 광택있는 |
| ミラーリップ | 미러 립 |

**립글로스 vs. 립틴트 MOQ:** 대형 ODM 기준 5,000pcs 동일. 립글로스는 구조 단순해 단가 낮음.

### 7.2 핵심 성분 규제

**폴리부텐 (Polybutene):**
- EU 마이크로플라스틱 규정(EU 2023/2055): 액상 사용이므로 **직접 적용 가능성 낮음** — 단, 법적 확인 권고
- 미국 FDA, 한국 MFDS: 현행 허용

**글리터 (Synthetic Glitter) — EU 규제 핵심:**

| 규제 내용 | 기한 |
|---------|------|
| 합성 글리터 포함 립글로스 — **라벨링 의무** | **2031년 10월** |
| 합성 글리터 포함 립글로스 — **판매 금지** | **2035년 10월** |

→ 바이오글리터(셀룰로오스 기반) 또는 마이카 기반으로 전환 준비 필요

**색소:**
- D&C Red 6, D&C Red 7: FDA 허용 (배치 인증 필수, 눈 주변 금지)
- EU Annex IV 양성 리스트 153개 색소만 허용 (CI 번호로 교차 확인 필수)

### 7.3 향료 알레르겐 라벨링 — 가장 급박한 규제 리스크

**EU Regulation 2023/1545:**
- 80종+ 향료 알레르겐 개별 표기 의무 (리브온 제품 0.001% 초과 시)
- **기한: 2026년 7월 31일** ← 현재 가장 임박한 규제 데드라인

### 7.4 일본
- 2025년 GMP 강화·라벨링 명확화 개정 진행 중
- 글리터 별도 규제 없음 (현행)
- 고시 66호 색소 양성 리스트 — D&C Red 6/7 해당 번호 확인 필수

### 7.5 규제 준수 체크리스트

1. **EU 향료 알레르겐 라벨 업데이트** — **2026년 7월 31일 기한** (최우선)
2. **EU 합성 글리터** — 2031년 라벨링 의무화 대비, 바이오글리터 전환 준비 시작
3. **미국 FDA 배치 인증** — D&C Red 6/7 등 사용 색소 배치 인증 현황 확인
4. **일본 색소 대조** — 고시 66호 대조; 라벨링 일본어 전성분 표기
5. **폴리부텐** — EU 최신 마이크로플라스틱 가이던스 업데이트 모니터링
