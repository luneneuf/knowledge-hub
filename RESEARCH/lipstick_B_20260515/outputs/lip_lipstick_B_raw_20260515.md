## 4. 소비자 언어 분석 (한·영·일)

### 4.1 한국어

**호평 표현**

| 한국어 | 영어 |
|---|---|
| 발색력이 좋다 | Good color payoff |
| 지속력이 뛰어나다 | Excellent staying power |
| 촉촉하다 | Moisturizing, hydrating |
| 발림성이 좋다 | Smooth application |
| 착색이 잘 된다 | Good staining / adhesion |
| 가볍다 | Lightweight feel |
| 탱탱한 볼륨 | Plump, bouncy volume |
| 고발색 | High pigmentation |
| 맑고 깨끗한 컬러감 | Clear, clean color tone |

**불만 표현**

| 한국어 | 영어 |
|---|---|
| 밀린다 | Pilling / product slides |
| 건조하다 | Drying |
| 뭉친다 | Clumping, uneven |
| 끈적인다 | Sticky |
| 발색이 약하다 | Weak color payoff |
| 오래 못 간다 | Doesn't last long |

**구매 결정 요인 우선순위**
1. 발색력 (color payoff)
2. 지속력 (longevity)

## 7. 주요 규제 요건

### 7.1 미국 (FDA)

**분류**
- 립스틱 = **화장품(Cosmetic)** (FD&C Act)
- SPF·플럼핑 클레임 시 의약외품(Drug) 분류 가능 → 추가 규제 적용

**색소 규정**
- 색소 첨가제는 FDA 21 CFR Parts 74·82 허가 목록 관리
- D&C Red 계열 등 **배치 인증(Batch Certification)** 필수
- 산화철·이산화티타늄은 배치 인증 면제이나 허가 목록 내 사용 요건 준수 필요
- Source: [FDA — Color Additives Permitted for Use in Cosmetics](https://www.fda.gov/cosmetics/cosmetic-ingredient-names/color-additives-permitted-use-cosmetics)

**납(Lead) 함량**
- FDA 권장 최대치: 립 제품 기준 **10 ppm** (mg/kg)
- 시판 립 제품 99%+가 10 ppm 이하
- Source: [FDA — Lead in Cosmetics](https://www.fda.gov/cosmetics/potential-contaminants-cosmetics/lead-cosmetics)

**라벨링 (2025 기준)**
- 성분 목록 (INCI명, 함량 多→少 순)
- 순중량, 제조사·유통사 주소
- 인증 색소: 색소명·사용 제한 문구·배치 인증 번호

### 7.2 유럽 (EU Regulation EC 1223/2009)

**핵심 프레임워크**
- Annex II (금지 물질): 2025년 5월 기준 **1,703종** 금지
- Responsible Person (RP): EU 내 주소 보유 RP가 규정 준수 책임
- CPNP (Cosmetic Products Notification Portal): 시판 전 통보 의무

**2025–2026 주요 규제 변경**

| 규제 | 내용 | 발효일 |
|---|---|---|
| EU 2025/877 (CMR 물질 21종 추가) | 발암·변이원성·생식독성 물질 추가 금지 [(CIRS Group)](https://www.cirs-group.com/en/cosmetics/eu-amends-cosmetics-regulation-21-cmr-substances-added-to-the-list-of-prohibited-substances) | 2025년 9월 1일 |
| Alpha-arbutin / arbutin | 철수 의무 | 2025년 11월 1일 |
| 나노소재 금속 (colloidal silver/gold/copper/platinum nano) | 전 화장품 금지 | 2024년 완료 |
| PFAS (REACH 규정) | 단계적 폐지 추진 | 진행 중 |
| 향료 알레르겐 라벨링 (2023/1545) | 80종+ 알레르겐 라벨 의무 공개 | 2023년~ |

**립 제품 특이사항:** 비의도적 섭취(non-intended ingestion) 가능성이 노출량 계산에 반영 → 다른 제품보다 엄격 기준 적용
- 납 <10 mg/kg, 비소 <2 mg/kg, 카드뮴 <5 mg/kg

### 7.3 일본 (MHLW / 薬機法)

**분류 체계**
- 화장품(化粧品) vs. **의약부외품(医薬部外品)** 이원 체계
- 일반 립스틱 = 화장품; 미백 효능·특정 자외선 차단 기능 포함 시 = 의약부외품 → 사전 허가 필요

**수입 요건**
- 의약부외품 외국 제조사: **MHLW 인정 외국 제조업체(Accredited Foreign Manufacturer)** 등록 필수
- 의약부외품 전 제품: **시판 전 MHLW 승인** + 일본 내 **MAH(Marketing Authorization Holder)** 지정
- 일반 화장품: 제조판매업 허가 보유 일본 현지 MAH를 통한 신고

**라벨링**
- **일본어 라벨 의무** (전 법정 항목)
- 배치 번호·배치 코드 표시 필수

Source: [ChemLinked](https://cosmetic.chemlinked.com/cosmepedia/japan-cosmetic-regulation); [REACH24H](https://en.reach24h.com/service/cosmetic/japan-quasi-drugs-and-cosmetics-compliance)

### 7.4 한국 (MFDS / 식품의약품안전처)

- 화장품법 적용. 일반 립스틱 = 일반 화장품 신고
- 색소: 화장품 안전기준 허가 색소 목록 내 사용 원칙
- 수입 화장품: 국내 화장품 책임판매업자(RAH)를 통한 신고 필요
- **중금속 기준:** 납 20 μg/g 이하, 비소 10 μg/g 이하, 수은 1 μg/g 이하

Source: [식품의약품안전처 — 화장품 안전기준 등에 관한 규정](https://www.mfds.go.kr)

1. **보습이 발색을 대체해 글로벌 #1 기본 기대치로 부상.** 4개 시장 전체에서 보습은 이제 필수 기준.
2. **한국 → 일본 → 글로벌 포맷 파이프라인.** 립 틴트·블러드 립은 한국에서 시작 → 일본(粘膜 어댑테이션) → TikTok 글로벌("Douyin lip") 순으로 확산.
3. **EU에서 "클린뷰티" = 규제 준수.** 마케팅 언어가 아닌 법적 기준 — PFAS-free, 알레르겐 공개, 리필 가능이 실질적 구매 결정 용어.
4. **TikTok "립 콤보"가 미국 소비 구조를 재편.** 멀티 제품 루틴이 인접 카테고리(라이너, 오일) 28–60% YoY 성장 견인.

### 8.2 유통 채널
5. **올리브영은 경쟁자 없는 한국 게이트키퍼.** @cosme은 일본에서 유사한 신뢰도 역할이나 소매 독점력은 없음.
6. **미국은 Amazon이 가장 빠른 구조적 변화.** 2021~2030년 뷰티 점유율 10% → 15% 예상; Ulta의 프레스티지+매스 하이브리드가 가장 방어적 포맷.
7. **2025 H1 반전:** 매스 뷰티가 프레스티지를 처음으로 성장률에서 추월 — 브랜드 포지셔닝 전략 재고 시점.

### 8.3 ODM·제조
8. **한국콜마의 리드타임 단축(9–12개월 → 3–6개월)이 인디 브랜드 시장 진입 장벽을 낮춤.** 단 MOQ(5,000개)는 여전히 초기 단계 진입자에게 부담.
9. **ODM 계약 시 IP 관리 조항 필수.** 2024년 한국콜마 vs. Intercos Korea 판결이 선례.

### 8.4 규제 리스크
10. **EU 2025–2026 규제 파고가 가장 높음.** CMR 물질 21종 추가(2025/9), Alpha-arbutin 철수(2025/11), PFAS 단계적 폐지(진행 중) — 포뮬러 재검토 긴급.
11. **일본 의약부외품 분류 리스크 사전 차단 필요.** 미백·UV 클레임 포함 시 MAH 지정 + 사전 승인 필수; 클레임 범위 설정이 진입 속도를 결정.
