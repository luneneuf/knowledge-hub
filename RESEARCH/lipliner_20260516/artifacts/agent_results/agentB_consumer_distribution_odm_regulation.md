## 3. ODM·규제 이슈

### 3.1 ODM (한국 중심)

#### (1) 주요 ODM사 라이너 생산 현황

| 회사 | 색조 라이너 역량 | 비고 |
|---|---|---|
| **코스맥스** | 쿠션·립(틴트/립스틱/**라이너**)·아이섀도 색조 핵심 카테고리 | 연 8,000+ 신제품 개발, AI 스마트 조색 시스템 도입, CGMP·ISO 22716/9001/14001 등 全 국제인증 보유 |
| **한국콜마** | 기초 강점 / 색조 확장 중 | "초개인화" 전략, 색조에서 코스맥스 추격 |
| **코스메카코리아** | OGM (Original Global Manufacturer) 프로세스 | 색조 컬러 매칭 강점, 글로벌 인디 브랜드 다수 수주 |

#### (2) ODM 단가 / MOQ / 리드타임

| 항목 | 내용 (코스맥스 기준) |
|---|---|
| **MOQ** | 종전 5,000–10,000개 → 마이크로 브랜드용 **1,000개**까지 인하, 궁극 목표 **"MOQ 1개"** 비전 |
| **신제품 개발 리드타임** | 일반 색조: **1–2개월** (포뮬러 컨펌 후) / AI 조색 활용 시 단축 |
| **소비자 반응 테스트 모델** | "10가지 제품 × 1,000개" 다품종 소량 테스트 → 빠른 피드백 |
| **단가 (참고)** | 펜슬형 라이너 ODM 단가는 공개 자료 부재. 업계 통념상 펜슬: $0.50–$1.50/EA, 리퀴드 라이너 (애플리케이터+튜브): $1.00–$2.50/EA (MOQ·셰이드 수에 따라 변동, **검증 출처 없음 — 추정치 표시**) |

#### (3) 펜슬 vs 리퀴드 ODM 차이

| 구분 | 펜슬 (트위스트업/샤프닝) | 리퀴드/브러쉬 라이너 |
|---|---|---|
| **포뮬러 베이스** | 왁스·오일 기반 (카르나우바·칸데릴라 왁스, 캐스터 오일) | 필름 형성제 (PVP, 아크릴레이트), 분산제 |
| **충진 공정** | 몰드 캐스팅 → 사후 절삭·트위스트 케이스 조립 | 액상 필링 (튜브) + 브러쉬 어플리케이터 |
| **개발 난이도** | 발색·심 경도 밸런스 | 점도·플렉서블 필름·건조 속도 |
| **MOQ** | 펜슬 케이스 금형 의존 → 케이스 변경 시 MOQ 상승 | 리테일 튜브 표준화로 MOQ 유연 |
| **트렌드** | 누디 펜슬, 듀얼 (한쪽 펜슬+한쪽 브러쉬) | 페리페라 잉크 씬씬 류 브러쉬 라이너 → 정밀 오버립 대응 |

### 3.2 규제 (Regulation)

#### (1) 미국 FDA / MoCRA

- **색소 기준**: 21 CFR Parts 73, 74, 82 — 립 제품(라이너 포함)에 사용 가능한 컬러 애디티브 목록 한정.
- **FD&C Lakes**: 수불용성 처리로 입술 외부 사용 시 색번짐 방지. 21 CFR §74.1101, 74.1705, 74.2101, 74.2705에 영구 등재.
- **배치 인증(Batch Certification)**: 일부 컬러 (FD&C Red No.40 Lake 등)는 FDA 자체 랩 배치 인증 필수.
- **MoCRA (2022 제정, 2024–2026 단계 시행)**:
  - 제조시설 등록 (Facility Registration) 의무
  - 제품 리스팅 (Product Listing) 의무
  - 부작용 보고 (Adverse Event Reporting) 의무
  - 안전성 입증 자료 (Safety Substantiation) 보유

#### (2) EU CPNP — 향료 알레르겐 라벨링 (2026.7.31 시행)

| 항목 | 내용 |
|---|---|
| **근거 규정** | Regulation (EU) 2023/1545 — Annex III 개정 |
| **신규 알레르겐** | 기존 26종 → **80+종**으로 확대 (56종 추가) |
| **표시 농도 임계치** | 리브온 제품 (립라이너 포함) **0.001% 초과**, 린스오프 **0.01% 초과** 시 라벨 의무 표시 |
| **시행일** | **2026년 7월 31일**부터 EU 시장 출시 금지 (신규 출시) |
| **소진 유예** | 2028년 7월 31일까지 시장 유통(소진) 허용 |
| **실무 영향** | CPNP 노티피케이션 업데이트, 원료사 IFRA 인증서·알레르겐 신고서 재확보 필요 |

#### (3) 한국 식약처(MFDS)

- **근거**: 「화장품 안전기준 등에 관한 규정」, 「화장품의 색소 종류 및 기준」 (별표 1·2).
- **원칙**: **사용기준이 지정·고시된 색소 외 사용 불가** (positive list). 립 제품 점막 접촉 고려, 외용에만 허용되는 일부 색소는 립 제품 금지.
- **별표 1**: 색소 종류·사용부위(전신/외용/점막제외 등)·사용한도 명시.
- **표시**: 「화장품법」에 따라 전성분·색소 표시 필수.

#### (4) 중국 NMPA

- **동물실험 면제**: 2021.5.1부 일반 화장품(립라이너 포함) 동물실험 면제 시행. **단, 색소·향료·기능성 등 일부 조건 미충족 시 여전히 동물실험 필요**.
- **2026.3.31 NMPA 등록·신고 사항 개정안 (Draft for Comments, 의견수렴 2026.3.31–4.30)**: First Launches in China 장려, 동물실험 요건 추가 완화, 국내 책임자 변경 절차 간소화.
- **색소 차이 안전성 평가**: 동일 포뮬러 + 다른 색소(립라이너 셰이드 라인업) 등록 시 **각 색소별 추가 안전성 평가 자료 필요** — 셰이드 풀 ROI 산정 시 핵심 변수.

#### (5) 일본 PMDA / 약기법 (薬機法)

- **화장품 분류**: 일반 화장품은 의약품·의약부외품과 달리 사전 허가 불필요, **사후 책임 신고제**.
- **표시 의무**: 전성분 표시 (INCI 일본명), 제조판매업자명, 원산국.
- **색소**: 「化粧品基準」에 따라 ポジティブリスト(허용 목록) + 네거티브 리스트 혼합. 립 제품에 타르 색소 사용 시 별표 명시 색소만 허용.

## 4. 종합 시사점

1. **카테고리 재정의**: 립라이너는 더 이상 "윤곽 보조 도구"가 아니라 **"립 콤보 코어 아이템"**. 한·미·일 3개 시장 공통.
2. **검색 키워드 융합**: "오버립=オーバーリップ=overlining=lip combo"이라는 4언어 트리거 키워드. 마케팅 카피·SEO 최적화 시 4언어 동시 노출 권장.
3. **유통 차별화**:
   - **올리브영**은 틴트 우위 → 라이너 단독 아젠다 시장이 여전히 열려있음
   - **Qoo10 JP**는 K-beauty 진입 골든 채널
   - **Sephora**는 럭셔리 (CT Pillow Talk), **TikTok Shop**은 번들/콤보, **Amazon**은 매스
4. **ODM 기회**: 코스맥스 MOQ 1,000개 인하 + AI 조색 → **신규 인디 브랜드의 다셰이드 라이너 론칭 진입장벽 급감**.
5. **규제 캘린더 (2026)**:
   - **2026.7.31**: EU 향료 알레르겐 신규 라벨링 발효 → 라이너 향료 첨가 시 재라벨링 필수
   - **2026 하반기**: 중국 NMPA 등록 간소화 개정 확정 예상
   - **상시**: 미국 MoCRA 시설 등록·제품 리스팅 의무
6. **포뮬러 트렌드**: 일본 "粘膜カラー", 한국 "누디 라이너", 미국 "MLBB" — **3국 공통으로 자신의 입술 본연색에 가까운 누디톤 펜슬** 수요 급등.
