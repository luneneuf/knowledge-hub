# 마카오 전투 (1601·1622)

VOC의 중국 무역 교두보 확보 시도와 두 번의 실패. 대만 전략 전환의 분기점.

## 개요

1601년과 1622년, 네덜란드 동인도회사(VOC)는 포르투갈령 마카오를 두 차례 공략했으나 모두 패퇴했다. 두 사건은 유럽 강대국이 중국 본토에서 벌인 유일한 주요 전투로 기록되며, 하나의 전략 실패 스토리로 묶인다: 1601년의 외교적 기습 피해 → 1622년의 전면 군사 침략 → 완패 → 1624년 대만 전략 전환.

## 1601년 — 외교 실패와 포로 처형

야콥 반 넥이 이끈 탐색대 11명이 마카오에 상륙하자마자 기습 체포됐다. 뒤이어 파견된 9명도 생포. 포르투갈 캡틴-마조르 파울로 드 포르투갈이 처형 명령에 서명해, 20명 중 17명이 교수형·수장으로 처형됐다.

전면 교전이 아닌 정보 부재의 패착. 네덜란드 측이 중국 당국과 접촉해 무역권을 얻을까 두려워한 포르투갈의 선제 처형이었다.

## 1622년 — 대규모 침략과 완패

VOC 총독 얀 피터르존 쿤이 기획. 13척 1,300명 규모 침략 함대. 코르넬리스 레이어르센 지휘.

**전투 경과 (6월 23~25일)**

| 단계 | 내용 |
|------|------|
| 양동 포격 | *흐로닝언* 등 3척이 남쪽 상 프란시스쿠 포대 포격 → 동쪽 카실하스 해변 상륙 위장 |
| 상륙 | 800명 상륙; 레이어르센 복부 총상 → 한스 루핀 대위 지휘 승계; 600명 도시 진격 |
| 역전 | 진격 대열 한가운데 화약 통 폭발 → 사기 붕괴; 몬테 요새 포격 |
| 총반격 | "산티아고!" 함성; 포르투갈+마카오 혼혈인+아프리카 노예 연합 반격; 루핀 전사 |
| 궤주 | 후위대 발포 없이 도주; 6월 25일 함대 마카오 포기, 펑후 제도 이동 |

**VOC 피해**: 공식 전사 136명, 부상 126명, 포로 40명, *갈리아스* 자침, 모든 대포·군기 상실.

**쿤의 패전 보고**: "포르투갈인들은 노예들로 우리를 몰아냈다. 병사가 아니었다."

## 패인 요약

| 원인 | 내용 |
|------|------|
| 화약 폭발 | 진격 중 탄약 자폭 → 사기 붕괴 |
| 양동 간파 | 포르투갈 지휘관이 50명 결정적 증원 투입 |
| 고지 통제 실패 | 기아 언덕·몬테 요새 방어군에게 선점 |
| 다민족 연합 방어군 과소평가 | 아프리카 노예 포함 예상 초과 전투력 |
| 후위대 붕괴 | 패닉 도주로 다수 익사·피살 |

## 전략적 결과

- **1624년**: VOC, 대만에 **제일란디아 요새(Fort Zeelandia)** 건설 → 마카오 포기·대만 전략 전환
- **1623년**: 포르투갈, 마카오에 총독 제도 설치
- **6월 24일**: 마카오 시의 날(City Day)로 지정 (1999년 반환까지 공휴일)
- 포르투갈-중국 관계 강화: 전사 네덜란드인 머리를 광저우 전달, 중국 해도가 아프리카 노예에게 쌀 200픽 하사
