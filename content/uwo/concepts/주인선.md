# 주인선 제도 (朱印船, Shuinsen)

## 개요

주인선(朱印船)은 에도 막부가 발행한 도항 허가증 **주인장(朱印状)**을 소지한 무역선을 말한다. 16세기 말~17세기 초 동남아시아·중국 방면 무역의 핵심 수단이었으며, 나가사키를 거점으로 한 상인·선장들이 주도했다. **1634년 막부의 일본인 해외 도항·귀국 전면 금지령**으로 사실상 종언을 맞았으며, 이는 쇄국 체제 완성의 핵심 조치 중 하나였다.

## 제도 구조

| 항목 | 내용 |
|------|------|
| 허가 주체 | 에도 막부 (쇼군 이름으로 발행) |
| 허가 형식 | 주인장(朱印状) — 붉은 인장을 찍은 도항 허가서 |
| 발급 대상 | 일본인 상인, 외국인 거주 상인, 번(藩) 영주 |
| 주요 항로 | 대만, 동남아시아(안남·루손·시암 등), 중국 남부 |
| 활동 기간 | 1592년경~1635년 (사실상 종언) |

## 나가사키 주인선 세력

에도 초기 주인선 무역의 실권은 나가사키 대관 **스에쓰구 헤이조**와 그 휘하 선장들이 장악했다. [[하마다_야효에|하마다 야효에]]는 스에쓰구 헤이조의 부하 선장으로 타이오완(대만) 경유 명나라 무역에 종사했다. 대만-일본 항로는 원래 중국 상인 수장 리단(李旦)이 지배했으나 1625년 리단 사망 이후 스에쓰구 세력으로 이전됐다.

## VOC와의 충돌

VOC가 1624년 질란디아 요새를 건설하고 대만을 교두보로 삼으면서 주인선 세력과의 충돌이 시작됐다. VOC의 관세 부과(1625)와 화물 압수는 주인선 이익을 직접 침해했고, 이것이 [[타이오완_사건|타이오완 사건(1628)]]으로 이어졌다.

## 종언: 쇄국령과의 관계

| 연도 | 조치 |
|------|------|
| 1633 | 제1~2차 쇄국령: 무허가 도항 금지 |
| 1634 | **일본인 해외 도항·귀국 전면 금지** — 주인선 제도 실질적 종언 |
| 1635 | 제3차 쇄국령: 귀국 금지 명문화 |
| 1639 | 제5차 쇄국령: 포르투갈 추방, 쇄국 완성 |
| 1641 | VOC 상관 데지마 이전 — 유일 서양 창구 확정 |

## 역사적 의미

주인선 제도의 종언은 일본 민간의 대외 진출 가능성을 약 200년간 차단한 구조적 전환점이었다. [[타이오완_사건|타이오완 사건]]은 이 전환의 촉매 중 하나로, 주인선 선장의 VOC와의 충돌이 역설적으로 VOC의 독점을 굳히는 방향으로 귀결됐다.
