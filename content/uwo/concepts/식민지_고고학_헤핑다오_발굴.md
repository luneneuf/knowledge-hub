# 식민지 고고학 — 헤핑다오 발굴

## 정의

2002년 이후 진행 중인 기륭(Keelung) 헤핑다오(Heping Island, 和平島) 고고학 발굴 프로젝트. [[산살바도르_요새]] 본체는 CSBC 조선소 하부에 매몰됐으나, 주변부 발굴에서 중요 유구와 유골이 출토됐다. 개발 vs 문화재 보존 갈등의 전형적 사례.

## 상세

### 발굴 경과

| 연도 | 내용 |
|------|------|
| 2002 | 지표면 투과 레이더(GPR, Ground Penetrating Radar) 탐사 → 요새 위치 확인 |
| — | CSBC 조선소(1937년 일제 시대 건설)로 인해 요새 본체 발굴 불가 판정 |
| 2002~ | 주차장 아래에서 Todos los Santos 수도원 유구 발견 |
| 2011 | 한-스페인 국제 공동 발굴 개시 (중앙연구원 + 스페인 CSIC) |
| 2011~ | 유골 24구 출토, 일부 유럽계 DNA 확인 |

### 주요 발굴 성과

- **유골 24구**: 일부 유럽계 DNA 확인 — "대만 최초의 유럽계 유골"로 기록됨[^스페인_대만_요새_리서치]
- **신석기시대 유물**: 약 4,000년 전 유물도 함께 출토 — 헤핑다오의 선사시대 거주 증거
- **수도원 유구**: Todos los Santos 수도원 기초부 — [[산살바도르_요새]] 부속 시설의 유일한 지상 흔적

### 발굴의 한계

요새 본체(사각형 4보루 구조)는 1937년 일제 시대에 건설된 CSBC(中國造船公司) 조선소 하부에 완전히 매몰돼 있다. 조선소 운영 중에는 직접 발굴이 불가능하다.

## 사례로서의 의미

헤핑다오 발굴은 식민지 고고학의 여러 문제를 동시에 보여준다:
- **다층 식민지 역사**: 신석기 원주민 → 스페인 → 네덜란드 → 청 → 일본 → ROC 순서로 쌓인 역사 지층
- **개발 vs 문화재**: 현대 산업 시설이 역사 유적 위에 건설된 이후의 발굴 제약
- **국제 협업**: 한국-스페인 공동 발굴 — 스페인 측이 자국 식민지 유산 연구에 참여하는 구조
