# UWO History 콘텐츠 스타일 가이드

> 참고 소스: [[대항해시대_오리진_통합참고자료]] · [[moc]]

---

## 1. 콘텐츠 목적 & 타깃 독자

### 목적
대항해시대 오리진(UWO)에 등장하는 실존 인물·사건·유물의 **실제 역사**를 SNS 콘텐츠로 풀어낸다.
게임 내 스펙(등급·성향·직업·특기)을 역사적 사실과 연결해, 유저가 캐릭터를 더 입체적으로 이해하도록 돕는다.

### 타깃 독자
- **1차:** UWO 플레이어 — 항해사 스펙의 역사적 근거가 궁금한 유저
- **2차:** 역사에 관심 있는 일반 SNS 유저 — 게임을 몰라도 읽히는 역사 스토리

### 채널
트위터(X) 스레드가 주력. 블로그·유튜브 스크립트 재활용을 전제로 작성한다.

---

## 2. 톤 & 보이스

| 항목 | 기준 |
|------|------|
| 언어 | 한국어. 고유명사는 한국어 표기 + 원어 병기 (첫 등장 시) |
| 문체 | 구어체 허용. 단, 경박하지 않게 — "~했다" 기반의 간결한 서술 |
| 시점 | 3인칭 관찰자. 주관적 해석은 "~로 보인다", "~라고 할 수 있다"로 명시 |
| 감정 | 아이러니·반전·의외성을 핵심 감정 훅으로 활용. 드라마화는 OK, 과장은 NG |
| 게임 언급 | 게임 스펙은 말미 또는 보충 파트로 분리. 역사가 본문, 게임이 부록 |
| 길이 | 트위터 스레드: 7~15트윗. 각 트윗은 독립적으로도 읽혀야 함 |

### 피해야 할 톤
- "사실 알고 계셨나요?" 류의 클릭베이트 오프닝
- 현대 기준으로 과거 인물을 일방적으로 평가하는 문장
- 게임 과금·픽업 일정을 역사 콘텐츠와 혼재

---

## 3. 글 구조 템플릿

### 트위터 스레드 (인물 중심)

```
[트윗 1 — 훅]
인물명 + 한 줄 반전/아이러니 팩트
→ 클릭을 유발할 가장 의외의 사실로 시작

[트윗 2 — 배경]
시대·지역·신분 간략 소개 (2~3문장)
게임 등급/성향 1줄 메모 가능

[트윗 3~N — 핵심 에피소드]
에피소드당 1~2트윗
팩트 → 의미/해석 순서로 전개
숫자·날짜·고유명사로 신뢰감 부여

[트윗 N+1 — 반전/클라이맥스]
가장 극적인 사실 배치
앞 내용과 대비되는 결말

[마지막 트윗 — 게임 연결]
UWO 항해사 스펙 요약 (등급·직업·특기)
등장 항구 또는 획득 방법
"이 사람, 게임에서는 이렇게 만날 수 있습니다"
```

### 발견물·아이템 중심 (게임 오브젝트 + 역사)

```
[훅] 게임 아이템명 → 실제 역사 기원
[본문] 역사적 맥락 3~5개 팩트
[연결] 게임 내 발견 조건·지역
```

## 4. 용어 통일 규칙

### 인물명 표기
| 표기 기준 | 예시 |
|-----------|------|
| 한국식 표기 우선, 첫 등장 시 원어 병기 | 이사벨라 데스테 (Isabella d'Este) |
| 동아시아 인물: 한국 한자 독음 또는 현지 독음 중 통용되는 쪽 | 정화(鄭和), 이순신(李舜臣) |
| 게임 표기 ≠ 역사 표기인 경우 게임 표기 먼저, 실명 병기 | 그레이스 오말리 (Gráinne Mhaol) |

### 지명 표기
- 현재 지명 + 당시 지명 병기: "리스본(당시: 리스보아)"
- 게임 내 항구명이 역사 지명과 다를 경우 주석 처리

### 등급·스펙 표기
- 등급: A급, S급 (대문자)
- 성향: 교역 / 전투 / 탐험 (게임 내 표기 그대로)
- 직업: 협상가, 항해사, 전사 등 (게임 내 표기 그대로)

### 시대 표기
- 연도는 서력 기준. 동양 인물의 경우 왕조·연호 병기 가능
- 예: "1592년(선조 25년, 임진왜란)"

## 5. 금지 사항

### 내용 관련
- **역사 왜곡**: 사실 확인 없는 추측을 단정문으로 쓰지 않는다
- **현대 정치 투영**: 역사 인물을 현재 이념으로 재단하는 해석 금지
- **국가·민족 우열 비교**: "우리 역사가 더 위대하다" 류의 비교 금지
- **미확인 출처 인용**: "~썰이 있다"는 OK, 그걸 사실처럼 확정하는 건 NG

### 구조 관련
- 훅 없이 연대기 나열로 시작하는 글 금지
- 게임 스펙 설명으로 글을 시작하는 글 금지 (역사가 먼저)
- 단일 트윗에 3개 이상 에피소드 압축 금지 (희석됨)

### 표현 관련
- "최초", "유일", "세계 제일" — 출처 없이 사용 금지
- "놀랍게도", "충격적으로" 등 감탄사 남발 금지
- 이모지: 스레드 당 5개 이하 권장 (과다 사용 시 신뢰도 저하)

## 6. 예시 문단

### 좋은 예 — 이사벨라 데스테 스레드 오프닝

> 레오나르도 다빈치에게 그림을 거절당한 여자가 있다.
> 그것도 한 번이 아니라 여러 번.
> 당대 "세계의 제일 귀부인"이라 불렸는데도.
> 그녀가 결국 손에 넣은 건 소묘 한 장뿐이었다.
> 그 소묘, 지금 루브르에 있다.

**왜 좋은가:** 반전 구조, 짧은 문장, 시각적 결말, 클릭 유도.

---

### 나쁜 예 — 연대기 나열

> 이사벨라 데스테는 1474년 페라라에서 태어났다. 6세에 약혼하였고, 1490년 만토바의 프란체스코 2세 곤차가와 결혼하였다. 르네상스 시대 이탈리아의 저명한 귀부인으로…

**왜 나쁜가:** 훅 없음. 위키피디아 요약과 구분 안 됨. 첫 문장에 아무 감정 자극이 없음.

---

### 좋은 예 — 게임 연결 마무리 트윗

> UWO에서 이사벨라 데스테는 A급 협상가.
> 협상 성공 확률 +2, 언어 스킬 라틴어·그리스어 LV4.
> 출현 항구: 낭트, 보르도, 마르세유, 몽펠리에.
> 역사가 스펙이 된 사람.

**왜 좋은가:** 스펙을 나열하되 마지막 줄에 의미 부여. 역사 콘텐츠의 맥락과 자연스럽게 연결됨.

---

### 나쁜 예 — 게임 광고성 오프닝

> 이번 픽업에 이사벨라 데스테가 등장합니다! A급 협상가로 스펙이 좋습니다. 어떤 인물인지 알아볼까요?

**왜 나쁜가:** 역사 콘텐츠가 아니라 게임 광고처럼 읽힘. 역사 독자를 잃는다.
