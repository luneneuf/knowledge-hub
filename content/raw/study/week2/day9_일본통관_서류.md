# 화장품 일본 수출 통관 서류

## 1. 일본 화장품 분류 체계

일본에서 화장품은 **약사법(薬事法, 현 薬機法)**에 따라 두 가지로 분류된다. 분류에 따라 필요한 허가 및 서류가 크게 달라진다.

| 구분 | 일반화장품 | 의약부외품(약용화장품) |
|------|----------|---------------------|
| 정의 | 청결 유지·미적 효과 목적 | 화장품과 의약품 중간 단계 |
| 효능·효과 표현 | 화장품 기준 내 표현만 허용 | 미백·주름개선 등 특정 효능 표현 가능 |
| 허가 절차 | 도도부현(都道府縣) 신고 | 후생노동성 승인 (PMDA 심사) |
| 소요 기간 | 짧음 | 수년 이상 |
| 브랜드 활용 | 일반적 | 고부가가치·기능성 마케팅 가능 |

> **K-beauty 브랜드가 '미백 효과' 또는 '주름개선' 표현을 원할 경우 의약부외품 승인 필수**. 단순 보습 기능은 일반화장품으로 수출 가능.

## 2. 일본 수입 통관 절차 (일반화장품)

```
1. 제조판매업 허가 취득
   └─ 일본 내 제조판매업자(수입사)가 필요

2. 제조업 허가 취득
   └─ 보관 창고 포함 (화장품 제조업 허가)

3. 제품 신고 (외국신고 + 제조판매 신고)
   └─ 도도부현에 신고

4. 성분 적합 확인
   └─ "화장품 기준" 금지·제한 성분 여부 확인
   └─ 일본 금지 성분 ≠ 한국 허용 성분 (사전 확인 필수)

5. 전성분 표기 명칭 확인
   └─ 일본화장품공업연합회(JCIA) 기준 명칭 사용

6. 라벨링 준수 확인
   └─ 일본어 전성분, 판매원 표기 등

7. 통관 신고 및 허가
   └─ 세관 신고 → 통관 허가 → 창고 반입
```

## 3. 브랜드사가 제공해야 하는 주요 서류

| 서류명 | 설명 | 비고 |
|--------|------|------|
| 제조판매업 허가증 | 일본 내 수입사/판매사 보유 서류 | 수입사 측 |
| 사업자등록증 (영문) | 한국 브랜드사 제공, 2부 | 제조사 포함 |
| 제품 성분표 | INCI명 및 농도 포함 (일본 기준 명칭) | 핵심 서류 |
| 제조공정도 | 제조 흐름 기재 | 요청 시 |
| COA (시험성적서) | 품질 검사 결과 | 로트(Lot)별 |
| MSDS | 위험물 분류 해당 시 | - |
| 판매 허가증/판매증명서 | 한국 내 정상 판매 입증 | 요청 시 |
| BSE 관련 서류 | 동물 유래 성분 포함 시 | 해당 시 |
| 라벨 사양서 | 일본어 라벨 최종본 | - |

## 4. 주요 규제 주의사항

### 금지 성분 차이
한국 허용이라도 일본에서 금지된 경우 포뮬러 변경 필요.

| 성분 | 한국 | 일본 |
|------|------|------|
| 트라넥삼산 (Tranexamic Acid) | 화장품 성분으로 허용 | 의약품 원료로 분류 → 화장품 배합 금지 |

### 기능성 표현별 분류

| 표현 | 분류 | 요건 |
|------|------|------|
| 피부 보습 | 일반화장품 | 일반 신고 |
| 미백 (美白) | 의약부외품 | 후생노동성 승인 |
| 주름 깊이 개선 | 의약부외품 | 후생노동성 승인 |
| 건조에 의한 잔주름 | 일반화장품 | 일본향장품학회 효과시험 통과 필요 |
| SPF 자외선 차단 | 의약부외품 | 후생노동성 승인 |

### 라벨링 특이사항

- **산화철 표기**: 여러 종류의 산화철을 사용하더라도 "산화철(酸化鉄)"로 통합 표기
- **전성분 표기 명칭**: 일본화장품공업연합회(JCIA) 기준 명칭 사용 (INCI명과 다를 수 있음)
- **과장 광고 금지**: "영양", "치료" 등 효능 범위 이외 문구 불가
- 제품명(판매명)에 효능·효과 범위 이외 명칭 사용 금지

## 5. 의약부외품 통관 절차

```
1. 제조판매업 허가 취득
2. 제조업 허가 취득
3. 각 제품별 의약부외품 승인신청
   └─ PMDA(의약품의료기기종합기구) 심사
   └─ 수년 이상 소요
4. 후생노동성 승인 취득
5. 도도부현 신고
6. 법정표시 준수
7. 통관 신고 및 허가
```

## 6. 수입대행 활용

일본 수출 시 직접 허가를 취득하지 않고 **기존 허가를 보유한 수입대행업자를 통해 수입**하는 방식이 일반적이다.

- 수입대행사가 제조판매업 허가 및 제조업 허가 보유
- 브랜드사는 제품 성분표·서류 제공 역할
- 수입대행사가 세관 신고, 통관, 창고 반입 처리
- 드럭스토어·버라이어티숍 납품은 도매상을 통해 처리

## 7. 통관 후 일본 내 유통 구조

```
한국 브랜드 → 수입대행사 → 화장품 도매상 → 드럭스토어·버라이어티숍
(제조·공급)    (통관·창고)   (유통 마진 취득)  (소매 판매)
```

- **직납 불가**: 일반적으로 소매점 직접 납품 어려움, 도매상 필수
- **마츠모토키요시, 스기약국, 로프트** 등은 도매상을 통한 입점
- 예외적으로 일부 대형 체인과 직접 거래 협상 가능
