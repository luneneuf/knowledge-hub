# Boots UK 바이어 리뷰·재오더 기준

## 1. Boots UK 개요

Boots(부츠)는 영국 최대 헬스·뷰티 소매 체인으로, 약 2,000개 이상의 매장을 운영한다. 처방전 의약품부터 화장품·스킨케어까지 폭넓은 제품군을 다루며, 영국 뷰티 시장에서 세포라와 함께 핵심 채널이다.

| 항목 | 내용 |
|------|------|
| 매장 수 | 약 2,000개 이상 (영국) |
| 채널 | 오프라인 매장 + boots.com |
| 주요 카테고리 | 스킨케어, 색조, 헤어케어, 의약품 |
| 신제품 소싱 | RangeMe 플랫폼 활용 |

## 2. Boots 바이어 조직

- **Senior Buying Manager** 중심으로 카테고리별 바이어 구성
- 바이어는 매일 전날 판매 데이터를 확인하고 신제품 출시 성과를 모니터링
- 연간 판매 계획 수립 및 수요 예측(Forecasting) 담당
- 신제품 발굴은 RangeMe 플랫폼을 통해 브랜드가 직접 제안 가능

## 3. 신제품 소싱 경로

### RangeMe 플랫폼

Boots UK는 **RangeMe**를 통해 신제품 제안을 공식 접수한다.

```
브랜드 → RangeMe 프로필 등록 → 제품 정보·사진·성분 업로드
                ↓
        Boots UK 바이어 검색 및 리뷰
                ↓
        관심 브랜드에 연락 → 추가 정보 요청 → 샘플 요청
                ↓
        내부 평가 → 입점 결정
```

- RangeMe 외에 직접 바이어에게 이메일·샘플 발송 방식도 병행
- Boots는 자체 소비자 조사 데이터를 기반으로 트렌드 수요 파악 후 소싱

## 4. 바이어가 주목하는 브랜드 기준

| 기준 | 세부 내용 |
|------|----------|
| 제품 혁신성 | "Think outside the box" — 기존 제품과 차별화 필수 |
| 소비자 이해도 | 영국 소비자의 니즈에 대한 깊은 이해 증명 |
| 지속가능성 | 친환경·지속가능 원료 브랜드 선호 |
| 성분 투명성 | 소비자들이 성분 중요성 증가, 투명한 성분 공개 필요 |
| 전문성 | 특정 피부 고민·스킨케어 조건에 특화된 전문 브랜드 |
| 시장 데이터 | 기존 판매 실적, 온라인 리뷰, 소비자 반응 자료 |

## 5. 재오더 판단 기준

Boots는 공식적으로 재오더 기준을 공개하지 않으나, 업계 관행 기반으로 다음 지표를 활용한다:

| 지표 | 설명 |
|------|------|
| 일별 판매 데이터 (POS) | 바이어가 매일 확인하는 핵심 지표 |
| 신제품 초기 성과 | 론칭 후 첫 몇 주간의 판매 속도 |
| 온라인 리뷰·평점 | boots.com 및 소셜미디어 반응 |
| 재고 회전율 | 발주 수량 대비 판매 속도 |
| 반품률 | 소비자 반품 비율 (화장품 CS 이슈 반영) |
| 시즌 트렌드 적합성 | 계절별 카테고리 성과 |

## 6. Boots CS 정책 (반품·교환)

Boots의 고객 서비스 정책은 **브랜드사가 Boots에 납품 시 반품 처리 방식을 이해하는 데 중요**하다.

| 항목 | 내용 |
|------|------|
| 반품 기간 | 35일 이내 |
| 미개봉 제품 | 영수증 지참 시 환불 또는 교환 |
| 개봉 화장품 | **원칙적으로 반품 불가** (결함 있는 경우 예외) |
| **화장품 교환 정책** | **개봉 후 35일 이내 교환 가능** (Make-up Exchange Promise) |
| 온라인 반품 | 우편 무료 반송 (Royal Mail 이용) |
| 환불 소요 기간 | 환불 처리 후 14 업무일 이내 |
| 매장 반품 | 영국 내 모든 Boots 매장에서 가능 |

### Make-up Exchange Promise
Boots는 업계 최초로 **개봉한 메이크업 제품도 교환 가능**한 정책을 도입했다.
- 35일 이내 교환
- 온라인 구매는 해당 없음 (오프라인 매장만)
- 결함 없는 제품도 교환 가능 (단, 환불은 불가)

## 7. 브랜드사 납품 실무

### Goods for Resale (재판매 제품) 공급업체 등록

Boots UK는 자체 공급업체 포털을 통해 신규 공급업체를 등록한다.

```
RangeMe 또는 직접 접촉 → 바이어 관심 확인
         ↓
공급업체 등록 신청 (boots-uk.com/suppliers/goods-for-resale/)
         ↓
서류 제출 (품질 인증, 제품 보험, 규제 준수 서류)
         ↓
계약 협의 → 마진율, 결제 조건, 재고 조건
         ↓
첫 발주 → 납품 → 성과 모니터링
```

### 주요 계약 조건 (일반 업계 수준)

| 항목 | 일반 수준 |
|------|----------|
| 마진율 | MSRP 대비 40~50% (협상 사안) |
| 결제 조건 | Net 30~60일 |
| 최소 발주량 | 카테고리·제품별 상이 |
| 반품 조건 | 판매 부진 제품 반품 가능성 계약 명시 필요 |
