# 세포라 브랜드 확장 구조 (테스트 도어 → 전국 롤아웃)

## 1. 세포라 입점 프로세스 개요

세포라에 입점하는 경로는 크게 두 가지다:

| 경로 | 설명 |
|------|------|
| **직접 바이어 피칭** | 브랜드가 카테고리 바이어에게 샘플 발송 → 미팅 → 계약 협의 |
| **Sephora Accelerate 프로그램** | 신흥 브랜드 대상 인큐베이터 (매년 지원 모집, 졸업 후 전 도어 진입 가능) |

## 2. 바이어 피칭 단계별 프로세스

```
1단계: 브랜드 자체 평가
   └─ 세포라 고객층과 제품 적합성 확인
   └─ FDA 준수 여부, 세포라 금지 성분 리스트 검토
   └─ GMP 인증 서류 준비

2단계: 바이어 식별
   └─ 세포라 SF 본사 카테고리 바이어를 LinkedIn으로 파악
   └─ (카테고리별 담당: 스킨케어, 색조, 향수, 헤어 등)

3단계: 샘플 발송
   └─ 풀사이즈 + 미니어처 포함한 브랜드 박스 발송
   └─ 브랜드 소개서, 개인 메모 동봉

4단계: 팔로업
   └─ 발송 3~4일 후 이메일·전화 연락
   └─ 최대 3회 이메일 팔로업

5단계: 바이어 미팅 (초대 시)
   └─ SF 본사 방문 또는 영상 미팅
   └─ 브랜드 스토리 + 판매 데이터 PPT 발표
   └─ 과거 DTC/소매 실적 데이터 필수

6단계: 계약 협상
   └─ 입점 조건 (마진율, 결제 조건, 독점 기간)
   └─ "Made for Sephora" 전용 제품 또는 조기 출시 독점권 협상 가능
```

## 3. 도어 확장 단계

### 단계 1: 테스트 도어 (Limited Doors)
- 미국 기준 약 50~100개 매장 선정
- 지역: 주요 도시 또는 특정 지역 집중
- 목적: 소비자 반응·셀스루 테스트
- 기간: 약 6개월

### 단계 2: 중간 확장 (Regional Expansion)
- 성과 우수 시 200~400개 도어로 확장
- 온라인(Sephora.com) 병행 판매 시작
- 캠페인·에디토리얼 지원 확대

### 단계 3: 전국 롤아웃 (All-Door Brand)
- 미국 전 세포라 매장 (~540개) 입점
- 시즌 프로모션·엔드캡 배치 가능
- "Sephora Collection" 수준의 노출 확보

## 4. 세포라 Accelerate 프로그램

세포라 Accelerate는 **신흥 브랜드를 위한 미국 최대 뷰티 인큐베이터 프로그램**이다.

| 항목 | 내용 |
|------|------|
| 대상 | 스킨케어, 색조, 헤어케어, 향수 신흥 브랜드 |
| 매년 선발 브랜드 수 | 약 10~15개 |
| 누적 졸업 브랜드 | 41개 이상 |
| 혜택 | 멘토링, 세포라 바이어 접근, 마케팅 지원, 전국 배포 가능성 |
| 졸업 사례 | EADEM (전 세포라 매장 확장), Kulfi (전 도어, 립스테인 Top 3) |
| 지원 시기 | 매년 공개 모집 (2026년 지원 접수 진행 중) |

## 5. 세포라가 브랜드에 요구하는 것

| 요건 | 세부 사항 |
|------|----------|
| 제품 혁신성 | 기존 제품과 차별화된 성분·포뮬러·포장 |
| 높은 디지털 발자국 | 인스타그램·틱톡 팔로워, 온라인 리뷰 실적 |
| DTC 판매 실적 | Sephora.com 외 자체 채널에서의 검증된 수요 |
| FDA/GMP 준수 | 미국 규제 충족 서류 |
| 지속가능성 | 친환경 패키지, 지속가능 원료 사용 선호 |
| 다양성 | 여성 창업자, 소수자 소유 브랜드 우대 가능성 |
| 공급망 안정성 | 대량 주문 처리 가능 제조·물류 역량 |

## 6. 계약 조건 핵심 사항

세포라는 일반적으로 다음을 요구한다:

- **독점 제품**: "Made for Sephora" 전용 SKU 또는 컬러
- **조기 출시 독점권**: 신제품 세포라 우선 출시 권한
- **마진율**: 일반적으로 MSRP 대비 50~55% 수준 (비공개, 협상 사안)
- **결제 조건**: Net 30~60일 (협상 사안)
- **재고 책임**: 반품 조건 및 재고 처리 기준 계약

## 7. K-beauty 브랜드를 위한 시사점

- 세포라 입점 전 **DTC(자사몰) 또는 아마존 실적 데이터** 확보 필수
- 미국 Responsible Person 지정 및 MoCRA 등록 선행 필요
- 독점 컬러 또는 세포라 전용 패키지로 차별화 포인트 제공
- 런칭 전 **Beauty Influencer + PR 에이전시** 병행 준비
- 셀스루 데이터 실시간 모니터링 및 바이어 보고 자료 준비
