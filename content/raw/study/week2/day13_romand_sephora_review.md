# rom&nd 세포라 리뷰 분석 및 CS 클레임

## 1. rom&nd 브랜드 개요

**rom&nd(롬앤드)**는 한국 K-beauty 브랜드로, 립 틴트 특히 **Juicy Lasting Tint(JLT)**로 유명하다. 세포라 입점을 통해 미국·영국 등 글로벌 시장에 진출한 주요 K-beauty 색조 브랜드다.

| 항목 | 내용 |
|------|------|
| 대표 제품 | Juicy Lasting Tint, Blur Fudge Tint |
| 특징 | 과일 향, 바이럴 색상, 저가·고품질 이미지 |
| 주요 채널 | 세포라, OLIVE YOUNG, Amazon |
| 경쟁 브랜드 | LAKA, BBIA, BLAYE, 3CE |

## 2. 2025년 CS 클레임 급증 사건

### 사건 개요

2025년 5월부터 rom&nd Juicy Lasting Tint 관련 **구순염(cheilitis, 입술 염증) 클레임이 약 300배 급증**했다.

| 항목 | 내용 |
|------|------|
| 발생 시기 | 2025년 5월 |
| 주요 증상 | 구순염, 입술 건조, 입술 트러블 |
| 클레임 급증 규모 | 약 300배 (이전 대비) |
| 주요 의심 성분 | 염료 Red 202/201, 빠른 증발 용제(알코올 유사 성분) |
| 3rd party 검사 결과 | 염료 자체의 자극성은 확인되지 않음 |
| 브랜드 자체 분석 | 빠른 증발 용제(알코올 flash-off)가 건조 원인 가능성 |

### 브랜드 대응

| 대응 | 내용 |
|------|------|
| 공식 발표 | "클레임 급증 인지, 원인 조사 중" |
| 대응 프로젝트 | **Project ZERO** — 문제 용제 없는 새로운 틴트 포뮬러 개발 |
| 커뮤니케이션 | 소비자·채널 바이어에게 투명하게 대응 공개 |
| 포뮬러 방향 | 빠른 증발 용제 제거한 신 포뮬러 개발 |

## 3. 세포라 리뷰 패턴 분석

세포라(Sephora.com) 및 Beauty Insider Community에서 K-beauty 립 제품 관련 주요 리뷰 패턴:

### 긍정 리뷰 패턴 (rom&nd JLT)
- 색상 발색 우수
- 지속력 우수 (식사 후에도 유지)
- 가벼운 텍스처, 끈적임 없음
- 바이럴 색상 다양성

### 부정 리뷰 패턴 (2025년 이후)
- 사용 후 입술 건조, 가려움
- 구순염 증상 (부어오름, 붉어짐)
- 색상이 줄어도 입술 건조감 지속
- "이 제품 사용 후 피부과 방문"

### 리뷰 등급 분포 (추정)
- 클레임 급증 전: 4.0~4.5/5.0 유지
- 클레임 급증 후: 1~2성 리뷰 비율 증가

## 5. K-beauty 브랜드 공통 리스크 포인트

rom&nd 사례에서 파악되는 K-beauty 세포라 입점 브랜드 공통 CS 리스크:

| 리스크 | 설명 |
|--------|------|
| 민감성 소비자 증가 | 서구 시장 소비자의 알레르기·피부 민감성 인식이 한국보다 높음 |
| 염료 규제 차이 | 한국 허용 염료가 미국·EU에서 제한될 수 있음 |
| 성분 클레임 모니터링 | 출시 후 6개월 이내 리뷰 패턴 집중 관찰 필요 |
| CS 대응 속도 | 미국·유럽 소비자는 브랜드 공식 대응을 기대하며, SNS 대응 속도 중요 |
| 바이어 관계 | CS 클레임 급증은 바이어와의 계약 재검토 트리거 |

## 6. LAKA 관점 시사점

LAKA가 세포라 입점 시 참고할 사항:

1. **성분 사전 검토**: 세포라 금지 성분 + 미국·EU 제한 성분 철저 검토
2. **포뮬러 안전성**: 알코올 계열 용제 비율 최소화 또는 제거 검토
3. **CS 모니터링 시스템**: 세포라 리뷰 + SNS 실시간 모니터링 체계 구축
4. **바이어 소통**: CS 이상 패턴 감지 시 선제적으로 바이어에 보고 및 대응 계획 공유
5. **이상사례 보고 체계**: MoCRA 기준 SAE 보고 시스템 사전 구축
