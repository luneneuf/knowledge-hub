# 미국·영국 화장품 이상사례 보고 차이 및 브랜드 책임

## 1. 용어 비교

| 지역 | 용어 | 기관 |
|------|------|------|
| 미국 | SAE (Serious Adverse Event) | FDA |
| EU | SUE (Serious Undesirable Effect) | 각국 국가 당국 |
| 영국 | SUE (EU 기준 계승) | OPSS (Office for Product Safety and Standards) |
| 한국 | 이상사례 / 부작용 | 식품의약품안전처 |

## 2. 미국 MoCRA 이상사례 보고 제도

### 적용 법령

**MoCRA (Modernization of Cosmetics Regulation Act of 2022)**
- 시행: 2022년 12월 29일 제정
- 의무화: 2023년 12월 29일부터 시행

### Serious Adverse Event (SAE) 정의

다음 중 하나에 해당하는 경우 SAE로 분류한다:

| 기준 | 예시 |
|------|------|
| 사망 | 제품 사용과 관련된 사망 |
| 생명 위협 | 아나필락시스 등 |
| 입원 치료 | 피부 반응으로 인한 입원 |
| 영구적 장애 | 피부 영구 손상 |
| 선천성 이상 | 임신 중 사용 관련 |
| 감염 | 오염 제품으로 인한 감염 |
| 심각한 외모 훼손 | 화상, 심각한 피부 손상 |

### 보고 의무자 (Responsible Person)

**Responsible Person = 제품 라벨에 이름이 표시된 제조사·포장사·배포사**

- 운영 지원은 외주 가능하지만, **규제 책임은 RP에게 귀속**
- RP 연락처(주소, 전화, 이메일 중 하나)를 라벨에 반드시 표기

### 보고 타임라인

```
이상사례 인지 시점
       ↓
【15 업무일 이내】FDA에 SAE 보고 (의무)
       ↓
【1년 이내】 추가 정보 확보 시 보완 보고
```

### 기록 보관

| 주체 | 보관 기간 |
|------|----------|
| 일반 기업 | **6년** |
| 소규모 유통업체 | **3년** |

### 라벨 요건

- 소비자가 이상사례를 보고할 수 있는 연락처 라벨 포함 의무
- 형식: 주소, 전화번호, 이메일 중 택일

## 3. 미국 MoCRA 시스템 구축 요건

SAE 보고를 위해 브랜드사가 갖춰야 할 내부 시스템:

| 항목 | 내용 |
|------|------|
| 복수 접수 채널 | 이메일, 웹폼, 전화 등 |
| 사안 분류 기준 | SAE vs 일반 이상사례 서면 기준 |
| 에스컬레이션 절차 | 15일 타임라인 내 보고 완료 절차 |
| 의료적 검토 | 인과관계 평가 (의료 전문가 검토 권장) |
| 제출 추적 시스템 | FDA 보고 번호 및 보완 보고 관리 |
| 라벨 아카이브 | 제품별 라벨 버전 관리 |
| 기록 보관 아키텍처 | 6년 보관, FDA 감사 대비 조직화 |
| 트렌드 분석 | 반복 패턴 감지 → 포뮬러·생산 개선 |

## 4. 영국 이상사례 보고 제도

### 적용 규정

**UK Cosmetics Regulation (Retained EU Regulation 1223/2009)**
- 브렉시트 이후 영국이 EU 화장품 규정을 자국법으로 수용

### 보고 구조

| 항목 | 내용 |
|------|------|
| 보고 기관 | OPSS (Office for Product Safety and Standards) |
| 보고 주체 | 영국 내 Responsible Person |
| 보고 방법 | 전화·이메일 또는 OPSS 온라인 시스템 |
| 보고 기한 | 중대 이상사례 발생 즉시 (정확한 일수 제도 운영) |

### SUE (Serious Undesirable Effect) 기준

EU/UK 기준에서 SUE는 미국 SAE와 유사하나, **사망·입원·영구 장애** 외에 "의료적 개입 없이는 위험할 수 있는 상태"도 포함.

## 5. 미국 vs 영국 이상사례 보고 비교

| 항목 | 미국 (MoCRA) | 영국 (UK Cosmetics Regulation) |
|------|------------|-------------------------------|
| 용어 | SAE (Serious Adverse Event) | SUE (Serious Undesirable Effect) |
| 보고 기관 | FDA | OPSS |
| 보고 기한 | **15 업무일** | 신속 보고 (명시적 일수는 구체적 규정 확인 필요) |
| 기록 보관 | 6년 (소기업 3년) | PIF와 함께 10년 보관 |
| RP 책임 | 라벨 표기 제조사·포장사·배포사 | 영국 내 Responsible Person |
| 라벨 연락처 | 의무 (이상사례 보고용) | 의무 |
| PIF 요건 | 별도 요건 없음 | 영어 PIF 10년 보관 |

## 6. 공통 컴플라이언스 시스템 설계 권장

미국·영국 양 시장에 진출한 브랜드는 **통합 컴플라이언스 시스템** 구축이 효율적이다.

```
[CS 접수 채널] (전화/이메일/SNS/바이어 통보)
       ↓
[중앙 CS 시스템에 기록 (다국어 지원)]
       ↓
[분류: 일반 CS vs 이상사례 vs SAE/SUE 해당 여부]
       ↓
├─ 일반 CS → 표준 응대
├─ 이상사례 → 기록·추적
└─ SAE/SUE 해당 → 15일 내 FDA + OPSS 보고
       ↓
[사내 의료 검토 → 인과관계 평가]
       ↓
[기록 보관 (6년/10년 기준)]
       ↓
[분기/반기 트렌드 리포트 → 포뮬러·생산 피드백]
```

## 7. 이상사례 급증이 채널에 미치는 영향

이상사례(CS 클레임)가 증가할 경우 채널 바이어가 취하는 조치:

| 단계 | 바이어 대응 |
|------|-----------|
| 1단계 | 브랜드사에 공식 문의 및 대응 계획 요청 |
| 2단계 | 온라인 상품 페이지 임시 비활성화 |
| 3단계 | 오프라인 진열 제거 (리콜 수준) |
| 4단계 | 계약 재검토 또는 디리스팅 |

> rom&nd 사례(2025): 세포라 입점 브랜드가 CS(구순염 클레임) 300배 급증 → 포뮬러 개선 'Project ZERO' 발표.
