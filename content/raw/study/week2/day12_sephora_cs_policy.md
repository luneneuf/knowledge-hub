# 세포라 CS 정책 (반품·교환·고객 서비스)

## 1. 반품 기본 정책 (2025년 4월 기준)

세포라는 2025년 4월 24일 이후 구매 제품에 대해 아래 정책을 적용한다.

| 항목 | 내용 |
|------|------|
| 반품 가능 조건 | 새 제품(New) 또는 살짝 사용한 제품(Gently Used) |
| 반품 기간 | 구매 후 **30일 이내** |
| 환불 방식 | 원래 결제 수단으로 전액 환불 |
| 영수증 없는 경우 | 매장 크레딧(Store Credit)으로만 환불 |

### "Gently Used" 기준
- "몇 번 사용했고 대부분의 내용물이 남아 있는 상태"
- 많이 사용한 제품은 반품 불가

## 2. 반품 경로

| 구매 방법 | 반품 가능 경로 |
|----------|-------------|
| Sephora.com (온라인) | 매장 방문 또는 우편 반송 |
| 오프라인 매장 | 전국 모든 Sephora 매장 |
| BOPIS (온라인 주문·매장 픽업) | 모든 Sephora 매장 |
| Same-Day Delivery | 모든 Sephora 매장 |
| Sephora at Kohl's | **Kohl's 매장만 가능** (Sephora 매장 불가) |
| Instacart/DoorDash/Shipt | 모든 Sephora 매장 |

## 3. 결제 수단별 환불 방식

| 결제 수단 | 환불 방식 | 주의사항 |
|----------|----------|---------|
| 신용카드/직불카드 | 원래 카드로 환불 | - |
| Afterpay | 선택한 신용/직불카드로 환불 | Afterpay 계정으로 환불 불가, Afterpay 잔금은 계속 납부 의무 |
| Klarna | Klarna 계정으로 환불 | 납부 계획 금액 및 기간 자동 조정 |
| 현금 | 현금 환불 | - |

## 4. 구매 증빙 서류

매장 반품 시 다음 중 하나 제출:

- 오리지널 패킹 슬립
- Sephora.com 주문 확인 이메일 또는 배송 확인 이메일
- 모바일 기기의 Sephora.com 주문 정보

## 5. 반품 제한 사항

| 항목 | 내용 |
|------|------|
| Sephora at Kohl's 제품 | Sephora 매장 반품 불가 (Kohl's만 가능) |
| Kohls.com 주문 | Sephora 반품 불가 |
| 30일 초과 제품 | 반품 불가 (정책 변경 전 구매 제품 기준 달라질 수 있음) |

## 6. 브랜드사 관점: 세포라 반품 정책의 영향

세포라의 30일 반품 정책은 **브랜드사 SCM 및 CS 운영에 직접 영향**을 미친다.

### 반품 재고 처리

- 세포라는 반품 제품을 브랜드사에 반송하거나 폐기 처리
- 브랜드사는 세포라와의 계약에서 **반품 처리 방식(반송/폐기) 및 비용 부담 주체** 사전 합의 필요

### CS 클레임 모니터링 의무

- 세포라 온라인 리뷰 및 반품 사유 데이터를 브랜드사에 공유 여부는 계약 사항
- 브랜드사는 세포라 리뷰 페이지를 직접 모니터링해 이상사례 조기 감지 필요

### 반품 급증 시 채널 대응

반품률이 비정상적으로 증가할 경우:
1. 세포라 바이어가 브랜드사에 문의
2. 온라인 페이지 임시 비활성화 가능
3. 오프라인 진열 축소 또는 제거
4. 계약 재검토 트리거

## 8. Beauty Insider 프로그램과 CS 연계

세포라 Beauty Insider(로열티 프로그램)는 CS 경험과 밀접히 연계된다.

| 등급 | 혜택 |
|------|------|
| Insider (무료) | 기본 혜택, 포인트 적립 |
| VIB (\$350 이상/년) | 추가 혜택, 시즌 할인 |
| Rouge (\$1,000 이상/년) | 프리미엄 혜택, 무제한 무료 배송 |

> Beauty Insider 상위 등급 소비자는 반품 후 재구매 비율이 높아, 이들의 CS 경험 관리가 브랜드 매출 유지에 중요하다.
