# 화장품 이상사례 보고 가이드 (한국)

## 1. 법적 근거

화장품법에 따라 화장품 책임판매업자 및 맞춤형화장품 판매업자는 **이상사례(부작용)를 식품의약품안전처장에게 보고**해야 한다.

### 관련 법령

| 법령 | 주요 내용 |
|------|----------|
| 화장품법 제5조의2 | 화장품 이상사례 보고 의무 |
| 화장품법 시행규칙 | 보고 절차 및 서식 |
| 화장품 안전기준 등에 관한 규정 | 안전성·유효성 관련 정보 보고 기준 |

## 2. 이상사례(Adverse Event) 정의

**화장품 이상사례**: 화장품 사용 후 발생하는 **의도하지 않은 반응**으로, 인과관계가 확인되지 않은 경우도 포함한다.

### 분류

| 구분 | 설명 | 예시 |
|------|------|------|
| 중대한 이상사례 | 생명 위협, 입원, 영구적 장애, 선천성 이상 등 | 심각한 알레르기 반응, 전신 부작용 |
| 일반 이상사례 | 중대하지 않은 반응 | 홍조, 건조, 가려움, 트러블 |

## 3. 보고 주체

| 주체 | 보고 의무 |
|------|----------|
| 화장품 책임판매업자 | 제품 이상사례 보고 의무 |
| 맞춤형화장품 판매업자 | 맞춤형 제품 관련 이상사례 보고 |
| 소비자 | 자발적 보고 가능 (FDA 한국어 안내 포함) |

## 4. 보고 절차

### 정기 보고 (자발적 이상사례)

```
이상사례 인지 (CS 접수, 소비자 신고, SNS 모니터링 등)
       ↓
이상사례 기록 및 분류 (중대 / 일반)
       ↓
내부 평가 (인과관계 가능성 판단)
       ↓
식약처 의약품통합정보시스템(DUR) 또는 지정 시스템 보고
       ↓
필요 시 안전대책 마련 (라벨 수정, 생산 중단 등)
```

### 중대 이상사례 보고 기한

- **알게 된 날로부터 15일 이내** 식약처에 보고 (긴급 보고)
- 추가 정보 확인 후 보완 보고 가능

### 일반 이상사례 보고 기한

- 연 2회 또는 수시 보고 (식약처 지침에 따름)

## 5. 이상사례 보고 내용

보고 시 포함해야 하는 정보:

| 항목 | 내용 |
|------|------|
| 제품 정보 | 제품명, 제조번호(Lot No.), 제조일자, 유통기한 |
| 소비자 정보 | 나이, 성별, 피부 타입, 사용 부위 |
| 이상사례 내용 | 증상, 발생 시기, 지속 기간, 중증도 |
| 의료 조치 | 병원 방문 여부, 처방 내용 |
| 조치 결과 | 회복 여부, 현재 상태 |
| 인과관계 | 제품과의 관련성 판단 |

## 6. 소비자 CS 접수 시 대응 프로세스

### 이상사례 CS 유입 경로

| 경로 | 설명 |
|------|------|
| 고객센터 (전화/이메일) | 직접 신고 |
| 온라인 리뷰 (세포라, 네이버, @cosme 등) | 모니터링 필요 |
| SNS (인스타그램, 틱톡, 커뮤니티) | 실시간 모니터링 |
| 세포라·Boots 바이어 통보 | 채널 파트너 경유 |
| 규제기관 통보 | FDA, 영국 OPSS 등 |

### 내부 대응 절차

```
CS 접수
   ↓
이상사례 여부 1차 분류 (CS 팀)
   ├─ 일반 불만 → CS 처리
   └─ 이상사례 의심 → 품질팀·규제팀 에스컬레이션
          ↓
   제품 회수 및 보관 (해당 Lot 추적)
          ↓
   내부 조사 (성분, 제조 공정, 미생물 검사)
          ↓
   중대 이상사례 여부 판단
   ├─ 중대 → 15일 이내 식약처 보고
   └─ 일반 → 정기 보고 포함
          ↓
   소비자 피드백 및 보상 처리
          ↓
   재발 방지 조치 (포뮬러 검토, 라벨 수정 등)
```

## 7. 세포라·Boots 채널 CS와 연계

세포라·Boots 등 채널에서 이상사례 관련 CS가 발생할 경우:

- 채널 바이어가 브랜드사에 즉시 통보하는 경우 있음
- **브랜드사는 자체 모니터링 시스템으로 채널 리뷰를 상시 추적**해야 함
- 이상사례 급증 시 채널이 선제적으로 제품 판매 중단 또는 진열 제거 가능
- rom&nd 사례: 세포라 입점 후 CS(구순염 클레임) 급증 → 브랜드 대응 및 포뮬러 개선

## 8. 기록 보관 의무

| 항목 | 보관 기간 |
|------|----------|
| 이상사례 기록 | 최종 판매일로부터 3년 이상 |
| 제조 기록 | 제조일로부터 3년 |
| 미국(MoCRA) | 6년 (소기업 3년) |
