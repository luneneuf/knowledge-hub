# K-beauty 일본 재고관리 및 유통 구조

## 1. K-beauty 일본 물류 생태계

K-beauty 제품의 일본 유통은 전문 K-beauty 도매·물류 업체를 통한 유통망이 발달해 있다.

### 주요 유통 참여자

| 유형 | 역할 |
|------|------|
| 한국 브랜드사 | 제품 기획·제조·공급 |
| 한국 ODM사 | 생산 (코스맥스, 한국콜마 등) |
| 일본 수입사/대행사 | 통관, 세관 신고, 일본 내 반입 |
| K-beauty 전문 도매상 | 일본 내 재고 보관·배송·소매점 납품 |
| 소매점 (드럭스토어·버라이어티숍) | 최종 판매 |
| 온라인 마켓플레이스 | 라쿠텐, Amazon.co.jp, @cosme 쇼핑 |

## 2. 일본 K-beauty 전문 물류 구조

K-beauty 전문 도매·물류 업체는 일본 내 **전용 창고**를 운영하며, 한국 브랜드사를 대신해 재고를 관리한다.

### 일반적인 물류 흐름

```
한국 브랜드사 → 선적 (항공/해상)
      ↓
일본 세관 통관 (수입사/대행사 처리)
      ↓
일본 내 K-beauty 전문 창고 반입
      └─ 예: 사이타마(埼玉) 등 도쿄 인근 창고 거점
      ↓
주문 접수 (소매점 또는 온라인)
      └─ 소량 주문부터 대량 주문까지 유연 대응
      ↓
피킹(Picking) → 포장 → 배송
      └─ ERP 시스템으로 주문 추적
      ↓
드럭스토어·버라이어티숍 납품 또는 직배송
```

## 3. 재고 관리 방식

### 3-1. 수입사/도매상 보유 재고 모델

- 도매상이 일괄 수입 후 창고 보관
- 소매점 주문에 따라 분할 납품
- 브랜드사는 수입사와 벌크 납품 계약
- **브랜드사 재고 가시성 낮음** (실시간 POS 데이터 공유 제한적)

### 3-2. 위탁 재고(Consignment) 모델

- 브랜드사가 재고를 수입사에 위탁
- 판매 성과에 따라 정산
- 재고 미판매 시 브랜드사 부담
- **리스크가 크지만 도매상 입장에서 진입 장벽 낮춰 줌**

### 3-3. 드롭십(Dropship) 모델 (온라인 채널)

- 소비자 주문 → 도매상(또는 브랜드사) 창고에서 직접 배송
- 소매점 재고 없음
- 온라인 마켓플레이스 특화 (라쿠텐, Amazon.co.jp)
- MOQ 없음, 소량 주문 대응 가능

## 4. 재고 관리 핵심 지표

| 지표 | 설명 | 관리 목적 |
|------|------|----------|
| 재고 회전율 | 월 판매량 / 평균 재고량 | 과잉 재고 및 품절 방지 |
| 리드타임 | 발주 후 입고까지 소요 기간 | 재발주 시점 설정 |
| 안전재고 | 리드타임 동안의 예상 판매량 | 품절 방지 버퍼 |
| 셀스루율 | 납품 수량 대비 판매 비율 | 소매점 성과 평가 |
| 반품율 | 납품 대비 반품 비율 | CS 이슈 조기 감지 |

## 5. 재발주(리오더) 프로세스

### 수입사/도매상 경유 시

```
소매점 POS 데이터 집계 (도매상 수집)
       ↓
도매상 재고 부족 감지
       ↓
도매상 → 수입사에 추가 발주
       ↓
수입사 → 한국 브랜드사에 발주
       ↓
한국 생산 → 선적 → 통관 → 입고
       ↓ (약 3~6주 소요)
드럭스토어 보충 납품
```

**리스크**: 브랜드사가 최종 소매점 재고 수준을 직접 확인하기 어려워, **품절 또는 과재고 파악이 늦어질 수 있음.**

## 6. 시즌별 재고 관리 고려사항

| 시즌 | 특이사항 |
|------|----------|
| 골든위크 (4월 말~5월) | 외국인 관광객 급증, 인바운드 수요 대응 재고 확보 |
| 여름 (6~8월) | 선케어·수분 제품 수요 증가 |
| 연말 (12월) | 기프트셋 수요 증가, 선물 포장 SKU 준비 |
| 연초 세일 (1월) | 소진 재고 정리 시즌 |

## 7. 온라인 vs 오프라인 재고 분리 관리

일본 시장에서는 온라인(라쿠텐·아마존)과 오프라인(드럭스토어) 가격 차이가 **직구 가격과 비교 이슈**를 일으킬 수 있다.

**권장 방법**:
- 온라인용 번들 구성 (오프라인 SKU와 상이하게)
- 온라인 전용 사이즈 또는 세트 구성
- MAP(Minimum Advertised Price) 정책으로 최저 판매가 기준 설정

## 8. ERP·재고 시스템 활용

현대 K-beauty 도매상은 **자체 개발 ERP 시스템** 또는 클라우드 기반 재고 관리를 도입해 효율화하고 있다.

| 기능 | 내용 |
|------|------|
| 실시간 재고 조회 | 한국 브랜드사가 도매상 재고 조회 가능 |
| 주문 추적 | 발주부터 납품까지 진행 상황 모니터링 |
| 판매 데이터 리포팅 | 채널별 판매 실적 공유 |
| 자동 재발주 알림 | 재고 부족 시 자동 발주 트리거 |

## 9. 브랜드사 SCM 실무 체크리스트

- [ ] 수입사/도매상과 재고 가시성(Inventory Visibility) 계약 조건 확인
- [ ] 리드타임 기준 안전재고 설정 (일본 내 최소 1~2개월치)
- [ ] 시즌 피크 시즌 전 선행 발주 계획 수립
- [ ] 반품·불량 처리 계약 조건 사전 확인
- [ ] 온라인·오프라인 가격 정책 일관성 유지
