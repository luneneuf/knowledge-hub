# 일본 드럭스토어 화장품 납품 프로세스

## 1. 일본 화장품 오프라인 유통 구조

일본 화장품 오프라인 유통은 **도매상 중심 구조**다. 소매점(드럭스토어·버라이어티숍)에 직접 납품하는 것은 일반적으로 불가능하며, 반드시 화장품 도매상을 거쳐야 한다.

```
한국 브랜드사
     ↓
[일본 수입사 or 수입대행사]
     ↓ (통관·보관)
[화장품 도매상]
     ↓
[드럭스토어 / 버라이어티숍 / 홈쇼핑]
     ↓
[소비자]
```

### 주요 소매 채널

| 채널 유형 | 대표 업체 |
|----------|---------|
| 드럭스토어 | 마츠모토키요시(マツモトキヨシ), 스기약국(スギ薬局), 토모스(Tomod's), 코코카라파인 |
| 버라이어티숍 | 도큐핸즈(東急ハンズ), 로프트(Loft), 앳코스메(@cosme), 플라자(PLAZA) |
| TV홈쇼핑 | 샵채널(ShopChannel), QVC, 자파넷(Japanet) |
| 온라인 | 라쿠텐, Amazon.co.jp, @cosme 쇼핑 |

## 2. 납품 프로세스 단계별 설명

### 총 소요 기간: 약 5~6개월 (빠른 경우)

```
1단계: 초기 접촉 (약 1~2개월)
   └─ 이메일·전화로 수입사 또는 도매상에 회사 소개
   └─ 제품 샘플 발송 (유상/무상 협의)
   └─ 주의: 해외 바이어 전용 연락처가 없는 경우 회신 어려움

2단계: 시장성 평가 (약 1~2개월)
   └─ 도매상·수입사 직영점포에서 테스트 판매
   └─ 점장 평가 기반 시장성 판단
   └─ 소비자 수요 예측 → 진행 여부 결정

3단계: 도매상 협의 (약 1개월)
   └─ 공급가 협의 (역산 방식)
   └─ 판매점 바이어·도매상과 유통 조건 협상
   └─ 배타적 유통권 여부 논의

4단계: 현지화 작업 (약 1~2개월)
   └─ 패키지 일본어 전성분 표기 (JCIA 기준 명칭)
   └─ 판매원(수입사) 표기
   └─ 집기(POP) 디자인 제작 (종이 재질 권장)
   └─ 테스트 샘플 제품 비치 계획

5단계: 마케팅 준비 및 론칭
   └─ SNS: YouTube, Instagram, TikTok
   └─ 일본 인플루언서 협업 (전속 또는 제휴)
   └─ 패션잡지 기사화
   └─ TV 방송 소개 검토
```

## 3. 가격 설정 방식 (역산법)

```
최종 소비자 판매가 (소매가)
   ↓ 소매 마진 차감 (약 30~40%)
도매상 납품가
   ↓ 도매 마진 차감 (약 15~25%)
수입사 공급가
   ↓ 물류비·통관비·마진 차감
한국 브랜드 공급가(FCA 또는 CIF 가격)
```

**핵심**: "판매가 설정 후 역산으로 물류 통관비 등을 감안한 공급가 설정"이 필요하며, 마진 구조를 모르고 접근하면 협상이 어려워진다.

## 4. 집기(POP) 전략

일본 드럭스토어는 **셀프판매(Self-service)** 방식이어서 집기와 판촉물의 역할이 크다.

| 항목 | 내용 |
|------|------|
| 집기 소재 | 종이 재질 권장 (아크릴보다 환경 친화적으로 인식) |
| 테스트 제품 | 반드시 비치 (소비자가 직접 체험 가능해야 구매 결정) |
| 패키지 현지화 | 인터넷 직구와 차별화, 가격 붕괴 방지 |
| POP 언어 | 일본어 완비 필수 |

## 5. K-beauty 일본 시장 현황

| 항목 | 데이터 |
|------|--------|
| 한국 화장품 일본 수입액 | 2019년 약 \$4.5M → 2020년 약 \$5.5M (YoY +22%) |
| 일본 내 랭킹 변화 | 2019년 수입국 4위 → 2020년 2위 |
| 핵심 소비층 | 10~30대 여성 |
| 구매 이유 1위 | 저렴한 가격 (전 연령) |
| 구매 이유 2위 | 고품질 (20~30대 우선) |
| 인기 제품 | 립 틴트, 쉐딩, 스킨케어 |

## 6. 마케팅 패턴별 전략

### 패턴 A: 일본 내 무명 브랜드
- 현지화 후 6개월 이상 장기 마케팅으로 브랜드 인지도 구축
- SNS 인플루언서 전속 계약으로 지속적 노출

### 패턴 B: SNS에서 이미 알려진 브랜드
- 일본 소비자가 이미 인식 → 빠른 유통 진입 가능
- 직구 수요를 오프라인으로 전환하는 전략

## 7. 실무 주의사항

| 항목 | 주의사항 |
|------|----------|
| 해외 바이어 연락 | 별도 해외 담당자 연락처 필수 (일반 대표 번호로 연결 안 됨) |
| 성분 확인 | 한국 허용 성분도 일본 금지 가능성 있음 (사전 검토 필수) |
| 가격 붕괴 방지 | 한국 직구 가격과의 차이를 패키지 현지화로 관리 |
| 재고 과잉 주의 | 첫 입고는 소량으로 시장성 검증 후 확대 |
| 도매상 마진 | 예상보다 높을 수 있어 공급가 사전 시뮬레이션 필수 |
