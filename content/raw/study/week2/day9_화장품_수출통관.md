# 화장품 미국·영국 수출 통관 절차 (AIR·SEA)

## 1. 수출 통관 전체 흐름

화장품 수출은 한국 세관에서 **수출 통관**과 수입국 세관에서 **수입 통관**, 두 번의 통관이 필요하다.

```
[한국 제조사 출고]
     ↓
[포워더(물류 주선업체) 선정 및 선적 의뢰]
     ↓
[수출 신고 → 한국 세관 → 수출신고필증(수출면장) 발급]
     ↓
[선적: AIR (항공) 또는 SEA (해상)]
     ↓
[수입국 도착 → 현지 통관 브로커 수입 신고]
     ↓
[수입국 세관 심사 + 규제기관(FDA 등) 스크리닝]
     ↓
[통관 허가 → 현지 창고 반입 → 유통사/채널 납품]
```

## 2. 필수 통관 서류

### 기본 서류 (공통)

| 서류명 | 설명 |
|--------|------|
| Commercial Invoice (상업송장) | 품목명, 수량, 단가, 총금액, 발송인·수신인 |
| Packing List (포장명세서) | 박스별 수량, 중량, 부피 |
| B/L 또는 Airway Bill | 해상: 선하증권(Bill of Lading), 항공: 항공화물운송장(AWB) |
| Certificate of Origin (원산지증명서) | 한국산 표시, FTA 적용 시 협정 원산지증명서 |
| 성분표 (Ingredient List) | 국가별 규제 심사용 |
| 제조공정도 | 일부 국가 요청 시 |
| 보험증명서 | CIF 조건인 경우 |

### 미국 추가 서류

| 서류명 | 설명 |
|--------|------|
| FDA 시설 등록번호 | 제조사 FDA 등록 확인 |
| Product Listing 확인 | FDA Cosmetics Direct 등록 여부 |
| 영문 라벨 샘플 | FDA 기준 충족 여부 확인 |
| MSDS (물질안전보건자료) | 위험물 분류 해당 시 |

### 영국 추가 서류

| 서류명 | 설명 |
|--------|------|
| SCPN 신고 확인서 | 신고번호 포함 |
| Responsible Person 지정 서류 | 영국 RP 계약서 |
| PIF 요약본 | 요청 시 제공 |

## 3. AIR vs SEA 비교

| 항목 | 항공 (AIR) | 해상 (SEA) |
|------|-----------|-----------|
| 운송 서류 | 항공화물운송장(AWB) | 선하증권(B/L) |
| 소요 기간 (한국→미국) | 2~5일 | 15~25일 |
| 소요 기간 (한국→영국) | 3~7일 | 25~35일 |
| 비용 | 높음 (kg당 과금) | 낮음 (CBM 또는 TEU 단위) |
| 적합한 경우 | 소량·고가 제품, 긴급 납품, 초기 테스트 물량 | 대량 발주, 정기 납품 |
| 위험물 규정 | IATA 규정 (에어로졸·인화성 제한 엄격) | IMDG Code (위험물 선박 운송 기준) |

### 화장품 위험물 분류 주의

일부 화장품은 인화성 성분(알코올 함유 토너, 향수 등)을 포함해 **위험물(DG, Dangerous Goods)**로 분류될 수 있다.

- 항공: IATA 규정 준수 (LQ — Limited Quantity 포장 가능 여부 확인)
- 해상: IMDG Code 분류 → 특수 선박 배치 또는 포장 기준 준수

## 4. 미국 통관 심사 구조

```
CBP(세관국경보호청) Entry 신고
       ↓
약식통관 (Informal Entry)
└─ Invoice 금액 \$2,500 미만 → 빠른 처리

정식통관 (Formal Entry)
└─ Invoice 금액 \$2,500 이상 → CBP + FDA 스크리닝
       ↓
FDA PREDICT 시스템 자동 스크리닝
├─ [통과] → 통관 허가
└─ [플래그] → 검사 또는 억류
       ↓
억류 시: 30일 이내 이의 신청 또는 반송/폐기
```

## 5. 영국 통관 심사 구조

```
HMRC(영국 세관) Entry 신고
       ↓
관세 납부 (한-영 FTA 적용 시 0% 또는 감세)
       ↓
화장품 규제 확인 (SCPN 신고 여부)
       ↓
통관 허가 → 영국 창고 반입
       ↓
세포라 또는 Boots 물류 창고 납품
```

### 한-영 FTA (UK-Korea FTA)

- 브렉시트 이후 한국과 영국은 별도 FTA 체결 (기존 한-EU FTA 계승)
- 화장품 관세: FTA 적용 시 0~5% 수준 (품목에 따라 상이)
- 원산지증명서 필수 (FTA 혜택 적용 시)

## 6. 포워더 선정 기준

화장품 수출은 **화장품 전문 포워더** 선정을 권장한다.

| 체크 항목 | 이유 |
|----------|------|
| 화장품 통관 경험 | 국가별 라벨·성분 규제 숙지 |
| FDA·SCPN 서류 지원 가능 | 서류 미비 시 통관 지연 |
| 위험물(DG) 처리 능력 | 알코올 함유 제품 등 |
| 현지 통관 브로커 네트워크 | 미국·영국 현지 연계 |
| 트래킹 시스템 | 납기 관리용 |

## 7. 납기 역산 (브랜드사 SCM 기준)

세포라 판매 시작일 기준 역산 예시:

| 단계 | 소요 기간 |
|------|----------|
| 제조사 생산 완료 | D-Day |
| 출고·선적 준비 | +3~5일 |
| 해상 운송 (한국→미국) | +20일 |
| 미국 통관 | +5~10일 |
| 창고 반입·검수 | +3~5일 |
| 세포라 입고 처리 | +5~10일 |
| **판매 시작 가능** | **D + 약 36~50일** |

> 항공 이용 시 총 소요 기간 약 15~25일로 단축 가능. 단, 비용 대폭 증가.
