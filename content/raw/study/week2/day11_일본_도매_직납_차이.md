# 일본 화장품 유통 도매 vs 직납 차이

## 1. 일본 유통의 기본 원칙

일본 화장품 유통은 **도매상 의존 구조**가 강하다. 소비자 90% 이상이 오프라인 매장에서 직접 구매를 선호하며, 전자상거래 비율이 낮다(2022년 기준 화장품·의약품 EC화율 8.24%).

| 구분 | 내용 |
|------|------|
| 전체 소비시장 EC화율 (2022) | 9.13% |
| 화장품·의약품 EC화율 (2022) | 8.24% |
| 오프라인 구매 선호율 | 약 90% 이상 |

## 2. 도매 경유(토오시) vs 직납(직접 납품) 구조

### 2-1. 도매 경유 구조 (일반적 방식)

```
브랜드사(한국)
     ↓ 공급가 (수출가)
수입사 / 수입대행사
     ↓ 보세창고 보관 → 통관
화장품 도매상 (지역 도매상 포함)
     ↓ 배송·선반 관리
드럭스토어 본사 → 각 매장
     ↓
소비자
```

**특징**:
- 도매상이 물류·결제·반품 위험 흡수
- 브랜드사는 수입사 한 곳과만 거래
- 단가 구조에서 도매 마진(15~25%) + 소매 마진(30~40%) 적용
- 브랜드사의 최종 소비자 가격 컨트롤이 어려움

### 2-2. 직납(直納) 구조

```
브랜드사(한국)
     ↓
일본 법인 or 현지 세일즈 팀
     ↓ 직접 수발주
드럭스토어 본사 물류센터
     ↓
각 매장
```

**특징**:
- 도매 마진 제거로 가격 경쟁력 확보
- 소매점과 직접 관계 유지 → 제품 정보·마케팅 주도권
- 재고·반품 관리 직접 담당 → 리스크 증가
- 일본 현지 법인 또는 세일즈 팀 필수 (高 진입 장벽)
- 대형 체인(마츠모토키요시, 코코카라파인 등)과의 협상력 필요

## 3. 도매 vs 직납 비교표

| 항목 | 도매 경유 | 직납 |
|------|----------|------|
| 진입 장벽 | 낮음 | 높음 (현지 법인 필요) |
| 가격 경쟁력 | 도매 마진 부담 | 도매 마진 절약 |
| 재고 관리 | 도매상이 담당 | 브랜드사 직접 담당 |
| 판매 데이터 접근 | 제한적 (수입사 경유) | 실시간 POS 데이터 접근 가능 |
| 반품 처리 | 도매상이 흡수 | 브랜드사 직접 부담 |
| 소매점 관계 | 간접적 | 직접적 |
| 마케팅 주도권 | 낮음 | 높음 |
| 적합 단계 | 초기 진입 | 안정적 매출 확보 후 |

## 4. 실제 유통 단계별 마진 구조 (추정)

| 단계 | 마진율 | 누적 가격 인상 |
|------|--------|-------------|
| 한국 브랜드 수출가 (FOB) | - | 기준 100 |
| 수입사 마진 | 10~15% | 115 |
| 도매상 마진 | 15~25% | 140 |
| 소매점 마진 | 30~40% | 195~200 |
| 최종 소비자 가격 | - | 약 2배 |

> 한국 소비자 판매가 1만원 제품이 일본에서는 2,000~2,500엔(약 2만~2만5천원)에 판매되는 구조.

## 5. 가격 붕괴(Price Erosion) 문제

일본 소비자의 해외 직구 증가로 **한국 판매가격이 그대로 노출**되면 일본 오프라인 판매 가격과 격차가 발생, 오프라인 바이어의 반발 또는 가격 인하 압박이 생긴다.

**해결 방법**:
- 일본 전용 패키지(현지화 SKU)로 직구 제품과 외관 차별화
- 일본 전용 용량 또는 구성품 변경
- 온라인 직구 가격 관리 (MAP 정책)

## 6. 브랜드 규모별 유통 전략

| 단계 | 전략 |
|------|------|
| 초기 진입 | 수입대행사 통한 도매 유통 → 드럭스토어 테스트 |
| 성장기 | 주요 도매상 파트너십 확대 → 채널 다양화 |
| 성숙기 | 직납 전환 검토 → 일본 법인 설립 또는 독점 수입사 계약 |

## 7. 일본 주요 화장품 도매상 (참고)

- **コーセー物流(코세이 물류)**: 코세이 그룹 계열 도매
- **ゼファー(Zephyr)**: K-뷰티 특화 수입·도매
- **トーカドー(Tokado)**: 드럭스토어 전문 도매

> 실제 거래 도매상은 제품 카테고리·지역에 따라 다르며, KOTRA 도쿄무역관 상담을 통해 매칭 가능.
