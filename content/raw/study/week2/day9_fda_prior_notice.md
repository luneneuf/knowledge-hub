# FDA Prior Notice 제도 및 한국 화장품 미국 수출 규제

## 1. FDA Prior Notice 개요

**Prior Notice(사전 통보)**는 미국 FDA가 수입 식품·의약품·화장품에 대해 요구하는 사전 신고 제도다. 화장품의 경우 통관 과정에서 FDA PREDICT 시스템을 통해 자동 스크리닝된다.

### 화장품에서 Prior Notice가 작동하는 방식

```
수출자(한국) → 미국 CBP(세관국경보호청) 전자 신고 (Customs Entry)
      ↓
FDA PREDICT 시스템 자동 스크리닝
      ↓
├─ [통과] → 통관 허가
└─ [플래그] → 물리적 검사 (Physical Examination) 또는 서류 심사
```

> 화장품은 FDA Prior Notice가 식품처럼 별도 의무 제출 항목이 아니지만, 통관 과정에서 FDA가 스크리닝하며, 문제 발견 시 억류(Detention) 조치를 취한다.

## 2. MoCRA 핵심 요건 (2022년 제정, 2023년 의무화)

**MoCRA(Modernization of Cosmetics Regulation Act of 2022)**는 미국 화장품 역사상 가장 큰 규제 변화다.

| 요건 | 세부 내용 | 적용 시점 |
|------|----------|----------|
| **시설 등록** (Facility Registration) | 한국 제조사(코스맥스, 한국콜마 등) FDA에 등록 의무, 2년마다 갱신 | 의무화 |
| **제품 등록** (Product Listing) | 성분 목록을 FDA Cosmetics Direct에 제출, 연간 업데이트 | 의무화 |
| **이상사례 보고** (Adverse Event Reporting) | 심각한 부작용 15 업무일 이내 FDA 보고 | 의무화 |
| **안전성 입증** (Safety Substantiation) | 안정성·미생물 검사·안전 데이터 보유 의무 | 의무화 |
| **GMP 준수** | FDA GMP 기준 준수 | 의무화 |

### 소기업 면제 (Small Business Exemption)
- 미국 화장품 연평균 매출 **\$1M 미만** 브랜드는 시설 등록·제품 등록 면제 가능
- **단, 이상사례 보고·라벨링·안전성 입증은 면제 없음**

## 3. 라벨링 요건 (필수 영문 표기)

| 항목 | 요건 |
|------|------|
| 제품 명칭 | 영문 |
| 순 내용량 | fl oz 또는 ml (이중 표기) |
| 브랜드명 및 배포사 주소 | 미국 Responsible Person 주소 포함 |
| 성분 목록 | INCI명, 함량 내림차순 (1% 이하는 순서 무관) |
| 경고문구 | 해당 시 필수 |
| 이상사례 보고 연락처 | MoCRA 이후 라벨에 포함 의무 |

> 한국 패키지를 그대로 사용하면 미국 기준 미달이다. 별도 영문 라벨 또는 스티커 부착 필요.

## 4. 화장품 vs 의약품 분류 — 클레임 주의

클레임 문구에 따라 화장품이 '의약품'으로 재분류될 수 있다.

| 표현 예시 | 분류 | 결과 |
|----------|------|------|
| "모이스처라이징" | 화장품 | 일반 통관 |
| "콜라겐 생성 촉진" | 의약품 클레임 | 사전 FDA 승인 필요 |
| "주름 감소" | 의약품 클레임 | FDA 승인 필요 |
| "자외선 차단 SPF" | 의약품(선스크린) | 별도 OTC 의약품 절차 |

## 5. 통관 억류 주요 원인

| 억류 원인 | 설명 |
|----------|------|
| 라벨 부적합 | 영문 미표기, 성분 미표기 |
| 의약품 클레임 | 화장품에 의약 효능 표현 |
| 금지 색소 | FDA 미승인 색소 사용 |
| 시설 미등록 | 제조사 FDA 등록 누락 |
| 금지 성분 | 수은 화합물, 클로로폼 등 |

## 6. 수입 통관 프로세스

```
1. 선적 준비 (한국)
   └─ FDA 등록 서류 확인, 라벨 검토

2. 수출 신고 (한국 세관)
   └─ 수출신고필증 발급

3. 미국 도착
   └─ CBP Customs Entry 전자 신고

4. FDA PREDICT 스크리닝 (자동)
   ├─ 통과 → 통관
   └─ 플래그 → 서류 심사 또는 물리적 검사

5. 세관 허가 후 창고 반입 → 세포라·유통사 납품
```

## 7. 브랜드사 사전 준비 체크리스트

- [ ] 한국 ODM 제조사 FDA 시설 등록 확인
- [ ] 제품 FDA Cosmetics Direct 등록 완료
- [ ] 영문 라벨 FDA 기준 충족 검토
- [ ] 성분 중 미국 금지 성분 여부 확인
- [ ] 선스크린 성분 포함 여부 → 별도 OTC 의약품 절차 확인
- [ ] 이상사례 보고 시스템 구축
- [ ] 미국 화장품 전문 통관 브로커 선정
