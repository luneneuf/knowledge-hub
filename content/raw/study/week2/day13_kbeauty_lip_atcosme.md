# K-beauty 립 제품 @cosme 일본 리뷰 분석

**@cosme(앳코스메)**는 일본 최대 화장품 리뷰 사이트로, 월간 수천만 명의 소비자가 리뷰를 작성·참고한다.

| 항목 | 내용 |
|------|------|
| 플랫폼 | @cosme (cosme.net) |
| 특징 | 소비자 구매 후기 중심, 별점·성분·효과 분류 |
| K-beauty 비중 | K-beauty 제품 카테고리 별도 운영 |
| 입점 방법 | @cosme 공식 스토어(@cosme SHOPPING) 입점 가능 |

| 분류 | 설명 |
|------|------|

| 효과별 | "보습", "발색", "지속력" 등 개별 평가 |

| 인기 랭킹 | 일간·주간·월간 랭킹 업데이트 |

### @cosme 사용자 사례 (실제 Q&A 기반)

한 사용자(ヲンム)가 공유한 사례:

**증상**: 원래 민감하지 않았던 입술이 갑자기 다양한 립 제품에 반응하기 시작

**반응 증상**:
- 붉어짐(赤み)과 부어오름(腫れ)
- 심한 경우 뾰루지(ぷつぷつ)
- 사용 3일 후 자연 회복

**반응한 브랜드/제품** (사용자 보고):
- キャンメイク(Canmake)
- エテュセ(Ettusais)
- ロムアンド(rom&nd) ← K-beauty
- KATE
- キュレル(Curel)
- ニベア(Nivea)
- メンソレータム(Mentholatum)

**반응하지 않은 제품**:
- Dior マキシマイザー — "全然荒れない(전혀 반응 안 함)"

**배경**: 해당 소비자는 키위·복숭아·파인애플·딸기·멜론·오이 알레르기가 있으며, 의사로부터 알레르기 반응 진단.

**시사점**: 알레르기 기저 질환이 있는 소비자에서 K-beauty 립 제품 클레임 발생 가능성 있음.
