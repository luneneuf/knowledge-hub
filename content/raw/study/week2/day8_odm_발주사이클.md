# 화장품 브랜드사 SCM ODM 발주 사이클 실무

## 1. SCM 전체 흐름

화장품 브랜드사의 SCM(Supply Chain Management)은 제품 기획 단계부터 소비자 판매까지의 전 과정을 포함한다.

```
[제품 기획] → [ODM/OEM 제조사 선정] → [샘플 개발] → [발주]
     → [원료 구매·생산] → [QC] → [납품] → [물류·창고]
          → [채널 배송] → [재고 관리] → [재발주 판단]
```

## 2. 발주 사이클 단계별 실무

### 2-1. 제조사 선정 기준

| 기준 | 체크 항목 |
|------|----------|
| 생산 능력 | MOQ, 최대 생산량, 자동화 수준 |
| 포뮬러 역량 | 특허 원료, 독점 처방 보유 여부 |
| 납기 신뢰도 | 평균 리드타임, 지연 이력 |
| 품질관리 | GMP 인증, QC 시스템 |
| 가격 구조 | 개발비·생산비·패키지비 분리 여부 |
| 수출 대응 | FDA, SCPN, CPNP 서류 지원 가능 여부 |

### 2-2. ODM vs OEM 선택 기준

| 항목 | ODM | OEM |
|------|-----|-----|
| 제품 기획 주체 | 제조사 | 브랜드사 |
| 개발 기간 | 짧음 (기존 포뮬러 활용) | 길음 (맞춤 개발) |
| 차별화 | 낮음 (유사 제품 가능) | 높음 |
| 개발비 | 일반적으로 일괄 포함 | 별도 청구 가능 |
| 비용 | 낮음 | 높음 |
| 권장 상황 | 빠른 신제품 출시, 소규모 브랜드 | 독창 제품 개발, 대형 브랜드 |

### 2-3. 발주서 발행 전 확인사항

- **MOQ 충족 여부**: 용기 MOQ(2,000~5,000개)와 충진 MOQ 별도 확인
- **납기 역산**: 판매 시작일로부터 리드타임(45~90일) + 통관 기간(수출 시 +2~4주) 역산
- **비용 내역 분리**: 개발비 / 생산비 / 패키지비 / 운송비 별도 확인
- **결제 조건**: 선급금 비율(T/T 30% deposit 일반적), 잔금 납기 조건
- **변경 조건**: 수량·포뮬러 변경 시 추가 비용 및 납기 지연 조건 계약서 명시

## 3. MOQ 결정 구조

화장품 MOQ는 **용기 MOQ가 가장 큰 제약**이 된다.

```
용기 MOQ (2,000~5,000개) ← 가장 높은 제약
   ↓
충진(내용물) MOQ
   ↓
박스·포장 MOQ (상대적으로 유연)
   ↓
발주 가능 최소 수량 = 위 항목 중 가장 높은 수량
```

### 제품별 MOQ 예시

| 제품 유형 | 일반 MOQ | 소량 생산 |
|----------|---------|----------|
| 스킨케어 (대형) | 10,000개 이상 | 1,000개 (소량 ODM 서비스) |
| 립 틴트 (색상별) | 5,000개 내외 | - |
| 마스크팩 | 30,000개 이상 | - |
| 파우치 | 25,000장 이상 | - |

> 소규모 브랜드는 코스맥스 마이크로 론칭(1,000개~) 또는 셀프코스 등 소량 전문 플랫폼 활용 검토 필요.

## 4. 재발주(리오더) 판단 기준

브랜드사 SCM 담당자가 재발주를 결정하는 주요 기준:

| 지표 | 설명 |
|------|------|
| 재고 회전율 | 채널별 월 판매량 대비 재고 수준 |
| 셀스루율 | 발주 수량 대비 실 판매 수량 비율 |
| 리드타임 | 재발주 시점 = 안전재고 ÷ 일평균판매량 이상 남을 때 |
| 채널 요청 | 세포라·Boots 등 바이어의 리오더 요청 |
| 프로모션 계획 | 시즌 행사·캠페인 전 선행 발주 |

### 안전재고 공식 (예시)

```
안전재고 = 일평균판매량 × 리드타임(일) × 안전계수(1.2~1.5)
재발주점  = 안전재고 + 일평균판매량 × 리드타임(일)
```

## 5. 수출 채널별 추가 고려사항

| 채널/시장 | 추가 요건 |
|----------|----------|
| 세포라 (미국) | FDA MoCRA 등록, 영문 라벨, 미국 내 RP 지정 |
| 세포라 (영국) | SCPN 신고, 영국 RP 지정, 영문 PIF |
| 일본 드럭스토어 | 약사법 성분 적합 확인, 일본어 전성분 표기 |
| 영국 Boots | UK SCPN 신고 완료, 35일 반품 정책 대응 재고 |

## 6. 브랜드사 SCM 담당자 연간 캘린더 (예시)

| 시기 | 주요 업무 |
|------|----------|
| Q4 전년도 | 다음 해 상품 기획·ODM사 협의 시작 |
| Q1 | 샘플 개발·승인 → 발주서 발행 |
| Q2 | 생산 진행·QC → 채널 납품 |
| Q3 | 판매 모니터링 → 재발주 판단 |
| Q4 | 시즌 프로모션 대응 재고 확보 |
