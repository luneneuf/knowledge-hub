# Boots UK CS 정책 (반품·교환·고객 서비스)

## 1. 반품 기본 정책

| 항목 | 내용 |
|------|------|
| 반품 기간 | 구매 후 **35일 이내** |
| 환불 대상 | 미개봉·미사용 제품 (영수증 지참 시) |
| 개봉 화장품 반품 | **원칙적으로 불가** (결함 있는 경우 예외) |
| 개봉 메이크업 교환 | **35일 이내 교환 가능** (Make-up Exchange Promise) |
| 환불 처리 기간 | **14 업무일** 이내 |

## 2. 반품 경로

| 구매 방법 | 반품 방법 |
|----------|---------|
| 매장 구매 | 영국 내 모든 Boots 매장 방문 |
| 온라인 (boots.com) | 우편 무료 반송 (Royal Mail) 또는 매장 방문 |
| 클릭 & 콜렉트 | 매장 방문 반품 가능 |

## 3. Make-up Exchange Promise (업계 최초 정책)

Boots는 영국 주요 뷰티 리테일러 중 **최초로 개봉한 메이크업 제품도 교환 가능**한 정책을 도입했다.

| 항목 | 내용 |
|------|------|
| 정책명 | Make-up Exchange Promise |
| 적용 범위 | 개봉 메이크업 제품 |
| 기간 | 35일 이내 |
| 교환 가능 여부 | **교환 가능** (결함 없어도) |
| 환불 가능 여부 | **환불 불가** (교환만 가능) |
| 온라인 구매 | **해당 없음** (오프라인 매장만 적용) |
| 도입 배경 | 소비자가 색상·질감을 매장에서 완전히 판단하기 어려운 특성 반영 |

> 예: Boots에서 립스틱을 구입해 사용해보니 자신의 피부 톤에 맞지 않는 경우, 35일 이내 다른 제품으로 교환 가능.

## 4. 화장품 반품 불가 예외사항

다음 제품은 반품·환불·교환 불가:

- 의약품 (Medicines)
- 식품 (Food)
- **개봉한 화장품** (단, Make-up Exchange Promise 적용 제품 제외)
- 위생상 이유로 반품이 부적절한 제품 (일부)

## 7. Boots Advantage Card (로열티 프로그램)

세포라 Beauty Insider에 해당하는 Boots의 로열티 프로그램이다.

| 항목 | 내용 |
|------|------|
| 포인트 적립 | \£1 지출 = 4포인트 |
| 포인트 사용 | 100포인트 = \£1 (매장·온라인 사용 가능) |
| 추가 혜택 | 회원 전용 할인, 더블 포인트 이벤트, 생일 쿠폰 |
| CS 연계 | 반품 시 Advantage Card 번호로 구매 이력 확인 가능 |

## 8. 브랜드사 관점: Boots 반품 정책의 시사점

### 반품율이 높아지는 상황

- Make-up Exchange Promise는 **소비자 리스크를 낮춰 구매를 촉진**하지만
- 브랜드 입장에서 **교환 수요 대비 재고 확보 계획** 필요
- 교환 제품이 다른 브랜드로 이동할 가능성 있음 (교환 대상 제한 없음)

### CS 클레임 관리

- Boots 매장 CS 담당자는 이상사례 관련 클레임을 바이어에게 보고
- **온라인 boots.com 리뷰 모니터링** 필수 (반품 사유 패턴 파악)
- 이상사례 급증 시 Boots 바이어가 브랜드사에 대응 요청 가능

### 납품 브랜드의 반품 비용 처리

- 반품된 제품의 처리(폐기 또는 반송) 비용은 계약에 명시
- 화장품 특성상 대부분 개봉 제품은 재판매 불가 → 폐기 처리 비용 브랜드 부담 가능성

## 9. 온라인 반품 절차 (boots.com 구매)

```
1. boots.com/contact-us 접속 → 반품 신청
   또는
   반송 양식 다운로드 (assets.boots.com)

2. Royal Mail 무료 반송 레이블 출력

3. 원래 포장에 제품과 영수증 동봉

4. 가까운 우체국에서 발송

5. Boots 수령 후 14 업무일 이내 환불 처리
```
