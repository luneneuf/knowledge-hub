# 영국 SCPN(Submit Cosmetic Product Notification) 신고 절차

## 1. SCPN이란?

SCPN(Submit a Cosmetic Product Notification)은 **영국 Great Britain(잉글랜드·스코틀랜드·웨일즈)에서 화장품을 유통하기 위해 반드시 거쳐야 하는 국가 온라인 신고 시스템**이다.

- 브렉시트 이후 EU CPNP(Cosmetic Products Notification Portal)와 분리·운영
- 공식 포털: [submit.cosmetic-product-notifications.service.gov.uk](https://submit.cosmetic-product-notifications.service.gov.uk/)
- 신고 없이 제품 판매 시 법적 제재 (무제한 벌금 + 최대 3개월 징역 가능)

> **북아일랜드는 별도**: 브렉시트 조약에 따라 북아일랜드에서는 EU CPNP 적용. SCPN과 구분 필요.

## 2. 누가 신고하는가?

**영국 소재 책임자(Responsible Person, RP)**만 신고 가능하다.

| RP가 될 수 있는 주체 | 조건 |
|--------------------|------|
| 영국 제조업체 | 영국 내 생산 |
| 영국 수입업체 | 영국 외에서 제품 반입 |
| 영국 내 유통업체 | 자체 브랜드로 제품 판매 |
| 지정 대행사/개인 | 제조사 또는 수입자가 위임한 영국 법인·개인 |

> 한국 브랜드(LAKA 등)는 영국 내 RP를 지정해야 한다. 규제 컨설팅 업체에 RP 대행을 맡기는 방식이 일반적.

## 3. 신고 프로세스

```
1. 영국 내 Responsible Person 지정
   └─ 제조사 또는 수입자가 RP 계약 체결

2. PIF(Product Information File) 준비
   └─ 영어로 작성, RP 주소지에 보관
   └─ 마지막 배치 출시 후 10년 보관 의무

3. SCPN 포털 계정 생성
   └─ gov.uk 계정 필요

4. 제품 정보 입력 및 제출
   └─ 아래 '필수 제출 정보' 참조

5. 신고 확인서 수령
   └─ 고유 SCPN 신고 번호 + PDF 신고 리포트 발급

6. 제품 판매 시작 가능
```

## 4. 필수 제출 정보

| 항목 | 내용 |
|------|------|
| 제품명 및 카테고리 | 제품 이름, 제품 유형(색조/스킨케어 등) |
| 제품 물리적 형태 | 액상, 고형, 분말 등 |
| 포장 유형 | 용기 종류 |
| Responsible Person | 이름, 영국 주소, 연락처 |
| PIF 보관 주소 | 서류 위치 |
| 긴급 연락처 | 의료 전문가용 성분 긴급 조회 연락처 |
| 나노물질 여부 | 나노 성분 포함 시 추가 정보 |
| CMR 물질 여부 | 발암성·돌연변이·생식독성 성분 CAS 번호 포함 |
| 전성분 조성 | INCI명 + 농도(%) |
| 라벨 이미지 | 최종 라벨 이미지 |
| 포장 사진 | 제품 외관 사진 |

## 5. 라벨링 요건 (영국 기준)

| 항목 | 요건 |
|------|------|
| Responsible Person 표기 | 이름 + 영국 주소 |
| 원산지 | 실제 제조국 (Made in EU 불가) |
| 내용량 | 수량 표기 |
| 유통기한 | 유통기한 또는 PAO(개봉 후 사용기한) |
| 성분 | INCI명 (영어) |
| 경고문구 | 해당 시 필수 |
| PIF 접근 | "Product information file available at [주소]" |

## 6. PIF (Product Information File) 구성

PIF는 SCPN 신고의 근거 서류로 RP가 보관해야 한다.

| PIF 포함 항목 |
|--------------|
| 제품 설명 |
| 화장품 안전 리포트(Cosmetic Safety Report, CSR) |
| 제조 방법 및 GMP 준수 선언 |
| 효능 주장 근거 자료 |
| 비임상 데이터 (동물실험 여부 포함) |
| 인체 적용 시험 자료 (있는 경우) |

## 7. SCPN vs EU CPNP 비교

| 항목 | 영국 SCPN | EU CPNP |
|------|----------|---------|
| 적용 지역 | Great Britain (GB) | EU 27개국 + 북아일랜드 |
| 시스템 | gov.uk SCPN 포털 | EU CPNP 포털 |
| RP 요건 | 영국 내 소재 | EU 내 소재 |
| 서로 인정 | X (별도 신고 필요) | - |
| 미신고 제재 | 무제한 벌금 (잉글랜드/웨일즈), £5,000 이하 (스코틀랜드/북아일랜드) + 최대 3개월 징역 | EU 각국별 |

## 8. 브랜드사 실무 체크리스트

- [ ] 영국 Responsible Person 계약 (규제 컨설팅사 또는 현지 수입사)
- [ ] PIF 영어 작성 완료 (화장품 안전사 서명 포함)
- [ ] SCPN 포털 신고 완료 → 신고번호 취득
- [ ] 라벨에 영국 RP 주소 반영 (UK SKU 별도 제작)
- [ ] Made in Korea 원산지 표기 확인
- [ ] PAO 심볼 표기 확인
- [ ] 금지 성분 UK 기준 검토 (EU 기준과 대부분 동일하나 일부 차이)
