# 세포라 셀스루(Sell-Through)·바이어 리뷰 구조

## 1. 세포라 바이어 구조

세포라(Sephora)의 바이어는 **카테고리별로 구분**되어 있으며, 미국 본사는 샌프란시스코에 위치한다.

- 바이어는 제품 선정, 재발주 수량, 프로모션 배치 결정을 담당
- **평균 재직 기간이 6개월~1년으로 짧아** 담당 바이어가 자주 바뀜
- 새 바이어가 부임하면 기존 계약 브랜드도 재평가 대상이 될 수 있음
- 바이어는 셀스루 목표 달성 압박이 강해 **신규 브랜드에 보수적**

## 2. 셀스루율(Sell-Through Rate) 개념

셀스루율(STR)은 **공급된 수량 대비 실제 판매된 수량의 비율**이다.

```
셀스루율(%) = (판매 수량 / 공급 수량) × 100
```

### 세포라 내부 기준 (업계 추정)

| 기간 | 셀스루 기대치 | 결과 |
|------|------------|------|
| 론칭 후 첫 6개월 | 60~70% 이상 | 재발주·확장 검토 |
| 6개월 내 60% 미달 | - | 바이어 재평가, 진열 축소 가능성 |
| 12개월 내 저조 | - | 디리스팅(Delisting) 위험 |

> 공식 수치는 비공개이나 업계에서는 첫 6개월 내 최소 60% STR을 기준으로 본다.

## 3. 바이어 리뷰 사이클

```
[론칭]
   ↓
[첫 6개월 성과 모니터링]
   └─ 일별 판매 데이터 (POS 데이터)
   └─ 온라인 리뷰·평점
   └─ 채널 트래픽 데이터

[6개월차 바이어 리뷰]
   ├─ 셀스루율 기준 달성 → 재발주 / 도어 확장 검토
   └─ 기준 미달 → 진열 공간 축소 / 재계약 협상 / 디리스팅

[재발주 or 디리스팅]
```

## 4. 신규 브랜드 온보딩 흐름

세포라는 신규 브랜드를 **단계적으로 테스트**하는 구조를 취한다.

```
1단계: 수백 개 선택 도어(Doors)에서 테스트 판매
       └─ 미국 기준 약 50~100개 도어 시작

2단계: 6개월 성과 평가
       └─ 셀스루, 리뷰 점수, 재방문 고객 비율 검토

3단계: 확장 결정
       ├─ 성과 우수 → 전국 도어(Full Chain) 확장
       └─ 성과 부진 → 테스트 종료 또는 조건부 유지

4단계: 전국 확장
       └─ 미국 기준 수백 개 전 매장 롤아웃
```

### 사례: Rhode at Sephora (2025)
- 2025년 9월 4일 전 도어 및 온라인 동시 런칭
- 런칭 첫 2일 만에 세포라 리테일 매출 \$10M 돌파
- 60% 이상 구매자가 세포라 신규 고객
- **교훈**: SNS 기반 인지도(Hailey Bieber)와 기존 DTC 실적이 전국 직행 런칭을 가능하게 했음

## 5. 바이어가 보는 평가 지표

| 지표 | 설명 |
|------|------|
| Sell-Through Rate | 공급 대비 판매 비율 |
| Average Transaction Value | 평균 구매 금액 |
| Repeat Purchase Rate | 재구매 고객 비율 |
| Online Rating | 세포라 온라인 별점·리뷰 수 |
| Cross-sell | 다른 세포라 제품과의 번들 구매 여부 |
| Social Media Engagement | 인스타그램·틱톡 언급, 해시태그 |
| Influencer Coverage | 세포라 Beauty Insider Community 노출 |

## 6. 진열 배치와 성과의 관계

- **프라임 위치(입구·베스트셀러 섹션)**: 셀스루 높음, 브랜드 협상력 강함
- **카테고리 내 일반 선반**: 평균 성과
- **클리어런스(정리)**: 성과 부진 브랜드의 마지막 단계

세포라는 **플래노그램(Planogram)** 기반으로 진열을 관리하며, 브랜드 성과에 따라 위치가 조정된다.

## 7. K-beauty 브랜드 관점 시사점

- 셀스루 데이터를 실시간으로 모니터링할 수 있는 **POS 데이터 공유 계약** 확보 필요
- **인플루언서 마케팅 + PR** 병행 없이는 초기 인지도 확보 어려움
- 런칭 첫 3~6개월이 세포라 관계의 핵심 결정 구간
- **Beauty Insider Community** 리뷰 관리(이상사례 포함) 적극 대응 필요
- 세포라 바이어 교체 주기가 짧아, 관계 유지보다 **데이터 기반 성과 입증**이 더 중요
