# 일본 화장품 성분 제한 목록 (MHLW 2024)

> 일본 화장품 성분 규제는 PMD법에 따른 MHLW 고시 기반.  
> 2025년 3월 10일: MHLW가 특정 성분 특별 라벨링 규정 개정 (1985년 이후 최초 전면 개정).

---

## 규제 방식

| 방식 | 설명 |
|------|------|
| 포지티브 리스트 (Positive List) | 보존제, 자외선차단제, 타르 색소 — 지정된 성분만 허용, 최대 허용 농도 명시 |
| 네거티브 리스트 (Negative List) | 금지 성분 목록 — 해당 성분 사용 불가 |
| 성분 기준 | 그 외 성분 — 기업의 안전성 확인 후 사용 가능 |

---

## 1. 금지 성분 목록 (Negative List)

MHLW가 지정한 화장품에 사용 불가 성분 (30여 종):

| 카테고리 | 주요 금지 성분 |
|---------|-------------|
| 호르몬류 | 에스트로겐, 프로게스테론 등 성호르몬 |
| 항생제 | 페니실린, 스트렙토마이신 등 |
| 발암성 물질 | 석면, 2-나프틸아민, 4-아미노비페닐 등 |
| 중금속 화합물 | 수은화합물, 비소화합물 (허용 기준 초과) |
| 기타 | 바비투르산 유도체, 아드레날린 등 의약 성분 |

> 전체 목록은 MHLW 고시 「화장품 기준」 별표에 수록.

## 4. 타르 색소 포지티브 리스트 (부분)

일본 후생노동성 고시에서 허가된 타르 색소만 사용 가능. 주요 허가 색소:

| 색소 | 사용 가능 부위 |
|------|-------------|
| 赤色102号 (Red 102) | 전용 |
| 赤色3号 (Red 3) | 전용 (점막 제외 권고) |
| 青色1号 (Blue 1) | 전용 |
| 黄色4号 (Yellow 4) | 전용 |
| 黄色5号 (Yellow 5) | 전용 |

> 눈 주위 사용 가능 색소 별도 지정. 음식에서 사용 가능한 색소라도 화장품 허가 필요.

## 5. 제한 성분 (특정 사용 기준)

### 알레르기 유발 향료 성분 (2025년 개정 기준)
2025년 3월 MHLW 개정: 1985년 이후 최초 전면 개정. 알레르기 유발 가능성이 있는 향료 성분의 특별 라벨링 의무 강화.

| 의무 표기 성분 (예시) | INCI 명 |
|--------------------|---------|

| シトロネロール | Citronellol |
| リナロール | Linalool |
| シンナミルアルコール | Cinnamyl Alcohol |
| シンナミルアルデヒド | Cinnamaldehyde |
| 安息香酸ベンジル | Benzyl Benzoate |
| ユーゲノール | Eugenol |

## 6. 특정 성분 특별 라벨링 (アレルギー表示)

일본 화장품법은 알레르기 유발 가능 특정 성분에 대해 **표시 의무**를 규정:

- 과거 알레르기 반응 보고가 있는 성분
- 소비자가 회피 가능하도록 성분명 개별 표기
- 2025년 개정으로 표기 대상 성분 목록 확대

---

## 7. 전성분 표시 의무 (全成分表示)

| 항목 | 내용 |
|------|------|
| 도입 연도 | 1998년 |
| 표기 방식 | 함량 내림차순 (1% 이하는 임의 순서) |
| 사용 명칭 | 일본 화장품 성분표시 명칭 (INCI 대응 일본어 명칭) |
| 표기 위치 | 외부 포장 의무 (용기에도 권장) |
| 향료 | "香料"로 일괄 표기 가능 (단, 알레르기 유발 성분은 개별 표기) |

---

## 8. 나노 소재 규제

- 나노 티타늄다이옥사이드, 나노 징크옥사이드: 자외선차단제 포지티브 리스트에 포함
- 분무형(에어로졸) 제품에 나노 소재 사용 시 별도 안전성 검토 필요
- 흡입 위험 주의

## 주요 검사 항목 (완제품)

| 검사 항목 | 목적 |
|---------|------|
| 중금속 (납·비소·수은 등) | 오염 검사 |
| 방부제 함량 | 포지티브 리스트 준수 |
| 허가 색소·향료 | 포지티브 리스트 준수 |
| 미생물 | 품질 관리 |
| pH·점도·안정성 | 품질 관리 |
| 자외선차단 성능 (SPF/PA) | 해당 제품 |

## 한국→일본 수출 시 주요 주의 성분

| 한국 사용 성분 | 일본 규제 현황 |
|-------------|------------|
| 나이아신아미드 (미백) | 의약부외품 활성 성분 — 3% 이하 화장품, 이상 의약부외품 |
| 레티놀 | 의약부외품 또는 화장품 (농도·클레임 따라 다름) |
| AHA (글리콜산 등) | 화장품 허용 (pH 기준 준수 필요) |
| 알부틴 | 의약부외품 활성 성분 |
| 에칠아스코빌에텔 | 의약부외품 활성 성분 |
| 자외선차단제 성분 | 포지티브 리스트 내 성분만 허용, 의약부외품 |
