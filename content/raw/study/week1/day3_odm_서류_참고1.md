# 화장품 ODM 납품 서류 및 수출 체크리스트

> 에시컬코리아코스메틱스(EK COS) 가이드 기반.  
> 브랜드사가 ODM 제조사로부터 납품받을 때, 또는 수출 시 필요한 주요 서류 정리.

---

## 1. INCI 성분표

**정의**: 화장품 성분을 국제적으로 일관되게 명명하는 시스템(International Nomenclature of Cosmetic Ingredients)에 따른 성분 목록.

**용도**:
- 바이어 요청 시 필수 제공 문서
- 수입국 성분 검토 기초 자료

**발급처**: ODM 제조사 (공장)

**포함 내용**:
- 성분명 (INCI 명칭)
- 함량 (%) 또는 내림차순 표기
- CAS 번호 (선택)

---

## 2. COA (Certificate of Analysis / 분석 성적서)

**정의**: 제품이 생산된 후 품질 기준 충족 여부를 확인하는 시험 성적서.

> "제품이 생산된 후에만 발행이 가능" — 발주 전 발행 불가.

**주요 포함 항목**:

| 항목 | 내용 |
|------|------|
| 제품명 | 제품 정식 명칭 |
| 제조번호 (Lot No.) | 배치/로트 번호 |
| 제조일자 | 제조 날짜 |
| 사용기한 | 유효기간 |
| pH | 산도 측정값 |
| 점도 | 점성 측정값 (해당 시) |
| 성상 | 색상, 냄새, 형태 등 |
| 중금속 | 납, 비소, 수은, 카드뮴 등 함량 |
| 미생물 | 총 호기성생균수, 진균수, 대장균 등 |
| 기능성 성분 함량 | 기능성 화장품의 경우 |
| 합격/불합격 판정 | Pass/Fail |

**아마존 등 플랫폼 요구 조건**:
- 발행자 회사명이 상품 리스팅의 브랜드명·제조사명과 일치해야 함
- **발행 날짜 6개월 이내** 요건 있음 (플랫폼별 상이)

---

## 3. MSDS (Material Safety Data Sheet / 물질 안전 보건 자료)

**정의**: 화학 물질의 위험 특성, 취급 방법, 응급 처치를 기술한 문서.

**필수 제출 시점**:
- 항공 운송 시 필수 (위험물 여부 확인)
- 유럽, 미국 등 해외 수출 시
- 대형 유통사 입점 시

**주요 포함 내용**:
- 제품 식별 정보
- 성분 정보 (위험 성분 CAS 번호 포함)
- 위험·유해성
- 응급 처치
- 화재 시 조치
- 누출 사고 시 대처
- 취급 및 보관
- 환경적 영향

---

## 4. GMP/ISO 22716 인증서

**정의**: 우수 화장품 제조 관리 기준 준수를 증명하는 인증서.

**주요 인증 종류**:

| 인증 | 설명 |
|------|------|
| GMP (ISO 22716) | 국제 화장품 제조 표준 |
| ISO 9001 | 품질 경영 시스템 |
| CGMP (FDA) | 미국 FDA 우수 제조 기준 |
| ECOCERT / COSMOS | 천연·유기농 화장품 인증 |

---

## 5. 아트워크 (Artwork)

**포함 내용**:
- 제품 외관 디자인
- 단상자 전개도
- 용기 도면
- 라벨 디자인 (성분표, 사용법, 주의사항 포함)

**용도**: 인증 기관 검토, 바이어 확인, 라벨 규제 검토

---

## 6. 제품 설명서 (Product Description)

- 제품 카테고리 및 용도
- 제형 (크림, 세럼, 파우더 등)
- 주요 특징 및 효능
- 사용 방법
- 인증 기관 제출용

---

## 7. 수출 지역별 추가 서류

### 유럽 수출 시
| 서류 | 내용 |
|------|------|
| CPNP 등록 | EU 화장품 신고 포털 등록 확인서 |
| CPSR | 화장품 안전 보고서 (자격을 갖춘 안전 평가자 서명) |
| PIF | 제품 정보 파일 |
| EU RP 지정 | EU 내 책임자 지정 서류 |

### 미국 수출 시 (MoCRA)
| 서류 | 내용 |
|------|------|
| FDA 시설 등록 | 제조 시설 등록 번호 |
| 제품 리스팅 | FDA 제품 리스팅 번호 |
| SAE 체계 | 중대 이상반응 보고 체계 구축 확인 |

### 중국 수출 시
| 서류 | 내용 |
|------|------|
| NMPA 등록 | 중국 국가의약품감독관리국 등록 |
| 일반 화장품 | 전자 등록 (신고제) |
| 특수 화장품 | 사전 허가 (심사제) |

### 일본 수출 시
| 서류 | 내용 |
|------|------|
| 화장품 제조 판매 신고서 | 후생노동성 신고 |
| 성분 적합성 검토 | 일본 약사법 준수 여부 |
| 일본어 라벨 | 의무 첨부 |

## 납품 서류 체크리스트 요약

```
□ INCI 성분표 (함량 포함)
□ COA (해당 배치)
□ MSDS
□ GMP/ISO 22716 인증서
□ 제조업체 사업자등록증
□ 화장품책임판매업등록증 (국내 판매 시)
□ 아트워크 최종본
□ 수출국별 추가 서류 (CPNP, FDA 리스팅 등)
```
