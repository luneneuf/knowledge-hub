# 화장품 안정성 시험 및 ODM 발주 프로세스

> 화장품 ODM 발주 시 브랜드사가 제조사로부터 받는 핵심 서류인 안정성 시험 성적서 기준과 ODM 발주 흐름 정리.

---

## 안정성 시험 (Stability Test)

### 목적
- 제품의 유통기한 및 품질 유지 기간 입증
- 물리적·화학적·미생물학적 안정성 확인
- 개봉 후 사용 기간(PAO) 근거 자료 확보

### 시험 종류

| 시험 종류 | 조건 | 기간 | 목적 |
|---------|------|------|------|
| 장기 안정성 | 25°C / 60% RH (실온) | 12~24개월 | 유통기한 최종 확인 |
| 가속 안정성 | 40°C / 75% RH | 6개월 | 장기 데이터 예측 |
| 스트레스 테스트 | 50°C / 냉동 (-15°C) | 1~3개월 | 극한 조건 저항성 |
| 사이클 테스트 | 4°C ↔ 40°C 반복 | 수 주 | 온도 변화 저항성 |
| 광안정성 | UV 및 가시광선 조사 | 규정 조건 | 빛에 의한 변질 확인 |

### 시험 항목

#### 물리·화학적 시험
| 항목 | 기준/방법 |
|------|---------|
| 성상 | 색, 냄새, 투명도, 형태 등 육안 평가 |
| pH | pH 미터 측정, 제품 규격 범위 내 |
| 점도 | 점도계(Viscometer) 측정 |
| 굴절률 | 굴절계 측정 (해당 제품) |
| 비중 | 비중계 측정 |
| 분리도 | 에멀전 분리, 침전 여부 |
| 성분 함량 | 유효 성분 농도 확인 (HPLC 등) |

#### 미생물 시험
| 항목 | 기준 (화장품 안전기준 규정) |
|------|--------------------------|
| 총 호기성 생균수 | 1,000 CFU/g(mL) 이하 (일반) |
| 총 호기성 생균수 (눈·어린이용) | 500 CFU/g(mL) 이하 |
| 진균수 | 100 CFU/g(mL) 이하 |
| 대장균 | 불검출 |
| 녹농균 | 불검출 |
| 황색포도상구균 | 불검출 |

#### 중금속 시험
| 중금속 | 기준 |
|--------|------|
| 납 | 20 μg/g 이하 (눈 화장용 35 μg/g) |
| 비소 | 10 μg/g 이하 |
| 수은 | 1 μg/g 이하 |
| 카드뮴 | 5 μg/g 이하 |
| 안티몬 | 10 μg/g 이하 |

### 방부력 시험 (Challenge Test / Preservative Efficacy Test)
- 목적: 보존제 시스템의 미생물 억제력 검증
- 기준: USP <51>, EP 5.1.3, ISO 11930 등
- 균주: 대장균, 녹농균, 황색포도상구균, 칸디다, 아스페르길루스 등
- 시험 기간: 28일 (접종 후 2, 7, 14, 21, 28일 측정)

## ODM 발주 프로세스 (코스맥스·콜마 등 주요 ODM사 기준)

### 단계별 흐름

```
1. 컨셉 기획
   └─ 브랜드사: 제품 카테고리, 타깃, 클레임, 가격대 결정

2. ODM사 제안 또는 발주 의뢰
   └─ 기존 베이스 포뮬라 활용 (코스맥스 ONE-STOP SOLUTION)
   └─ 맞춤 개발 의뢰

3. 샘플 개발
   └─ ODM사: 처방 설계, 샘플 제조
   └─ 브랜드사: 샘플 평가 (성상, 사용감, 향, 색상)

4. 처방 확정
   └─ 성분표(INCI 목록) 확정
   └─ 수출 대상국 성분 규제 검토 동시 진행

5. 시험·인증
   └─ 안정성 시험 (가속·장기)
   └─ 미생물 시험
   └─ 중금속 시험
   └─ 기능성화장품 해당 시 → 식약처 심사/보고

6. 제조 계약 체결
   └─ MOQ (최소발주량) 협의
   └─ 단가, 납기 확정

7. 아트워크·포장재 확정
   └─ 라벨 규제 검토 (국내/수출국)
   └─ 인쇄 교정 (Color Proof)

8. 제조 발주 (PO 발행)
   └─ 브랜드사 → ODM사 발주서 발행

9. 제조 및 품질 검사
   └─ 원료 입고 검사
   └─ 인공 제조 (in-process 검사)
   └─ 완제품 검사 (COA 발행)

10. 납품 및 서류 전달
    └─ 완제품 납품
    └─ COA, MSDS, INCI 성분표, GMP 인증서 전달
```

## 브랜드사가 ODM사에 제공해야 하는 서류

| 서류 | 내용 |
|------|------|
| 발주서 (PO) | 품목, 수량, 납기, 단가 |
| 브랜드 가이드라인 | 로고, 컬러, 폰트 등 |
| 라벨·아트워크 파일 | 최종 디자인 |
| 성분 요청 사항 | 특정 성분 포함/제외 요청 |

## ODM사가 브랜드사에 제공하는 서류

| 서류 | 내용 |
|------|------|
| COA | 완제품 분석 성적서 |
| INCI 성분표 | 전성분 목록 (함량 포함) |
| MSDS | 물질 안전 보건 자료 |
| GMP 인증서 | ISO 22716 또는 CGMP |
| 안정성 시험 성적서 | 가속/장기 안정성 결과 |
| 방부력 시험 성적서 | Challenge Test 결과 |
| 기능성 심사 서류 | 기능성화장품 해당 시 |

## 코스맥스·콜마 주요 특징

| 구분 | 코스맥스 | 한국콜마 |

| 서비스 | ONE-STOP SOLUTION | 컨설팅~완제품 납품 |
