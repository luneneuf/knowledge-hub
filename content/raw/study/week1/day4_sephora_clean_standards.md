# Sephora "Clean at Sephora" 기준

> Sephora가 "잠재적으로 해로운 성분"으로 분류한 **50개 이상의 성분**이 포함되지 않은 제품에 Clean at Sephora 씰(seal)을 부여하는 프로그램.  
> 2018년 도입. 2024년 기준 120개 이상 브랜드 참여.

---

## 프로그램 개요

| 항목 | 내용 |
|------|------|
| 도입 연도 | 2018년 |
| 기준 업데이트 | 정기적 갱신 (브랜드는 통상 1년 전 사전 통보) |
| 적용 대상 | Clean at Sephora 씰 획득 원하는 모든 브랜드 |
| 씰 표시 | 제품 페이지 및 매장 내 표시 |
| 엄격성 비교 | 주요 리테일러 중 상대적으로 덜 엄격 (Credo > Whole Foods > Sephora > Ulta 순) |

## 금지 성분 목록 (Formulated Without)

### 합성 화학물질

| 성분 | 사유 |
|------|------|
| 파라벤 (Parabens) | 내분비계 교란 우려 |
| 프탈레이트 (Phthalates) | 내분비계 교란 우려 |
| 포름알데히드 (Formaldehyde) | 발암 가능성 |
| 포름알데히드 방출제 | DMDM 하이단토인, 이미다졸리디닐우레아, 쿼터늄-15 등 |
| 톨루엔 (Toluene) | 신경 독성 |
| 석탄 타르 (Coal tar) | 발암 가능성 |
| 합성 향료 (Synthetic fragrances / Parfum) | 알러지 유발 가능성 |
| BHA·BHT | 내분비계 교란, 발암 우려 |
| 트리클로산 (Triclosan) | 내분비계 교란 |
| 트리클로카반 (Triclocarban) | 내분비계 교란 |
| 옥시벤존 (Oxybenzone) | 내분비계 교란, 산호초 훼손 |
| 레조르시놀 (Resorcinol) | 내분비계 교란 |
| 에탄올아민류 (DEA/TEA/MEA/ETA) | 발암성 전구체 |
| 하이드로퀴논 (Hydroquinone) | 발암 가능성 (특정 농도 이상) |
| 카본 블랙 (Carbon black) | 흡입 발암 위험 |
| PFAS (과불화화합물) | 잔류성, 건강 위해 우려 |

### 중금속·오염물질

| 성분 | 사유 |
|------|------|
| 납 (Lead) 및 납 아세테이트 | 신경 독성 |
| 수은 및 수은 화합물 | 신경 독성 |
| 알루미늄 염 (Aluminum salts) | 흡수·축적 우려 (2024.04.01 이후 추가) |

### 석유계 성분

| 성분 | 사유 |
|------|------|
| 미네랄 오일 (Mineral oil) | 불순물 오염 우려 |
| 파라핀 (Paraffin) | 석유 유래 |

### 계면활성제

| 성분 | 사유 |
|------|------|
| 소듐 라우릴 설페이트 (SLS) | 피부 자극 |
| 소듐 라우레스 설페이트 (SLES) | 에톡실화 불순물 우려 |

### 아크릴레이트 (2024 추가)

- 아크릴레이트/에틸헥실아크릴레이트 코폴리머 등 일부 아크릴레이트류
- 네일 접착제 성분 일부

## 2024년 4월 1일 이후 추가된 금지 성분

- 아세톤 (Acetone)
- 일부 아크릴레이트류 (Acrylates)
- 알루미늄 염 (Aluminum salts)

---

## 조건부 허용 성분 (Conditionally Allowed)

### 에톡실화 성분 (Ethoxylated Ingredients)
- 허용되지만, **배치(batch)마다** 1,2-다이옥산(1,2-Dioxane) 및 에틸렌 옥사이드(Ethylene Oxide) 테스트 의무
- 테스트 증빙 서류를 Sephora에 제출해야 함

### 동물 유래 성분
- 꿀(Honey), 밀랍(Beeswax), 라놀린(Lanolin) 등 → 허용
- 잔혹성 기반 동물 성분 → 비건 씰로 별도 관리

---

## 향료 요건

- 모든 향료는 Clean at Sephora 전체 목록 준수 필요
- Parfum/Perfume 단독 제품 포함
- 다른 제품에 "fragrance" 또는 "parfum"으로 표기된 경우도 해당
- **EU 규정에 따른 향료 알러겐 성분 개별 표기** 필요

---

## Clean at Sephora 씰 취득 절차 요약

1. 브랜드 담당 Sephora 바이어와 Clean 자격 심사 진행
2. 전 포뮬라 감사(Audit): 금지 성분 미포함 확인
3. 에톡실화 성분 함유 시 배치 테스트 증빙 제출
4. Sephora Restricted Substances List (RSL) 준수 서약
5. 미래 신제품도 동일 기준 적용 동의

## Clean at Sephora vs. 기타 리테일러 기준 비교

| 기준 | Sephora Clean | Whole Foods | Credo | Ulta |
|------|-------------|-------------|-------|------|
| 금지 성분 수 | 50개+ | 100개+ | 2,700개+ | 일부 |
| 에톡실화 성분 | 조건부 허용 | 금지 | 금지 | 허용 |
| 향료 기준 | EU 기준 준수 | 엄격 | 매우 엄격 | 완화 |
| 상대적 엄격성 | 중간 | 높음 | 매우 높음 | 낮음 |
