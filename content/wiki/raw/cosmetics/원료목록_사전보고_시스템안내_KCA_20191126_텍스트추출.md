## 보고 절차

### 1. 회원가입 또는 로그인

- 실적보고 ID 있음 → **로그인**
- ID 없음 → **이용신청** ("생산실적 및 원료목록 보고전용 ID" 신청)
- 2019년 신규 업등록 또는 미보고 회사는 이용신청
- **책임판매업등록번호** 및 **등록연월일**은 ['책임판매업 등록필증'] 확인
  - 등록번호: "제 OO호" 부분의 아라비아 숫자만 입력
  - 등록년월일: 등록필증의 발급일자 정확히 입력
- ID/PW 분실 시: 보고 담당자 이름·이메일 기입 → 확인 가능

### 2. 보고유형 선택 (3종)

| 유형 | 사용 시점 |
|------|-----------|
| **최초보고** | 제품 유통·판매 전 |
| **변경보고** | 기보고한 제품의 전성분이 변경되거나 제조업자가 변경되는 경우 (변경 이력 조회 가능) |
| **단종보고** | 보고한 제품이 단종된 경우 |

> 실적보고 웹사이트에 **수기등록** 또는 **엑셀등록** 방식으로 보고

### 3. 최초보고 절차

```
보고회사 및 담당자 정보 작성
   ↓
원료목록 작성 (엑셀 업로드 또는 수기 등록)
   ↓ 원료검사 (표준명·배합금지 여부 확인)
   ↓ 표준명 미등록 / 금지원료 → 수정
제출
```

#### 제출 시 4가지 케이스 안내
1. 수정사항 필요 없는 경우 → 제출 가능
2. 단서조항 확인 필요한 배합금지 성분 → 확인 후 제출
3. **순수 배합금지 성분** → 수정 필수
4. **표준명 미등록 성분** → 수정 필수

### 4. 변경보고

1. 메인 화면에서 변경보고 클릭 (또는 결과조회에서 제품명 클릭 → 변경보고)
2. 변경사항 있는 제품명 클릭
3. 회사 및 담당자 정보 작성
4. 수정 (3가지 방식):
   - **화면에서 직접 변경보고하기** (등록이력 불러오기)
   - **선택 제품만 엑셀로 변경보고하기**
   - **전체 엑셀로 변경보고하기**
5. 제출

### 5. 단종보고

1. 메인 화면에서 제품단종 클릭 (또는 결과조회 → 단종할 제품명 클릭 → 제품단종 클릭)
2. 결과조회로 확인

### 6. 결과조회

- 보고 연도별 제출내역 확인
- 수정 가능
- 식약처 제출 보고서 서식대로 인쇄 가능

## 원료 작성 시 주의사항 — 혼합물 처리

**혼합물은 개별 성분으로 기재할 것**: 개별 원료로 분리하여 **각 셀당 한 개 원료** 기재 (반드시 표준화된 명칭으로)

| 잘못된 예 | 올바른 예 |
|-----------|-----------|
| 정제수 & 소듐하이알루로네이트 & 라피노오스 & 산자나무추출물 | 정제수 / 소듐하이알루로네이트 / 라피노오스 / 산자나무추출물 (각 행 분리) |
| 정제수 * 안개초뿌리추출물 | 정제수 / 안개초뿌리추출물 (각 행 분리) |
| 정제수 / 글리세린 / 포도씨추출물 | 정제수 / 글리세린 / 포도씨추출물 (각 행 분리) — **"/" 사용 부적절** |

**기호 "/" 사용 규칙**:
- **단일성분으로 간주하는 복수원료**의 성분 명칭을 [`/`] (slash, 사선)로 이어서 표시
- 예: `아크릴레이트/C1-2석시네이트/하이드록시아크릴레이트코폴리머`
