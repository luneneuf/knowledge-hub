## 전제 재설정 — 이 4주의 성격이 바뀌었다

v1에서 "시스템을 이식하기 전에 기계를 해독하라"는 프레임은 유효하다.

그런데 리서치 결과, **일부 문제는 해독이 끝나기 전에 이미 발생해 있을 가능성이 높다.**

두 가지가 특히 그렇다.

첫째, 한국 OEM이 공급한 색소가 FDA batch certification을 받지 않은 lot일 경우, 세포라 80개 매장에 현재 진열된 SKU가 미국 법상 adulterated cosmetic이다. 이것은 입점 이전 통관을 통과했더라도 적용된다. 재주문 물량이 들어올 때마다 import detention 리스크가 반복된다.

둘째, UK Boots 91개 매장의 경우 SI 2024/1334(64종 CMR 금지)와 SI 2024/455(BHT 강화)의 off-shelf 시한이 이미 경과했다. 해당 성분이 포함된 SKU가 진열 중이라면 지금 이 순간 OPSS 제재 대상이다.

따라서 이 4주의 성격은 "진단 → 보고"가 아니라 **"진단 + 긴급 처리 병행"** 이다. 1주차부터 생산·SCM 팀을 즉시 끌어들여야 한다.

## Week 1 — 해독 + 긴급 신호 탐지

**핵심 질문:** 지금 당장 불이 나 있는 곳이 어디인가?

### 1-1. 이해관계자 미팅 착수 — 목적 재설정

5명과의 1:1 미팅은 v1과 동일하다. 단, 대표 미팅의 어젠다가 달라진다.

| 대화 상대 | 핵심 질문 (v2 추가/변경) |
|-----------|--------------------------|
| **이지철 대표** | "현재 미국·영국 시장에서 컴플라이언스 담당이 명확히 지정되어 있습니까?" / "RP 지정 현황이 어떻게 됩니까?" / "브랜드 정체성에서 절대 바꾸지 않을 것이 무엇입니까?" (크리에이티브 부문 직접 겸임 중) |
| **임지현 CD** | "마케팅 카피 영문화 검토 프로세스가 있습니까? 누가 최종 승인합니까?" |
| **손여진 본부장** (글로벌사업) | "세포라에서 Vendor Compliance 관련 연락이 온 적 있습니까? / ODM 파트너가 어디입니까?" |
| **생산·SCM 담당자** | "각 SKU에 사용된 색소의 FDA batch certification 번호를 ODM에서 받은 적 있습니까?" |

### 1-2. 즉시 확인 항목 — Day 1~3 착수

아래 4개는 미팅 전에 SCM 팀을 통해 확인을 시작해야 한다. 결과가 나쁠수록 빨리 아는 게 낫다.

**① 색소 FDA Batch Certification 현황 (최우선)**

ODM에 연락해서 각 SKU별 사용 색소의 공급사명과 batch cert 번호를 요청한다.

특히 확인해야 할 것:
- CI 15850(D&C Red No. 7 Calcium Lake) — 한국 립틴트 핵심 색소, batch cert 없으면 즉시 문제
- CI 15510(D&C Orange No. 4) — 미국에서 립 사용 불가, 외용 전용. 함유 SKU 있으면 reformulation 필수
- 식물성 색소(비트, 자색고추, 블루베리 추출) — "color" 목적 사용 시 미국에서 위반

ODM이 이 정보를 즉시 주지 못하면, 그것 자체가 중요한 신호다.

**② UK SI 2024/1334 + SI 2024/455 위반 여부**

Boots 진열 중인 SKU에 아래 성분이 포함되어 있는지 ODM에 즉시 확인:

- SI 2024/1334: 64종 CMR 목록 해당 성분 (off-shelf 시한 2025.10.20~2026.03.02 — 이미 경과)
- SI 2024/455: BHT leave-on oral 한도 0.001% (EU 0.8% 대비 800배 엄격, off-shelf 2025.06.24 — 이미 경과)

위반이 확인되면 Boots 해당 SKU 즉시 회수 또는 판매 중단 조치가 필요하다.

**③ 세포라닷컴 + 라벨 카피 Drug Claim 스캔**

현재 세포라닷컴 제품 페이지와 패키지 영문 카피를 검토해서 아래 표현을 찾는다.

- Regenerate, restore, heal, treat, anti-aging, stimulate, repair
- 한국 원문의 "재생", "복원", "주름 개선", "트러블 진정" 영문 직역본

발견 즉시 마케팅팀에 수정 의뢰. 이것은 미국 통관 거부 1위 사유(의약품 오분류)다.

**④ ODM QA 계약 형태 확인**

현재 ODM 파트너와의 계약이 단일 OEM/ODM 계약서 형태인지, 별도 Quality Agreement 부속서가 분리되어 있는지 확인한다. 분리되어 있지 않으면 Week 2 ODM 파트 협의의 최우선 과제가 된다.

### 1-3. SKU 포트폴리오 첫 스캔 — 수정된 관점

400 SKU는 K-뷰티 인디 색조 평균(80~300개) 대비 높지만 관리 가능한 수준이다. SKU당 매출 5,725만원은 "healthy catalog" 기준(\$10,000/SKU/년)의 4배로 효율이 우수하다. 따라서 "합리화"보다 **시장별 배분 구조 파악**이 이 단계의 목적이다.

확인할 것:
- 4개 시장(한국·일본·미국·영국)에 SKU가 어떻게 배분되어 있는가
- 세포라 입점 SKU(현재 3종)와 전체 400개의 관계
- 세포라 6개월 리뷰에서 shade range 확장 여지(현재 10색 → 16~20색 로드맵)

## Week 2 — 데이터 수집 및 4개 시장 규제 매트릭스

**핵심 질문:** 400개 SKU 각각이 4개 시장에서 법적으로 안전한가?

### 2-1. 4개 시장 × SKU 규제 매트릭스 작성

이것이 Week 2의 핵심 산출물이다. Week 4 보고서의 가장 임팩트 있는 슬라이드가 된다.

```
[ 매트릭스 구조 ]
SKU명 | KR(MFDS) | JP(약기법) | US(MoCRA/Clean@Sephora) | UK(UKCR SI) | 비고
-------|-----------|------------|-------------------------|-------------|-----
프루티 글램 틴트 #01 | ✅ | ✅ | Pb 확인 필요 / Batch cert? | SI 2024/1334? | 세포라 입점
프루티 글램 틴트 #02 | ... | ... | ... | ... | ...
```

각 셀의 표기 기준:
- ✅ : 확인됨
- ⚠️ : 확인 필요
- 🔴 : 위반 가능성 / 즉시 처리

전체 400개를 2주 안에 다 채울 수 없다. **세포라 입점 SKU(3종) → 부츠 입점 SKU 전체 → 나머지** 순으로 우선순위를 정해 진행한다.

### 2-2. 시장별 핵심 규제 체크포인트

**미국 (세포라 8월 리뷰 준비)**

| 체크 항목 | 기준 | 확인 방법 |
|-----------|------|-----------|
| Pb 함량 | 세포라: 1ppm / 캘리포니아 안전권: 5ppm 이하 | ICP-MS 성적서 확인 또는 재발주 |
| 색소 Batch Cert | 21 CFR 74 등재 색소 전체 | ODM에 lot별 FDA cert 번호 요청 |
| Mineral Oil/Petrolatum | Clean at Sephora 전면 금지 | ODM 처방서 확인 |
| Drug Claim | FDA misbranding | 세포라닷컴 + 라벨 스캔 (Week 1 착수) |
| MoCRA: FEI 등록 + 제품 리스팅 | 2024.7.1 본격 집행 | FDA Cosmetics Direct 확인 |
| RP 미국 연락처 라벨 | 2024.12.29 시행 | 패키지 확인 |
| Carmine 표기 | "Contains carmine as a color additive." 명시 | 라벨 확인 |

**영국 (Boots 91매장)**

| 체크 항목 | 기준 | 확인 방법 |
|-----------|------|-----------|
| SI 2024/1334 CMR 64종 | off-shelf 이미 경과 | ODM 처방 감사 (Week 1 시작) |
| SI 2024/455 BHT | leave-on oral 0.001% | ODM 처방 감사 |
| Formaldehyde-releaser | 2026.07.15부터 10ppm + 라벨 문구 의무 | ODM 처방 + 라벨 준비 |
| 중금속 | OPSS 공식 기준 없음 → BVL 기준(Pb 2ppm) 적용 | ICP-MS 성적서 + PIF 비치 |
| UK RP 라벨 | 2027.12.31 전환 (보수적으로 2026년 내 완료 권장) | UK RP 대행사 계약 필요 |
| CPSR Part B 서명자 | 약학·독성학·의학 학위 보유 UK Safety Assessor | 현재 서명자 자격 확인 |
| SCPN 등록 | UK RP 명의 필수 | 포털 등록 현황 확인 |

**UK RP 대행사 선정:** GCRS, REACH24H, 와이즈컴퍼니, Biorius 한국어 데스크 4곳 견적 요청. 평가 기준은 비용보다 SUE 대응 한국어 정합성.

**일본 (JT 법인 통해 운영)**

JT가 일본 유통을 담당하고 있어 PMDA 관련 1차 대응은 JT 측에서 처리된다. 단 확인할 것:
- JT와의 계약에서 품질 클레임 발생 시 LAKA HQ에 통보 SLA가 명시되어 있는가
- JT가 Qoo10 11분기 1위 관련 소비자 클레임 데이터를 LAKA에 공유하고 있는가

### 2-3. ODM QA 운영 — 현실적 3가지

콜마·코스맥스 계약에는 감사권·배치 기록 열람권이 이미 들어 있다. 문제는 조항이 없는 게 아니라 브랜드가 그 권리를 행사하지 않는다는 것이다. 계약을 재협상하거나 실험실을 운영하는 게 아니라, **이미 있는 권리를 루틴으로 쓰는 것**이 현실적인 접근이다.

Week 2에서 확인하고 설계할 것은 아래 3가지다.

**① TPT(Third Party Testing) 연간 계획 수립**

외부 공인 검사기관(SGS Korea, Intertek Korea, Eurofins Korea, 식약처 인증 위탁기관)을 활용한다. 전수 검사도 매 배치도 아니다. 결정해야 할 것은 단 세 가지다 — 어떤 리스크 유형에 대해, 어떤 SKU를, 어떤 주기로.

현실적 기본 설계:
- **신제품 출시 전 1회**: 립 SKU 중금속 5종(Pb/As/Cd/Hg/Sb) ICP-MS + 색소 CI→D&C 매핑 확인
- 비용: 중금속 SKU당 20~50만원 / 미생물 배치당 10~30만원
- 연간 전체 예산: 수천만원 이내 — 세포라 리뷰 실패 비용 대비 압도적으로 저렴

**② ODM 배치 기록 요청 루틴화**

계약 재협상 없이, 운영상 분기별 요청만. 요청 항목:
- 색소 공급사명 + FDA batch cert 번호 (CoA에 포함 요청)
- 처방 변경 내역 (MoCRA product listing 업데이트 트리거)
- 배치별 IPC 결과 요약

대부분의 브랜드가 안 하는 이유는 법적으로 못 해서가 아니라 그냥 안 하기 때문이다. 분기 1회 이메일 요청이면 충분하다.

**③ 영문 마케팅 카피 Ban List 운영**

비용 제로, 인력 추가 불필요. FDA Warning Letter의 상당수가 제품이 아니라 웹사이트 카피에서 시작한다.

금지 키워드 목록 (내부 체크리스트로 운영):

| 한국 원문 | 영문 직역 (금지) | 대체 표현 |
|-----------|-----------------|-----------|
| 재생 | regenerate / restore | hydrating / nourishing |
| 복원 | repair / heal | smoothes the appearance of |
| 주름 개선 | reduces wrinkles | minimizes the look of fine lines |
| 트러블 진정 | treats / heals skin | soothes the appearance of |
| 회복 | recovers / cures | replenishes |

신제품 출시 전, 세포라닷컴 카피 업데이트 전 이 리스트로 1회 체크하는 것을 게이트로 만든다.

**IAF CertSearch(iafcertsearch.org):** ODM 인증서 유효성·정지 이력 5분 온라인 검증. Week 1에 해두면 된다.

### 2-4. 클레임 데이터 분석

v1 프레임 유지하되 추가 포인트:
- 세포라 입점 후 3개월(2026.02~04) 리뷰 데이터 — 초기 평점 4.5/5 유지 여부
- FDA CAERS(Cosmetic Adverse Event Reporting System) 공개 DB에서 LAKA 브랜드명 검색 — 네거티브 신호 사전 인지
- "Fruity Glam Tint 식후 색이 빠진다"는 영어권 불만 빈도 — 이건 제품 결함이 아니라 소비자 기대치 설정 실패(Communication Failure)로 분류. 제품 수정이 아닌 카피 수정으로 해결 가능.

## Week 3 — 갭 분석 및 Quick Win 확정

**핵심 질문:** 1~2주 진단 결과, 무엇이 불이 나 있고 무엇이 구조적 문제인가?

### 3-1. 리스크 매트릭스 — 5개 리서치 기반 예상 분포

```
              [위험도 높음]
                    │
```

### 3-2. Quick Win 3개 — 기준 재설정

v2에서의 Quick Win 기준: 4주 이내 완료 가능 + 가시적 결과 + 8월 세포라 리뷰에 직접 영향

**Quick Win 1: Drug Claim 카피 정화**
- 세포라닷컴 제품 페이지 + 패키지 영문 카피에서 drug claim 표현 제거
- "regenerate" → "hydrating", "restore" → "smoothes the appearance of"
- 마케팅팀과 1~2주 내 완료 가능. 비용 거의 없음.
- 효과: FDA misbranding 리스크 즉시 제거. 통관 거부 사유 1위 대응.

**Quick Win 2: 색소 Batch Cert 현황 문서화**
- ODM으로부터 전체 SKU 색소 공급사명 + FDA cert 번호 취합
- CI 15510 함유 SKU 확인 → 해당 SKU 세포라 공급 즉시 중단 조치
- 결과물: FDA-compliant 색소 현황표 → 세포라 Vendor Compliance 패키지에 직접 활용

**Quick Win 3: UK SI 위반 SKU 정리**
- SI 2024/1334 + 2024/455 해당 성분 함유 SKU를 Boots 채널에서 즉시 분리
- 생산팀 + Boots 영업 담당자와 협의, 해당 SKU 재고 처리 방향 결정
- 이것은 "잘하는 것"이 아니라 "지금 하지 않으면 제재받는 것"

### 3-3. 삼성전자 QA 배경이 가장 효과적으로 쓰이는 지점

리서치 결과를 보면 LAKA에 필요한 것이 선명해졌다. 삼성식 "시스템 이식"이 아니라 삼성에서 당연하게 갖추어져 있던 것들 — batch records 체계, 변경관리 사전승인 분류, supplier qualification 문서, 색소 공급망 추적 — 이 LAKA에 없거나 ODM에 흩어져 있는 상태다.

이것을 "삼성처럼 해야 한다"는 언어가 아니라 **"세포라와 Boots가 요구하는 것을 갖추는 것"** 이라는 언어로 제안하면 저항이 없다. 경영진 입장에서 "삼성 시스템 도입"보다 "세포라 6개월 리뷰 통과 요건"이 훨씬 수용하기 쉬운 프레임이다.
