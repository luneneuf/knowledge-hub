# Fenty Beauty

## 요약

2017년 리한나 설립. LVMH 산하 [[Kendo_Brands]]와 50:50 합작. "Beauty for All" 포용적 뷰티. **세포라+부츠 동시 입점의 유일한 비교 대상.** 첫 해 매출 ~$5.7억.

## 핵심 데이터

| 항목 | 내용 |
|------|------|
| 설립 | 2017.09 |
| 히어로 제품 | Gloss Bomb Universal Lip Luminizer ($22), Stunna Lip Paint ($29) |
| 매출 | 첫 40일 $1억, 첫 해 ~$5.7억, 현재 ~$7.5억+ |
| 인프라 | LVMH/Kendo, 리한나 1.6억 팔로워 |

## [[투_트랙_채널_전략]]

**세포라**: 론칭과 동시에 17개국 1,620개 매장 동시 입점(역대 최대). 미국 독점 4.5년(2017~2022) 후 울타 확장.

**부츠**: 2019.05, 32개 매장에서 시작 → boots.com 전체 + 2,500개 매장 주문·수령. 이전 UK는 Harvey Nichols 독점. 부츠 론칭일 Gloss Bomb **21초에 1개씩 판매**.

## [[히어로_SKU_프랜차이즈화]]

Gloss Bomb → Heat, Ice, Oil, Stix, Cream 등 **6개+ 파생 제품**으로 프랜차이즈화.

## 카테고리 확장

파운데이션(40 셰이드) → 립 → 컨실러·파우더 → 바디 → 스킨케어(Fenty Skin, 2020) → 향수(2021) → 헤어케어(Fenty Hair, 2024). 매 카테고리마다 "for all" 포용성 유지.

## [[LAKA]] 시사점

유일한 세포라+부츠 투 트랙 비교 대상이나, LVMH 인프라·리한나라는 **조건이 근본적으로 다름**. 부츠에서 32개 매장 시작이라는 점은 LAKA(91개)와 비교 가능.
