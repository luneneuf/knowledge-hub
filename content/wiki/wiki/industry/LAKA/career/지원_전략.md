# 지원 전략

[[LAKA]] 이직을 위한 접근 방법, 타이밍, 채널 전략.

## 채용 모니터링

### 현황
- LAKA **11개월 채용 공백** 확인 (2026.04 기준)
- 공개 채용 공고 없음
- 소유권 전환([[구다이글로벌]] → [[이지철]] 체제) 후 조직 정비 기간으로 추정

### 모니터링 채널
- 원티드, 사람인, 잡코리아: LAKA 키워드 알림 설정
- LinkedIn: LAKA 회사 페이지 팔로우, 직원 변동 추적
- 잡플래닛/블라인드: 내부 분위기 체크
- 뉴스: "라카코스메틱스 채용", "LAKA 인재" 키워드 모니터링

## 직접 아웃리치

### 이지철 대표 직접 접근

- 11개월 채용 공백 → **공개 채용이 아닌 경영진 직접 영입** 가능성
- 접근 경로:
  1. LinkedIn 다이렉트 메시지
  2. 업계 네트워크 통한 소개
  3. 성수 플래그십 방문 후 간접 접점 구축
- 메시지 핵심: "세포라 글로벌 확장 시점에 B2B 파트너십 + 조직 빌딩 경험을 가진 인재"

### 아웃리치 메시지 구조 (초안)

```
[제목] LAKA의 글로벌 확장에 기여하고 싶은 경영지원/해외사업 전문가

- 글로벌 B2B 파트너십 10년+ (게임 IP → 채널 파트너십 전환 가능)
- 성과관리 시스템 설계 경험 (BSC 도입, 인사평가 체계 구축)
- 소비재 경험 (F&B 창업, PB 상품 개발)
- LAKA의 세포라 입점 → 풀롤아웃 전환기에 조직 역량 강화 기여 희망
```

## 타이밍

### 최적 시점: 2026년 7~8월

- [[세포라]] 6개월 바이어 리뷰 시점 (2026.02 입점 → 2026.08 리뷰)
- 리뷰 통과 시: **풀롤아웃 준비를 위한 조직 확장** 예상
- 리뷰 미통과 시에도: 위기 대응 인력 필요
- 어느 쪽이든 **8월 전후로 인력 수요 발생** 가능성 높음

### 타이밍 시나리오

| 시점 | 상황 | 접근 |
|------|------|------|
| 2026.05~06 | 사전 접근 | 네트워크 구축, 비공식 만남 |
| 2026.07~08 | 세포라 리뷰 전후 | 직접 아웃리치 최적 시점 |
| 2026.09~ | 리뷰 결과 반영 | 채용 공고 대응 또는 2차 접근 |

## 복합 포지션 제안

### 왜 복합 포지션인가

- 90명 조직에서 글로벌사업 전담 + 경영지원 전담을 각각 뽑기 어려움
- **글로벌사업 + 경영지원 겸직** 포지션을 제안하면:
  - 회사 입장: 한 명의 채용으로 두 가지 과제 해결
  - 지원자 입장: 경쟁자 감소 (두 역량을 동시에 가진 인재 희소)

### 제안 포인트

- "세포라·부츠 채널 관리 + 성과관리 시스템 구축을 동시에 추진할 수 있습니다"
- "인디 브랜드의 특성을 이해합니다 — 한 사람이 여러 모자를 쓰는 문화에 익숙합니다"
