# Charlotte Tilbury

## 요약

2013년 영국 런던에서 셀럽 메이크업 아티스트 샬롯 틸버리 설립. 럭셔리-접근성(luxury-accessible) 포지셔닝. 히어로 SKU 프랜차이즈화의 정석 — Pillow Talk 셰이드.

## 핵심 데이터

| 항목 | 내용 |
|------|------|
| 설립 | 2013, 영국 런던 |
| 히어로 제품 | Pillow Talk Matte Revolution Lipstick (£29/$35) |
| 매출 | £26M(2015) → £100M(2018) → £310M+(2022) |
| 인수 | Puig 인수 (2020년, ~$15억) |
| 소비자 언어 | "My go-to nude", "cult favorite", "your lips but better" |

## Pillow Talk 프랜차이즈

전 세계 **3초에 1개씩 판매**. "누구에게나 어울리는 누드 핑크"라는 유니버설 어필. 립라이너 1개에서 블러셔, 아이섀도, 마스카라, 립 글로스 등 **13개+ 카테고리로 확장**. 하나의 히어로 셰이드가 전체 브랜드 프랜차이즈가 된 사례.

## 채널 전략

Selfridges(2013) → Nordstrom(2014) → 독립 플래그십(2015) → **세포라 미국 40개(2018)** → 울타·아마존(2025).  
**부츠 미입점** — Space NK, John Lewis, Selfridges 등 프레스티지 채널 중심. $35 가격대가 부츠 매스 포지셔닝과 맞지 않음.
