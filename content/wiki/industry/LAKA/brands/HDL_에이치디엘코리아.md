# HDL (에이치디엘코리아)

LAKA SCM 사슬의 **국내 물류 허브** — 김포 거점 3PL 창고 + 국제물류 포워더. 일평균 2,500PT(파렛트) 보관, 국문·일문·영문 재고 통합 운영. **국내 보관·라벨·출고·해외 통관**까지 일괄 위탁의 가장 결정적 거점.

> **포워더 + 3PL 결합형의 의미**: 단순 창고가 아니라 **해외 수출입 통관·항공/해상 운송 주선** + **국내 보관·라벨링·출고**가 한 사업자에서 일괄 가능. LAKA가 해외 셀-인을 김포 거점에서 처리하는 구조에 잘 맞음 ([[Sell-in_Sell-out]]).

### 1일 착인캐파 7~8천개 (출고 가이드 컨텍스트)

본부장 공유 가이드 中:
> "HDL 1일 착인캐파 7~8천개 / 월말 출고물량 몰려있을 경우 / 품목별 50개 이하 출고되는 품목"

해석 ([[SCM_용어집|#착인 항목 참조]]):
- HDL 작업자가 1일 처리할 수 있는 **출고 단상자 라벨·검수·표기 확인 작업 = 7~8천개**
- **월말 출고 폭주** + **품목당 50개 이하 소량 다품종 출고** = 셋업·계수·라벨링 오버헤드로 처리 효율 급락
- → 사내 운영 룰 필요 (월말 분산, 소량 출고 묶기, 우선순위 SKU 사전 협의 등)

## 운영 재고 분류 (HDL 보관 기준)

LAKA는 HDL 창고에서 재고를 라벨 기준으로 4분류 (2026-04-W5부터 카테고리 재정의):

| 카테고리 | 정의 | 2026-04-W5 기준 |
|---|---|---:|
| **국문** | 국내 + 해외 + 큐텐 물량 | 72.7억 |
| **일문** | JT 안전재고 + 큐텐 일본향 | 35.1억 |
| **영문** | 미주 향 | 10.3억 |
| **국/영문** | 전 채널 통합 라벨 (신규, 2026-04부터 등장) | 5.3억 (4/6 2.79억 → **+2.51억**) |

→ "국/영문" 통합 라벨 카테고리 신설은 **다국가 통합 라벨링 전략** 진행 시그널.

## 시사점 — LAKA SCM 사슬 內 위상

| 단계 | 거래처 | 위키 |
|---|---|---|
| 생산 (ODM) | [[한국콜마]] · [[코스맥스]] · 코스메카 | ✓ |
| **국내 보관·라벨·출고·통관** | **HDL 김포물류센터** | **본 페이지** |
| 해외 풀필먼트 | [[어크로스비]] (미·일·중 거점 + UK·DE 비공식) | ✓ |
| 일본 PMDA·세관 대응 | [[株式会社JT]] | ✓ |

1. **단일점 의존 리스크**: 국내 보관·출고·통관이 HDL 김포 1곳에 집중. **BCP(업무 연속성 계획) 검토 필요** — 화재·홍수·노조·시스템 장애 시 대안 거점 부재.
2. **착인캐파 vs 월말 출고 패턴 미스매치**: 7~8천개/일 캐파 + 월말 몰림 → 룰 기반 분산 운영 필요. [[laka_sop_v1|S&OP]] Step 3(Supply Review)에 출고 capacity 점검 포함 가치.
3. **다국가 통합 라벨링 전환**: "국/영문" 신규 카테고리 등장 → HDL 라벨링 워크플로 변경 진행 중. 변경 직후 단상자 혼입 클레임 ([[퍼펙트-트윈-립]] #902·903·905호 혼입 건) 발생 — 인과 가능성 점검 필요.
