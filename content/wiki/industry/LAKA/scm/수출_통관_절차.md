# 수출 통관 절차

> 화장품 수출은 한국 세관의 **수출 통관**과 수입국 세관의 **수입 통관**, 두 번의 통관이 필요하다.

---

## 수출 통관 7단계 공통 흐름

```
1. 제조사 출고 및 포워더 선정
   └─ 화장품 전문 포워더 선정 권장 (국가별 라벨·성분 규제 숙지)

2. 수출 신고
   └─ 한국 세관 → 수출신고필증(수출면장) 발급

3. 선적
   └─ AIR (항공) 또는 SEA (해상) 선택

4. 수입국 도착
   └─ 현지 통관 브로커 수입 신고

5. 수입국 세관 심사 + 규제기관 스크리닝
   └─ 미국: FDA PREDICT / 영국: HMRC + 화장품 규제 확인 / 일본: 세관 + 약사법 확인

6. 통관 허가
   └─ 현지 창고 반입

7. 채널 납품
   └─ 세포라·Boots·드럭스토어 물류 창고 납품
```

---

## 기본 서류 목록 (공통)

| 서류명 | 설명 |
|--------|------|
| Commercial Invoice (상업송장) | 품목명, 수량, 단가, 총금액, 발송인·수신인 |
| Packing List (포장명세서) | 박스별 수량, 중량, 부피 |
| B/L 또는 Airway Bill | 해상: 선하증권(B/L), 항공: 항공화물운송장(AWB) |
| Certificate of Origin (원산지증명서) | 한국산 표시, FTA 적용 시 협정 원산지증명서 |
| 성분표 (Ingredient List) | 국가별 규제 심사용 |
| MSDS | 위험물 분류 해당 시 (알코올 함유 제품 등) |
| 보험증명서 | CIF 조건인 경우 |

## 미국·영국·일본 통관 구조 비교

| 항목 | **미국** | **영국** | **일본** |
|------|---------|---------|---------|
| 관할 기관 | CBP (세관국경보호청) + FDA | HMRC (영국 세관) + OPSS | 재무성 관세국 |
| 규제 스크리닝 | FDA PREDICT 자동 스크리닝 | SCPN 신고 여부 확인 | 약사법(PMD법) 기준 성분 확인 |
| FTA 적용 | 미-한 FTA | UK-Korea FTA (브렉시트 후 별도 체결) | 한-일 FTA 없음 |
| 관세율 | FTA 적용 시 0~5% | FTA 적용 시 0~5% | 일반 관세율 적용 |
| 억류 시 이의 기한 | 30일 이내 이의 신청 | — | — |
| 추가 서류 | FDA 등록번호, 제품 리스팅, 영문 라벨 | SCPN 신고 확인서, UK RP 서류 | 제조판매업 허가증, 성분 적합 확인, 일본어 라벨 |

### 미국 통관 심사 구조

```
CBP Entry 신고
       ↓
약식통관 (Invoice \$2,500 미만) → 빠른 처리
정식통관 (Invoice \$2,500 이상) → CBP + FDA 스크리닝
       ↓
FDA PREDICT 자동 스크리닝
├─ [통과] → 통관 허가
└─ [플래그] → 검사 또는 억류 (30일 이내 이의 신청 또는 반송/폐기)
```

### 일본 통관 구조

```
1. 제조판매업 허가 확인 (수입대행사)
2. 성분 적합성 확인 (JCIA 기준)
3. 일본어 라벨 부착 확인
4. 세관 신고 → 관세·소비세 납부
5. 통관 허가 → 창고 반입
6. 검품 (GQP) → 출하 판정
```

## AIR vs SEA 비교

| 항목 | **항공 (AIR)** | **해상 (SEA)** |
|------|-------------|-------------|
| 운송 서류 | 항공화물운송장(AWB) | 선하증권(B/L) |
| 소요 기간 (한국→미국) | **2~5일** | **15~25일** |
| 소요 기간 (한국→영국) | **3~7일** | **25~35일** |
| 소요 기간 (한국→일본) | 1~2일 | 3~7일 |
| 비용 | 높음 (kg당 과금) | 낮음 (CBM 또는 TEU 단위) |
| 적합한 경우 | 소량·고가 제품, 긴급 납품, 초기 테스트 물량 | 대량 발주, 정기 납품 |
| 위험물 규정 | IATA 규정 (에어로졸·인화성 엄격) | IMDG Code (위험물 선박 운송 기준) |

> 알코올 함유 토너, 향수 등 인화성 성분 포함 제품은 **위험물(DG)**로 분류될 수 있음. 항공 시 IATA LQ(Limited Quantity) 포장 가능 여부 확인 필수.

## 납기 역산 계산법 (세포라 미국 기준)

| 단계 | 소요 기간 |
|------|----------|
| 제조사 생산 완료 | D-Day |
| 출고·선적 준비 | +3~5일 |
| 해상 운송 (한국→미국) | +20일 |
| 미국 통관 | +5~10일 |
| 창고 반입·검수 | +3~5일 |
| 세포라 입고 처리 | +5~10일 |
| **판매 시작 가능** | **D + 약 36~50일** |

> 항공 이용 시 총 소요 기간 약 15~25일로 단축 가능. 단, 비용 대폭 증가.

## 통관 억류 주요 원인 및 예방

| 억류 원인 | 예방 포인트 |
|----------|-----------|
| 라벨 부적합 (영문 미표기, 성분 미표기) | 수출 전 각국 라벨 기준 전수 검토 |
| 의약품 클레임 (화장품에 의약 효능 표현) | 마케팅 클레임 사전 법률 검토 |
| 금지 색소 사용 | FDA/MHLW 포지티브 리스트 대조 |
| 제조사 FDA 등록 누락 | 코스맥스·콜마 등 ODM사 FDA 등록 여부 사전 확인 |
| 금지 성분 포함 | 수출국별 네거티브 리스트 성분 검토 |
| SCPN 미신고 (영국) | 출시 전 SCPN 신고 완료 |
| 일본어 전성분 표기 미이행 | JCIA 기준 명칭으로 전성분 일본어 표기 |
