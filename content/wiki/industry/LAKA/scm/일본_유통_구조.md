# 일본 유통 구조

> 일본 화장품 유통은 **도매상 중심 구조**가 강하다. 소비자의 90% 이상이 오프라인 매장 구매를 선호하며, 전자상거래 비율은 8.24%(2022년)에 불과하다.

---

## 일본 화장품 유통 구조 개관

### 전체 흐름

```
한국 브랜드사
     ↓
[일본 수입사 or 수입대행사]
     ↓ (통관·보관)
[화장품 도매상]
     ↓
[드럭스토어 / 버라이어티숍 / 홈쇼핑]
     ↓
[소비자]
```

### 주요 소매 채널

| 채널 유형 | 대표 업체 |
|----------|---------|
| 드럭스토어 | 마츠모토키요시(マツモトキヨシ), 스기약국(スギ薬局), 코코카라파인, Tomod's |
| 버라이어티숍 | 도큐핸즈, 로프트(Loft), @cosme 스토어, PLAZA |
| TV홈쇼핑 | ShopChannel, QVC, Japanet |
| 온라인 | 라쿠텐, Amazon Japan, @cosme 쇼핑, Qoo10 Japan |

> **도매상 필수 경유 원칙**: 대형 체인 드럭스토어·버라이어티숍에 직접 납품하는 것은 일반적으로 불가능.

---

## 도매 vs 직납 비교표

| 항목 | **도매 경유** | **직납 (直納)** |
|------|------------|--------------|
| 진입 장벽 | 낮음 | 높음 (현지 법인 필요) |
| 마진 구조 | 도매 마진(15~25%) + 소매 마진(30~40%) 부담 | 도매 마진 절약 |
| 재고 관리 | 도매상이 담당 | 브랜드사 직접 담당 |
| 판매 데이터 접근 | 제한적 (수입사 경유) | 실시간 POS 데이터 접근 가능 |
| 반품 처리 | 도매상이 흡수 | 브랜드사 직접 부담 |
| 소매점 관계 | 간접적 | 직접적 |
| 마케팅 주도권 | 낮음 | 높음 |
| 적합 단계 | **초기 진입** | **안정적 매출 확보 후** |

---

## 마진 구조 (소비자가 기준 역산법)

| 단계 | 마진율 | 누적 가격 인상 |
|------|--------|-------------|
| 한국 브랜드 수출가 (FOB) | — | 기준 100 |
| 수입사 마진 | 10~15% | 약 115 |
| 도매상 마진 | 15~25% | 약 140 |
| 소매점 마진 | 30~40% | **약 195~200** |
| 최종 소비자 가격 | — | **약 2배** |

> 한국 소비자 판매가 1만원 제품이 일본에서는 2,000~2,500엔(약 2만~2만5천원)에 판매되는 구조.

---

## 드럭스토어 납품 5단계 (약 5~6개월)

```
1단계: 초기 접촉 (약 1~2개월)
   └─ 이메일·전화로 수입사 또는 도매상에 회사 소개
   └─ 제품 샘플 발송 (유상/무상 협의)
   └─ 주의: 해외 바이어 전용 연락처 별도 확보 필요

2단계: 시장성 평가 (약 1~2개월)
   └─ 도매상·수입사 직영점포에서 테스트 판매
   └─ 점장 평가 기반 시장성 판단

3단계: 도매상 협의 (약 1개월)
   └─ 공급가 협의 (역산 방식)
   └─ 배타적 유통권 여부 논의

4단계: 현지화 작업 (약 1~2개월)
   └─ 패키지 일본어 전성분 표기 (JCIA 기준 명칭)
   └─ 판매원(수입사) 표기
   └─ 집기(POP) 디자인 제작 (종이 재질 권장)
   └─ 테스트 샘플 제품 비치 계획

5단계: 마케팅 준비 및 론칭
   └─ YouTube, Instagram, TikTok 일본어 콘텐츠
   └─ 일본 인플루언서 협업
   └─ 패션잡지 기사화 검토
```

## 가격 붕괴 방지 전략

일본 소비자의 해외 직구 증가로 한국 판매가격이 노출되면 오프라인 바이어의 가격 인하 압박이 생긴다.

| 전략 | 설명 |
|------|------|
| 일본 전용 패키지 | 현지화 SKU로 직구 제품과 외관 차별화 |
| 전용 용량·구성 변경 | 일본 전용 사이즈 또는 구성품 |
| MAP 정책 | Minimum Advertised Price — 최저 판매가 기준 설정 |
| 집기 현지화 | 오프라인 매장 POP(종이 재질 권장) 일본어 완비 |

## K-beauty 재고 모델 3가지

### 1. 보유 재고 모델 (수입사/도매상)
- 도매상이 일괄 수입 후 창고 보관
- 소매점 주문에 따라 분할 납품
- **장점**: 브랜드사 리스크 낮음
- **단점**: 실시간 재고 가시성 낮음

### 2. 위탁 재고(Consignment) 모델
- 브랜드사가 재고를 수입사에 위탁, 판매 성과에 따라 정산
- **장점**: 도매상 입장에서 진입 장벽 낮춤
- **단점**: 재고 미판매 시 브랜드사 부담

### 3. 드롭십(Dropship) 모델 (온라인 채널)
- 소비자 주문 → 도매상(또는 브랜드사) 창고에서 직접 배송
- 소매점 재고 없음, MOQ 없음
- **적합**: 라쿠텐, Amazon Japan 등 온라인 마켓플레이스

## 재고 관리 KPI

| 지표 | 설명 | 관리 목적 |
|------|------|----------|
| 재고 회전율 | 월 판매량 / 평균 재고량 | 과잉 재고 및 품절 방지 |
| 리드타임 | 발주 후 입고까지 소요 기간 | 재발주 시점 설정 |
| 안전재고 | 리드타임 동안의 예상 판매량 | 품절 방지 버퍼 |
| 셀스루율 | 납품 수량 대비 판매 비율 | 소매점 성과 평가 |
| 반품율 | 납품 대비 반품 비율 | CS 이슈 조기 감지 |

## 시즌별 재고 관리 전략

| 시즌 | 특이사항 | 대응 |
|------|----------|------|
| 골든위크 (4월 말~5월) | 외국인 관광객 급증, 인바운드 수요 | 선행 재고 확보 |
| 여름 (6~8월) | 선케어·수분 제품 수요 증가 | 시즌 제품 중심 재고 확보 |
| 연말 (12월) | 기프트셋 수요 증가 | 선물 포장 SKU 준비 |
| 연초 세일 (1월) | 소진 재고 정리 시즌 | 과재고 사전 조정 |

## 브랜드사 SCM 실무 체크리스트

- [ ] 수입사/도매상과 재고 가시성(Inventory Visibility) 계약 조건 확인
- [ ] 일본 내 최소 1~2개월치 안전재고 유지
- [ ] 시즌 피크 전 선행 발주 계획 수립
- [ ] 반품·불량 처리 계약 조건 사전 확인
- [ ] 온라인·오프라인 가격 정책 일관성 유지 (가격 붕괴 방지)
- [ ] 해외 바이어 전용 연락처 별도 준비 (일반 대표 번호 연결 안 됨)
- [ ] 한국 허용 성분 → 일본 금지 가능성 사전 성분 검토
