# ORBIS × DATAFLUCT

일본 중견 화장품 D2C 브랜드 오르비스(ORBIS)와 ML SaaS DATAFLUCT의 박스 최적화 도입 사례. 주문의 약 15%에서 박스 다운사이징 실현, 연간 약 2,000만엔 배송비 절감 효과 확인.

## 기본 정보

- **ORBIS**: 폴라·오르비스 그룹의 화장품 D2C·EC 직판사, 다품종 소량 출하
- **DATAFLUCT Perswell**: 머신러닝 SaaS, AirLake 데이터 플랫폼 기반

## 도입 전 문제

EC 발송 시 시스템이 **상품 사이즈 기준으로 큰 박스를 자동 선택**해 운송비 과다. "2024년 문제"(트럭 운전수 시간외근로 규제) 대응 필요.

## AI 도입: DATAFLUCT Perswell

[[DATAFLUCT_Perswell]] (ML SaaS) + AirLake 데이터 플랫폼. 상품·출하·포장재 가격 데이터 학습. **의류 접기·공기빼기 같은 사람의 노하우까지 모델에 반영**.

## 결과

- 주문의 **약 14.96%에서 박스 다운사이징 실현**
- 시뮬레이션 단계에서 **연간 약 2,000만엔 배송비 절감** 효과 확인
- 2023년 일본 내 全 EC 발송 거점 본격 도입

## LAKA/JT 시사점

수요예측보다 **도입 장벽이 훨씬 낮고 정량 ROI가 즉각적으로 보이는 영역**. 도매 제거 후 소매 직거래 전환으로 SKU 폭증·소량 다빈도 출하가 늘면, 박스 최적화 AI는 **"AI 도입 첫걸음 — 6개월 ROI 입증"** 용도로 가장 적합하다. 일본 트럭 인건비·2024년 문제로 운송비 압력이 지속되는 환경 변수도 동일하게 적용.
