# ODM 발주 프로세스

> 코스맥스·한국콜마 등 주요 ODM사 기준. 브랜드사 SCM 담당자 실무 참고용.

---

## ODM 발주 전체 흐름 (10단계)

```
1. 컨셉 기획
   └─ 브랜드사: 제품 카테고리, 타깃, 클레임, 가격대 결정

2. ODM사 제안 또는 발주 의뢰
   └─ 기존 베이스 포뮬라 활용 (빠른 출시)
   └─ 또는 맞춤 개발 의뢰

3. 샘플 개발
   └─ ODM사: 처방 설계, 샘플 제조 (1차, 2차, N차 반복)
   └─ 브랜드사: 색상·텍스처·향·안전성 확인

4. 처방 확정
   └─ 성분표(INCI 목록) 확정
   └─ 수출 대상국 성분 규제 검토 동시 진행

5. 시험·인증
   └─ 안정성 시험 (가속·장기)
   └─ 미생물 시험
   └─ 중금속 시험
   └─ 기능성화장품 해당 시 → 식약처 심사/보고

6. 제조 계약 체결
   └─ MOQ (최소발주량) 협의
   └─ 단가, 납기 확정

7. 아트워크·포장재 확정
   └─ 라벨 규제 검토 (국내/수출국)
   └─ 인쇄 교정 (Color Proof)

8. 제조 발주 (PO 발행)
   └─ 브랜드사 → ODM사 발주서 발행

9. 제조 및 품질 검사
   └─ 원료 입고 검사 → 생산 → 인공 검사 → 완제품 QC
   └─ COA 발행

10. 납품 및 서류 전달
    └─ 완제품 납품
    └─ COA, MSDS, INCI 성분표, GMP 인증서 전달
```

## 리드타임 구조

| 구분 | 코스맥스 | 한국콜마 |
|------|---------|---------|
| 기존 포뮬러 활용 (ODM) | **45~60 업무일** | **45~60 업무일** |
| 신규 포뮬러 개발 (OEM/커스텀) | **60~90 업무일** | **60~90 업무일 이상** |
| 색조 (색상별 추가) | AI 조색 시스템으로 일부 단축 가능 | 색상별 별도 개발 기간 추가 |

> **수출 시 납기 역산**: 리드타임 + 통관 기간(2~4주) + 채널 입고 처리(1~2주) = 총 70~120일

## MOQ 결정 구조

화장품 MOQ는 **용기 MOQ가 가장 큰 제약**이 된다.

```
용기 MOQ (2,000~5,000개) ← 가장 높은 제약
   ↓
충진(내용물) MOQ
   ↓
박스·포장 MOQ (상대적으로 유연)
   ↓
발주 가능 최소 수량 = 위 항목 중 가장 높은 수량
```

### 제품별 MOQ 예시

| 제품 유형 | 일반 MOQ | 소량 생산 |
|----------|---------|----------|
| 스킨케어 (대형 브랜드) | 10,000개 이상 | 1,000개 (마이크로 론칭 서비스) |
| 립 틴트 (색상별) | **5,000개 내외** | — |
| 마스크팩 | 30,000개 이상 | — |

### 재발주 판단 기준

| 지표 | 설명 |
|------|------|
| 재고 회전율 | 채널별 월 판매량 대비 재고 수준 |
| 셀스루율 | 발주 수량 대비 실 판매 수량 비율 |
| 리드타임 기준 | 재발주 시점 = 안전재고 ÷ 일평균판매량 이상 남을 때 |
| 채널 요청 | 세포라·Boots 등 바이어의 리오더 요청 |
| 프로모션 계획 | 시즌 행사·캠페인 전 선행 발주 |

**안전재고 공식**:
```
안전재고 = 일평균판매량 × 리드타임(일) × 안전계수(1.2~1.5)
재발주점  = 안전재고 + 일평균판매량 × 리드타임(일)
```

## 코스맥스 vs 한국콜마 비교

| 항목 | **코스맥스** | **한국콜마** |
|------|------------|------------|
| 설립 특이사항 | 한국 최대 화장품 ODM 기업 중 하나 | 한국 최초 ODM 전문 기업 (1990년 설립) |
| 주요 강점 | 색조·스킨케어 전반 / AI 조색 시스템 | 기초·스킨케어 / ODM 방식 최초 도입 |
| 온라인 플랫폼 | 자체 E2E 플랫폼 (비대면 발주) | **플래닛 147** (소규모 브랜드 대응) |
| 소량 생산 | 마이크로 론칭 서비스 (1,000개~) | 플래닛 147 통해 소규모 대응 |
| 해외 법인 | 중국, 미국, 태국 등 | 중국, 미국, 캐나다 등 |
| 특이사항 | AI 조색으로 리드타임 단축 중 | 수출 브랜드 규제 서류 지원 가능 |

## 필수 납품 서류 체크리스트

### ODM사 → 브랜드사 제공

| 서류 | 내용 | 비고 |
|------|------|------|
| **COA** (Certificate of Analysis) | 완제품 분석 성적서 | 제조 후에만 발행 가능 |
| **INCI 성분표** | 전성분 목록 (함량 포함) | 핵심 서류 |
| **MSDS** | 물질 안전 보건 자료 | 항공 운송 시 필수 |
| **GMP 인증서** | ISO 22716 또는 CGMP | — |
| 안정성 시험 성적서 | 가속/장기 안정성 결과 | — |
| 방부력 시험 성적서 | Challenge Test 결과 | — |
| 기능성 심사 서류 | 기능성화장품 해당 시 | — |

### 수출 지역별 추가 서류

| 지역 | 추가 서류 |
|------|----------|
| **미국** | FDA 시설 등록번호, 제품 리스팅 확인, SAE 체계 구축 확인 |
| **영국** | SCPN 신고 확인서, UK RP 계약서, PIF 요약본 |
| **일본** | 화장품 제조판매 신고서, 성분 적합성 검토 결과, 일본어 라벨 |
| **EU** | CPNP 등록 확인서, CPSR, PIF, EU RP 지정 서류 |

## 안정성 시험 항목

| 시험 종류 | 조건 | 기간 | 목적 |
|---------|------|------|------|
| **장기 안정성** | 25°C / 60% RH | 12~24개월 | 유통기한 최종 확인 |
| **가속 안정성** | 40°C / 75% RH | 6개월 | 장기 데이터 예측 |
| 스트레스 테스트 | 50°C / 냉동(-15°C) | 1~3개월 | 극한 조건 저항성 |
| 사이클 테스트 | 4°C ↔ 40°C 반복 | 수 주 | 온도 변화 저항성 |

### 미생물 기준값

| 항목 | 기준 |
|------|------|
| 총 호기성 생균수 (일반) | 1,000 CFU/g(mL) 이하 |
| 총 호기성 생균수 (눈·어린이용) | 500 CFU/g(mL) 이하 |
| 진균수 | 100 CFU/g(mL) 이하 |
| 대장균 | **불검출** |
| 녹농균 | **불검출** |
| 황색포도상구균 | **불검출** |

## 브랜드사 담당자 체크포인트

1. **계약서**: 변경 절차(포뮬러, 수량, 납기) 명확히 기재
2. **비용 구조**: 개발비·생산비·패키지비·물류비 분리 여부 확인
3. **결제 조건**: 선급금 비율(T/T 30% deposit 일반적), 잔금 납기 조건
4. **MOQ 기준**: 초기 투자비(MOQ × 단가) 사전 시뮬레이션
5. **변경 관리**: 포뮬러 또는 수량 변경 시 추가 비용·납기 지연 계약서 명시
6. **QC 기준**: 불량률 허용 범위, 재작업 기준 사전 합의
7. **수출 서류**: ODM사가 FDA, SCPN, CPNP 서류 지원 가능 여부 사전 확인
