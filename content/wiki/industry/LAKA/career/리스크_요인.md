# 리스크 요인

[[LAKA]] 이직 과정에서 솔직하게 인식해야 할 약점과 위험 요소.

## 지원자 측 리스크

### 1. 게임 → 뷰티 업종 전환

- **문제**: 뷰티/화장품 업계 직접 경험 전무
- **심각도**: 높음
- **대응**:
  - 소비재 경험 강조 (와인바 창업, 에그드랍, PB 상품 개발)
  - B2B 협상 역량은 업종 불문 전이 가능
  - "뷰티 도메인 지식은 빠르게 학습 가능, 구조적 역량은 즉시 활용 가능"
  - LAKA 제품 직접 사용·분석 → 면접 시 제품 이해도 시연

### 2. 해외사업 경험이 게임 IP 중심

- **문제**: KOEI TECMO IP 협상 ≠ 세포라 바이어 협상
- **심각도**: 중간
- **대응**:
  - IP 라이선스 협상과 채널 파트너십 협상의 **구조적 유사성** 설명
  - 로열티 구조, 지역 독점권, 마케팅 공동투자 등 공통 요소 강조
  - 다만 **리테일 바잉 사이클**, 매대 관리, POS 데이터 분석은 학습 필요

### 3. LAKA 규모(90명) vs 이전 직장 규모 차이

- **문제**: LINE GAMES, 넥슨 네트웍스 등 대규모 조직 경험 → 90명 소규모 적응
- **심각도**: 낮음~중간
- **대응**:
  - 넥슨 네트웍스 초기(소규모 시절) 경험
  - 에그드랍 등 소규모 F&B 운영 경험
  - "큰 조직에서의 체계적 경험 + 작은 조직에서의 실행력 모두 보유"

### 4. 연봉 수준 불확실성

- **문제**: 인디 뷰티 브랜드의 연봉 수준이 대기업/게임사 대비 낮을 가능성
- **심각도**: 중간 (개인 상황에 따라)
- **대응**:
  - 사전 시장 조사 (원티드, 잡플래닛 연봉 정보)
  - 기본급 외 스톡옵션, 성과급, 복리후생 패키지 확인
  - 연봉 협상에서 "복합 포지션의 시장 가치" 논리 활용

## 회사 측 리스크

### 5. 이지철 체제 안정성

- **문제**: [[이지철]] 대표 취임 1년 미만, [[구다이글로벌]] 소유권 전환 직후
- **심각도**: 중간~높음
- **우려 사항**:
  - 경영 방향 급변 가능성
  - 기존 구성원과의 갈등
  - 구다이글로벌의 추가 개입 가능성
- **대응**:
  - 입사 전 이지철 대표와 직접 면담으로 비전·방향 확인
  - 경영진 교체 빈도, 조직 문화 변화 사전 조사
  - ⚠️ 정보 제한적 — 내부 분위기 파악 어려움

### 6. 11개월 채용 공백의 의미

- **문제**: 장기간 채용을 하지 않은 이유가 불분명
- **가능 해석**:
  - 긍정적: 조직 정비 기간, 전략 수립 후 채용 재개 예정
  - 부정적: 재정 긴축, 성장 둔화, 내부 혼란
- **대응**: 아웃리치 시 자연스럽게 확인, "조직 확장 계획이 있으신지" 질문

## 종합 판단

업종 전환과 이지철 체제 안정성이 가장 큰 리스크. 그러나:
- 업종 전환은 **내러티브와 준비**로 상당 부분 극복 가능
- 이지철 체제 리스크는 **직접 면담**으로 판단해야 함
- LAKA의 **수익성(OPM 30%)**과 **글로벌 전환점(세포라 입점)**은 기회 요인으로 리스크를 상쇄
