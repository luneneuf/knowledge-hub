# 메종 드 서울 (Maison de Seoul)

## 요약

일본인 운영자(Yoonji)가 "서울을 일본에 가져온다"는 콘셉트로 만든 **온라인 K-뷰티 부티크**. 오너가 직접 선별한 한정 브랜드 큐레이션이 핵심. 대형 유통 채널이 아닌, 일본 코어 K-뷰티 팬층에게 "신뢰성 있는 큐레이션 채널"로 인지되는 니치 커머스. 규모보다 팬덤·LTV 중심의 구조.

## 핵심 데이터

| 항목 | 내용 |
|------|------|
| 운영자 | Yoonji (일본인 개인 운영) |
| 채널 타입 | 온라인 K-뷰티 부티크 (자체몰 + SNS) |
| 취급 품목 | 한국 스킨케어·메이크업 + 라이프스타일 소품 |
| 재무 공시 | 없음 (개인 사업 수준) |
| 대표 베스트셀러 브랜드 | [[디어클레어스]] (Klairs) — 오픈 후 한 달 내 베스트셀러 상위 |

## 포지셔닝

- "서울을 큐레이션해 일본에 가져온다"는 감성 커머스.
- SNS·블로그 기반 인플루언서 성격의 커머스 구조.
- 규모보다 **큐레이션의 신뢰성·콘텐츠·반복 구매(LTV)** 중심.

## 한국 브랜드 입장 전략적 의미

- **적합 브랜드**: 일본 대형 유통에 아직 진입하지 않은 초기~성장 단계 K-뷰티. 스토리텔링·패키지 감성이 강한 브랜드.
- **활용 방법**: "대형 유통 전 단계 테스트베드 + 인플루언서 채널"로 활용.
- 일본 코어 K-뷰티 팬덤·리피트 고객 형성에 유리.
- 일본어 콘텐츠·리뷰·SNS 협업 필수.
