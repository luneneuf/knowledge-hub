# FINN

독일 뮌헨 자동차 구독 스타트업. Rossum Aurora IDP 도입으로 인보이스 처리 시간을 일 2시간에서 15분으로 단축. 중견기업 구매 AI 도입의 "단일 use case로 시작하는" 패턴의 대표 사례.

## 기본 정보

- **소재**: 독일 뮌헨
- **사업**: 자동차 구독 서비스
- **규모**: 2024년 매출 €160M, 약 25,000건 차량 구독

## 도입 전 문제

차량 라이프사이클 동안 약 **200개 공급업체**로부터 PDF 인보이스가 들어오고 포맷이 모두 달랐다. 'Total amount' / 'Brutto' / 'Balance due'가 같은 의미로 쓰이고, 기존 **regex 기반 OCR로는 안정적 추출 불가**. 회계팀 매일 약 2시간 수기 처리.

## AI 도입: Rossum Aurora

[[Rossum_Aurora]] (LLM 기반 IDP). 3단계 인지 흐름(skim-reading → 필드 위치 파악 → 정밀 추출)으로 템플릿·regex 없이 변동 포맷 처리. 신뢰도 **0.8 이상은 무인 처리**, 미만은 검증 큐로 라우팅. FINN이 도메인 특화 학습(VIN·차량 번호판 추출)을 추가 적용.

## 결과

| 지표 | 수치 |
|------|------|
| 문서당 처리 시간 | **71% 단축** |
| 평균 검증 시간 | **22초/건** |
| 일일 처리 시간 | **2시간 → 15분** |
| 추출 정확도 | **6주 내 90%+** |
| MVP 셋업 | **2일** |

## 구매AI 시사점

**200개 공급업체 = 브랜드 7-8개의 100배 규모**에서 작동한다. MVP 2일, 6주 내 90% 정확도는 도입 속도 기준이 된다. 구매팀이 직접 공급업체 추가(1클릭 마스터 등록)하는 Human-in-the-loop 설계가 핵심.
