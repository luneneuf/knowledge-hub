# Sociolla (소시올라)

인도네시아 No.1 뷰티 옴니채널 — **Social Bella 그룹** 산하 EC + 60+ 오프라인 매장. LAKA 입장에서는 **'26-2월 신규 빅 라인이자 동시에 부진재고(C등급) 클리어런스 시범 채널** — 26년 8월까지 셀-아웃 추이로 SKU 단종 여부 결정.

## 회사 개요 — Social Bella 그룹

| 항목 | 내용 |
|---|---|
| 모회사 | **PT Social Bella Indonesia** |
| 설립 | 2015년 |
| 본사 | 인도네시아 자카르타 |
| 누적 펀딩 | **$226M+** (Temasek, L Catterton, EV Growth, East Ventures, Jungle Ventures 등) |
| 밸류에이션 | **$564M+** (2022 라운드 후, 유니콘 목표 협상) |
| 사용자 | 30M+ (2022 기준), 42M (2020) |
| 매출 | 2020 약 $43.5M (전년 대비 거의 2배) |
| SKU·브랜드 | **400+ 브랜드, 12,000+ 제품** |

### 비즈니스 라인업 (5개 사업부)

1. **Sociolla** — 인도네시아 No.1 뷰티 EC + 옴니채널 매장
   - **인도네시아 40개 도시에 60+ 매장**
   - **베트남 4개 도시에 12 매장**
2. **SOCO** — 뷰티 슈퍼앱 (커뮤니티·리뷰·EC 통합)
3. **Beauty Journal** — 뷰티·라이프스타일 미디어
4. **Lilla** — 엄마 대상 EC (2022 첫 매장)
5. **SDI (Social Distribution Indonesia)** — End-to-end 디스트리뷰션 사업부 (브랜드 입점 컨설팅·물류)

> **시사점**: LAKA는 SDI를 통해 입점했을 가능성이 높음. SDI는 글로벌 브랜드를 Sociolla 매장·EC + 베트남 진출까지 풀-서비스 패키지로 묶음. 단순 마켓플레이스 입점이 아닌 **컨설팅·디스트리뷰션 일괄 위탁**.

## 핵심 재해석 — Sociolla = 부진재고 클리어런스 채널 ⭐

03-W1 SCM 보고를 보면 Sociolla 입점은 단순 신규 시장 확대가 아닌 **부진재고 TF 활동**의 일환:

| 등급 정의 (03-W1 §6) | 처리 |
|---|---|
| **S, A 등급** | 정상 판매 |
| **신제품** | MOQ 발주 / 누적 재고금액 조정 |
| **B 등급** (월평균 500개 미만) | 월평균 × 3개월 수준 유지, 일시적 품절 허용 |
| **C 등급** (영업 니즈 낮고 1년 MOQ 커버 불가) | **순차적 단종 처리 (대표님 컨펌 완료)** — 예: **팝핑벌룬틴트 본품·미니 전품목, 젤링누드글로스 미니 전품목, 스킨4코렉터 전품목** |

→ **C등급 단종 SKU의 마지막 시장 테스트 채널이 Sociolla(인도네시아) + POYA(대만)**. 8월까지 셀-아웃 미달이면 그대로 단종.

이 컨텍스트는 같은 부진재고 TF의 다른 두 협상 라인과 평행:
- **Adeline Group** (북미·유럽·오세아니아 K-Beauty 입점 컨설팅, 글로벌 뷰티 유통 15% MS) — 3/5 LAKA 본사 미팅
- **TJ Maxx** (미국 오프프라이스 1,333 매장) — 단상자 리패키징·후킹 라벨 부착 협상

## [[Sell-in_Sell-out|셀-인 vs 셀-아웃]] 관점

| 단계 | 데이터 | 해석 |
|---|---|---|
| 셀-인 (LAKA → Sociolla) | '26-2월 5.75억 | **확정 매출** — 클리어런스 SKU 대량 출하 |
| 셀-아웃 (Sociolla → 소비자) | **블랙박스** | Sociolla 매장·EC 매대 회전율 미공유 |
| 8월 cutoff 판정 | **셀-아웃 추이가 결정 변수** | 미달 시 LAKA 측에서 SKU 단종 |

> **운영 시사점**: Sociolla 셀-아웃 모니터링 KPI 필요 — Beauty Journal·SOCO 리뷰 카운트, 매장 진열 잔존율, 재주문 PO 발생 여부. 본부 측 셀-아웃 데이터 협의 또는 [[안전관리정보_자동수집_제안|자동수집 파이프라인]]에 SOCO 리뷰 출처 추가 검토.

## 시사점

1. **부진재고 채널과 신규 빅 라인이 같은 채널**: Sociolla는 시장 확대 + 부진재고 클리어런스 듀얼 성격. 단순 "신규 빅 라인 800M"으로만 보면 안 됨.
2. **8월 cutoff = 결정 트리거**: C등급 SKU 단종 여부가 Sociolla 셀-아웃에 직결. 셀-아웃 가시성 확보가 가장 중요한 KPI.
3. **계획-실적 갭 (4→2→5.75억) 점검**: SCM 예측 정확도 또는 PO 추가 발생 메커니즘 본부장 컨펌.
4. **POYA·Adeline·TJ Maxx 동시 wiki화 필요**: 부진재고 TF 4채널 통합 관점에서 비교 분석 가치.
5. **SDI 활용 시 베트남 동시 진출 옵션**: Sociolla 베트남 12 매장은 추가 채널 확장 카드.
