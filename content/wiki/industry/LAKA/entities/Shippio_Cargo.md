# Shippio Cargo

일본 디지털 포워딩 SaaS. 화물 트래킹 + 수입 서류 관리 + AI 인보이스 자동 대조 기능을 통합. 일본 NACCS 통관 자동 연계로 일본 수입 업무 전체를 클라우드에서 처리.

## 핵심 기능

| 기능 | 내용 |
|------|------|
| 화물 트래킹 | 본선 동정 자동화, 수동 ETA 추적·Excel 갱신 폐지 |
| 서류 관리 | B/L·Invoice·Packing List 클라우드 통합 관리 |
| AI-OCR 대조 | 인보이스 자동 대조, 정확도 97%+, [[IDP_인텔리전트_문서처리]] 기술 기반 |
| NACCS 연계 | 일본 수출입 통관 시스템 자동 연계 — 일본 시장 필수 요건 |

## 검증 사례

|------|------|

## LAKA/JT 적용 포인트

- 한국 본사 7-8개 브랜드의 서로 다른 인보이스 포맷을 AI-OCR이 단일 처리 가능
- 일본 NACCS 표준 연계는 일본 시장에서 사실상 필수
- 한국 측 트레드링스(B/L 추적 정확도 99.5%)와 페어링하면 부산→일본 전 구간 단일 가시성 확보
- 12-18개월 로드맵의 **1단계(0~3개월)** 선행 도입 권장 솔루션
