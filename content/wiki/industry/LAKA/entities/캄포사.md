# 캄포사

LAKA **원료 공급사 (간접)** — [[한국콜마]]가 [[수딩-비건-립-오일]] 제조에 사용한 색소 원료 공급처. 2026 Q1 **수단레드(Sudan Red) 검출 의혹**의 발원지. 후속 정밀 정보 수집 필요한 stub 페이지.

## 거래 구조 (특이)

- **라카는 캄포사와 직접 거래 없음** — [[한국콜마]]가 ODM 자체 조달 원료
- 그러나 책임판매업자로서 라카가 **1차 식약처 대응 + 회수 비용 중간 송금** 부담
- 한국콜마 → 라카 → [[Amiid]] 송금 라인 (한국콜마-Amiid 직거래 미계약)

## 수단레드 (Sudan Red) 배경

- **공업용 색소** (Sudan I~IV 등) — 식품·화장품 사용 금지
- 한국·EU·일본 등 대부분 시장 사용 금지
- K-뷰티 색조 (특히 립 카테고리)에서 과거 사례 다수 — 일부 색소 원료 오염 경로
- 식약처 자체시험 결과로만 판정 (외부 시험성적서 무효 — [[수딩비건립오일_수단레드_대만회수_2026Q1]] 참조)

## 정보 갭 (후속 조사 필요)

- 회사 정식 사명 / 영문명 → 확인 필요 (캄포사가 약칭일 가능성)
- 본사 위치 (한국 / 해외 / OEM 경유)
- 공급 원료 카테고리 (색소 단일인지, 다른 원료도 공급하는지)
- 한국콜마 내 사용 비중 / 다른 ODM에도 공급하는지
- 라카 외 영향 받은 다른 K-뷰티 브랜드
- 검출 불검출 후 캄포사 측 후속 대응 (원료 변경 / 공정 점검 등)

## 시사점

1. **원료 공급사 리스크 = ODM 리스크 = 라카 리스크** — 직접 거래 없어도 책임판매업자로서 1차 대응. 원료 공급사 리스크 스코어링 ([[공급업체_리스크_스코어링]])에 ODM 원료 공급사까지 포함 검토
2. **K-뷰티 색조 색소 원료 = 반복 리스크** — 수단레드는 한 번의 사고가 아닌 업계 반복 패턴. 색조 SKU 사전 검출시험 정례화 가치
3. **[[차승호_1차미팅_아젠다]] 처방서 25종 향료 알레르겐 명시 영역과 연계** — 처방서에 사용 원료 + 공급사 + 시험 성적 명시 강화 필요
4. **stub 상태 = 후속 deep-research 우선순위 高** — 본 이슈가 라카 2026 Q1 최대 안전성 사건 中 1. 외부 정보 보강 필요
