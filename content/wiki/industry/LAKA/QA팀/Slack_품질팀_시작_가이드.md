# 품질팀 Slack 시작 가이드

> 차승호 팀장·김건 책임 두 분 모두 Slack 처음 사용. 1차 미팅에서 1~2분 시연으로 충분히 익혀지는 핵심 5가지만 정리. 본격 운영 룰은 [[품질팀_슬랙운영_초안_2026-05]] 참조.

## 1. 메시지와 댓글로 이어쓰기 (스레드)

- 채널에 새로 글을 올리면 다른 사람들이 다 봅니다 → "**메시지**"
- 그 메시지 아래 댓글로 이어 답할 수 있습니다 → "**스레드**" (Thread)
- 스레드를 쓰면 채널 메인 화면이 깨끗해지고, 같은 이슈는 한 곳에 묶입니다

**클레임 처리 8단계를 예로**:

```
[새 메시지] 클레임 접수 — 제품 A 로트 B12345 (1단계 접수)
  └─ [스레드 답글] 2단계 검토 완료, 제조사 동일 로트 관리품 확인 중
  └─ [스레드 답글] 3단계 개선지시서 작성 첨부
  └─ [스레드 답글] 4단계 제조업체 통보 완료
  └─ ...
```

→ 카톡 단체방의 "↑ 위에 어디서부터 봐야 하지" 문제가 해소됩니다.

## 2. 특정 사람 호출 (멘션)

- 메시지 안에서 `@` 입력 후 이름 선택 → 그 사람한테 즉시 알림이 갑니다
- 멘션 안 하면 채널 사람들이 알림 못 받을 수 있으니, 답변 필요하면 멘션 권장

**예시**: `@차승호 위 클레임 2차 검토 부탁드립니다`

## 3. 검색

- 화면 상단 검색창에 키워드 입력 → 모든 채널·메시지·파일 검색
- 카톡과 달리 메시지를 잃지 않습니다. 1년 전 클레임도 키워드로 찾을 수 있음

## 4. 채널 상단 안내 (핀고정)

- 각 채널 상단에 운영 룰·자주 쓰는 링크 고정 가능 → "**핀고정**"(Pin)
- 본부장이 채널 신설 시 핀고정 메시지로 채널 용도·정형 보고 양식을 정리해둠
- 신규 멤버가 들어와도 핀고정 메시지부터 보면 됨

## 5. 비밀 채널 vs 공개 채널

- **공개 채널** (Public): 다른 부서 사람도 검색하면 보임. 합류 자유
- **비밀 채널** (Private): 멤버만 보임. 검색해도 안 보임. 인사·비용·계약 정보용

품질팀 5개 채널 중 **#qa-팀내**만 비밀 채널. 나머지 4개는 공개.

## 첫 주 권장 사용법

1. **카톡 단체방에서 하던 그대로** Slack 채널에 옮겨 쓰기 (메시지 + 답장)
2. 클레임 진행 단계만 **스레드**로 묶기 (위 예시)
3. 큰 이슈 발생 시 **본부장 멘션** (`@천승범`)
4. 막히는 부분은 본부장한테 채팅으로 물어보기 (5/26~6/2 1주 적응 기간)
5. 6월 첫 주 회고에서 **운영하면서 거부감·보완점** 자유롭게 얘기 — 채널 통폐합·신설·룰 조정

## 핀고정 메시지 템플릿 (본부장이 채널 신설 시 게시)

### #qa-팀내

```
📌 #qa-팀내 (비밀 채널)
멤버: 천승범 본부장 / 차승호 팀장 / 김건 책임 3명만
목적: 일상 조율·인사·일정·내부 공유
다른 부서 검색에 안 보임 — 안심하고 사용
```

### #qa-신제품-출시검증

```
📌 #qa-신제품-출시검증
목적: 신제품·러닝품 QC, 라벨 문안 검토, 자재 견본 검토
멤버: 품질팀 + 프로덕트(BM) + 디자인 + 마케팅
정형 보고: 기존 SCM 주간보고 QC 양식 그대로 (제품명·로트·판정·특이사항)
```

(나머지 채널도 같은 형식)

## 자주 묻는 질문

**Q. 잘못 보낸 메시지 어떻게 지우나요?**
A. 메시지 위에 마우스 올리면 점 3개(⋯) 메뉴 → "삭제" 또는 "수정". 카톡과 동일.

**Q. 파일 첨부?**
A. 메시지 입력창 옆 클립(📎) 아이콘. 끌어다 놓기도 됨.

**Q. 알림이 너무 많은데?**
A. 채널 우측 상단 알림 아이콘 → "Mentions only" 선택 → 본인 멘션될 때만 알림.

**Q. 모바일에서도 쓰나요?**
A. Slack 앱(Android/iOS) 설치. 같은 워크스페이스 로그인.

**Q. 본부장한테 1:1로 말하고 싶어요.**
A. 좌측 사이드바 "Direct messages(DM)" → 본부장 선택 → 메시지. 다른 사람은 못 봄.
