## 3. 파일 네이밍 컨벤션 (ODM별)

각 ODM이 자체 양식. **표준화되지 않음** — R&R #8 QC 서류 관리에서 통합 검색 시 부담.

### 한국콜마 — 가장 표준화됨
```
CTC[YYMMDDNNNNN]-XX (라카코스메틱스) [SKU] [용기공급사]_[재질] C.T결과-V[N]([테스트유형])-CT연구팀-[YYMMDD].pdf
```
예: `CTC25120056-01 (라카코스메틱스) 프루티글램틴트#182,186 일창_리퀴드(솔대) C.T결과-V3(낙하)-251210.pdf`

옛 형식: `[5자리번호] (LAK) [SKU] C.T결과-V[N]-CT연구팀-[YYYYMMDD].pdf`

### 코스맥스 — 단순화
```
YYYYMMDD [주식회사 라카코스메틱스] [SKU] CT결과서.pdf
```
예: `20260210 [주식회사 라카코스메틱스] 라카 듀얼립 (색조틴트) CT결과서.pdf`

### 코스메카 — 자체 번호 + 자유 기술
```
YYYY-NNNN [SKU] [특이사항].pdf
```
예: `2025-2834 라카 톤업베이스 (다층브로우 박킹변경) (최종).pdf`

### 씨앤씨 — 변경 사유 명시
```
YYYY.MM.DD [라카] [SKU] [태그(용기 재질 변경/후가공 테스트/2차)].pdf
```

### 화성 — **판정 결과를 파일명에 직접 포함**
```
[SKU] ([판정]).pdf
```
예: `라카 스테인 립 라이너 (조건부적합).pdf`, `라카 립펜슬 1w_협의필요.pdf`

→ **화성 양식이 가장 LAKA에 친화적** (위키·검색에서 판정 상태가 즉시 노출).

## 4. 표준 CT 결과서 외 별첨 문서 카탈로그 (이슈/특이사항 트래커 후보 47건)

CT 결과서 본문이 아닌 **개별 이슈·테스트 결과·공유 문서** — R&R #1 클레임 처리·#11 부적합 처리의 사전 신호로 활용 가능.

### 🚨 미해결 또는 조건부 적합 케이스

| 파일 | ODM | 판정 | 이슈 |
|------|-----|-----|------|
| **라카 스테인 립 라이너 (조건부적합).pdf** (2025-11-03) | 화성 | **조건부 적합** | ①유기자차 도포 시 코팅 벗겨짐 ②역립 낙하 시 심빠짐. 고객사(LAKA) 11/03 승인 완료. **카트리지·바디·플러그 모두 ABS 재질** ← 코스맥스 가이드라인상 ABS 금지 |

### 🎯 트윈립 = 듀얼립 = 코스맥스 신제품 (2026-02-10 CT)

| 파일 | 내용 |
|------|------|

| `20260210 [LAKA] 라카 듀얼립(글로스) CT결과서.pdf` | 글로스 측 |

### 화성 ABS 재질 vs 코스맥스 가이드라인 충돌
- 코스맥스 LM1팀 유분산 disclaimer: ABS 금지 (장기간 내용물과 반응)
- 화성 스테인 립 라이너: 카트리지·바디·플러그·심집 모두 ABS·POM 사용
- → **5/19 미팅 트윈립 금형 이슈가 화성 스테인 립 라이너의 ABS 문제일 가능성** 도 배제 못 함 (단일 SKU이긴 하지만 "트윈" 명칭이 별개 SKU에서 사용됐을 수도)
- 차승호 회신 메일이 SKU를 명확히 짚어주면 즉시 확정
