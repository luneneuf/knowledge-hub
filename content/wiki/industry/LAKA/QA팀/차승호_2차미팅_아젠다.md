## 아젠다 #5 — 정부(외부 기관) 담당자 R&R 표 등재 검토 + R&R 문서 작성 배경

### 배경

[[품질관리_RR_2026-05]] R&R 표에는 **LAKA 내부 담당자(차승호·김건)** 만 명시되어 있고, 각 업무가 인터페이스하는 **외부 기관 측 담당자**(식약처 화장품정책과 주무관, KCA 사무국, 환경공단 EPR센터, KTR 시험기관 등)는 표에 등장하지 않습니다.

R&R 표 22개 업무 중 외부 기관 인터페이스가 명시적으로 있는 업무는 최소 8건:

| R&R # | 업무 | 외부 채널 |
|------|------|-----------|
| #5 | 화장품 원료목록보고 | [[대한화장품협회]] (KCA) — 원료목록보고 사이트 |
| #6 | 포장재 재질구조등급평가 | 환경공단 EPR |
| #8 | QC 서류 관리 (시험성적서) | KTR 등 시험기관 |
| #13 | 제조업체 관리 (정기·신규 평가) | ODM 5사 ([[한국콜마]]·코스맥스·화성·코스메카·씨앤씨) — R&R 표 외부 인터페이스 명시 있음 |
| #14 | 안전성 정기보고 | 식약처 (의약품안전나라) |

### 본부장 입장

본부장 부임 직후 첫 외부 인터페이스 이슈가 발생했을 때(예: 식약처 정기 감시, 환경공단 EPR 점검, KCA 신규 양식 변경 등), **본부장·차승호 부재 시 김건 책임이 즉시 연락 가능한 외부 담당자 마스터**가 필요합니다. 차승호 팀장 6년 운영 감각에 의존하는 정보를 명시화해두는 작업.

또한 5/18에 작성해주신 R&R PPT의 **작성 배경**(자발적 정리·본부장 부임 대응·대표 지시·식약처 감시·외부 트리거 등)이 명확해지면, 본부장이 R&R 표 갱신·확장 방향을 더 정확히 잡을 수 있습니다.

### 차승호 팀장 확인 요청 (3문항)

**Q1. 외부 담당자 마스터 보유 여부**
- 식약처 화장품정책과·KCA 사무국·환경공단 EPR센터·KTR 시험기관·외부 자문사(법무·인허가) 담당자(부서명·이름·연락처·이메일) 리스트가 어딘가에 정리되어 있는지?
- 정리되어 있다면 어떤 형태인지(엑셀·문서·차승호 개인 명함첩·메일함 검색)? 본부장도 동일 정보에 접근 가능한지?
- 정리되어 있지 않다면, 차승호 팀장 머릿속 또는 메일 이력에서 1차 추출 가능한지?

**Q2. R&R 표에 외부 담당자 컬럼 추가 의견**
- R&R 표에 **"외부 담당자" 열**을 추가하거나, **R&R 부속표**(외부 기관 마스터)를 별표로 두는 안 어떠신지?
- 책임판매업자 등록·정기보고·교육·EPR 업무별로 외부 채널이 분산되어 있어 통합 마스터가 있으면 본부장·김건 모두 도움. 다만 외부 담당자가 자주 바뀌면 유지 부담이 크니, **업무별 부서 단위 + 차승호 보유 개인 컨택은 별도 노트** 형태가 현실적인지 차승호 팀장 의견 듣고 결정.

**Q3 (사이드). R&R 문서 작성 배경** ✅ [회신: 2026-05-19 미팅 + 이관표 확인]
- ~~5/18 「품질관리 시스템 체계확립」 PPT 작성이 어떤 동기였는지~~
- **답**: 차승호 팀장이 **전직장부터 작성·운영해오신 양식**을 LAKA 컨텍스트에 맞춰 정리한 것. LAKA 자체 신규 작성이 아니라 기존 운영 양식의 이식·갱신본. 17개 R&R 구조는 차승호 팀장 6년+ 운영 노하우 누적의 결과.
- **추가 확증 (2026-05-19 이관표 발견)**: [[QA팀_이관업무_2026|이관업무 엑셀]] 마지막 ToDo "*품질팀 업무 이관 후 업무 R&R 작성*"의 결과물 = 5/18 R&R PPT. BM·구매·디자인·영업·해외영업·본부장 6개 부서로부터 15건 이관 완료 후 작성.
- **시사점**: 별첨 22개와의 갭은 LAKA 이식 단계에서 단순화 과정의 산물. 이관표엔 있는데 R&R 22개에 빠진 업무가 있음 — **🚨 화장품 생산실적 보고**. 차승호 회신 시 22개→23개 확장 가능성. Q1·Q2 (외부 담당자 마스터) 우선순위는 별도 결정 — SharePoint QualityManagement 어드민 권한자도 마스터에 포함.
- **✅ 2026-05-19 후속 확정**: SCM 주간보고 2026-03-W1 slide 9에 **2025년 화장품 생산실적 보고 완료** 명시 확인 (450품목 / 540아이템 / 1,380만개 / **348억원**, [[대한화장품협회]] 보고). → R&R 22개 표에서 **단순 누락**된 것이지 업무 자체는 운영 중. **22→23개 확장 확정**. 8년 시계열은 [[LAKA_생산실적_시계열_2018_2025]] 참조.

**Q4 (신규, 본 이관표에서 도출). 품질 매뉴얼 규정서 작성 진행 상태**
- 이관표 마지막 두 ToDo 중 ①번 "*품질팀 품질 매뉴얼 규정서 작성*"의 진행 상태가 미확인. ② R&R 작성은 완료(5/18 PPT). 매뉴얼 규정서는?
- 작성 중이라면 어느 단계인지, 작성 안 됐다면 본부장 부임을 계기로 일정 합의 (예: 6월 첫 주 회고에서 마감 협의)
- 본부장 입장: 매뉴얼 규정서가 있어야 R&R 표 운영의 실제 SOP·체크리스트가 일관됨. 우선순위는 5/22 Slack 신설·트윈립 금형 대응 다음.

## 아젠다 #6 — Slack 5채널 외 — 다른 팀과의 업무 커뮤니케이션 매핑 + 5/22 이전 이관 데이터 수집 🚨

### 배경

[[차승호_1차미팅_아젠다]] 아젠다 #2 / [[품질팀_슬랙운영_초안_2026-05]]에서 설계한 **5채널은 품질팀 내부 + 본부장 라인 중심**입니다. 그러나 5/22 Slack 일원화 후 실제로 발생할 트래픽의 상당 부분은 **다른 팀이 품질팀에게 보내는 문의·요청**:

- 영업팀(국내·글로벌) → 클레임 접수 기안, 수출 서류 요청 (COA·MSDS·CGMP·ISO)
- BM·프로덕트팀 → 신제품 출시 검증 요청, 원료목록 변경, 표시문안 검토
- 디자인팀 → 자재 표준견본·포장사양서 검토
- 구매팀(SCM/조민경) → 제조업체 audit, 정품여부 확인
- 물류팀(SC기획팀) → 정품여부 확인 (출고 정보 연동)
- 마케팅팀 → 광고문안·상세페이지 검토
- 글로벌사업부(RA파트 허승은·임혜인) → 미국·영국·일본 채널 컴플라이언스
- 법무·재무 → 인허가·계약·예산 인터페이스
- 대표실 → 식약처 위해정보·중대 클레임 에스컬레이션

[[품질관리_RR_2026-05]] §부서간 협업 인터페이스에 5개 부서만 명시되어 있고, 글로벌사업·법무·재무·마케팅·대표실 등은 R&R 표 외 영역. **이 트래픽이 어디로 들어와야 하는지** Slack 5채널 운영 룰에 반영해야 함.

### 본부장 입장

다음 운영 결정이 필요:
- 다른 팀의 문의·요청을 5채널 중 어디에 게시하도록 안내할지?
- 외부 팀원을 5채널 중 어디에 초대할지(전 채널 vs 사안별)?
- DM으로 들어오는 문의를 채널로 유도하는 룰(예: "DM 받으면 채널로 옮겨주세요" 핀고정 메시지)
- 다른 팀의 응답 SLA — 품질팀 24시간 내 1차 응답 룰 명시할지?
- Slack Workflow 또는 Form 도입 시점 (요청 유형별 정형 양식)

이 결정에 앞서 **현재 실태 데이터**가 필요 — 차승호 팀장·김건 책임의 메일함·단톡방에 흩어져 있는 정보를 본부장에게 흘려주시면 사례 기반으로 룰 만들 수 있습니다.

### 차승호 팀장 확인 요청 (3문항)

**Q1. 현행 다른 팀 → 품질팀 요청 경로 매핑**
- 현재 다른 팀에서 품질팀에 요청·문의를 보내는 경로는 — Teams 단톡방·카톡 단체방·1:1 카톡·이메일·대면·전화 — 어떤 게 있는지?
- 각 경로의 대략적 빈도(주 N건 수준)? 가장 무거운 경로는?
- 그 중 5/22 Slack 일원화 후 자연스럽게 Slack으로 옮겨갈 것 / 옮기기 어려울 것(예: 영업팀이 카톡 1:1로 클레임 접수 요청하는 관습)?

**Q2. 최근 1개월 다른 팀 요청 사례 포워딩**
- 차승호 팀장 또는 김건 책임의 메일함·단톡방에서 **최근 1개월(4/19~5/18) 다른 팀 요청 사례 5~10건**을 본부장에게 포워딩 또는 정리해주실 수 있는지?
- 형식은 자유 — 메일 전달, 캡처, 또는 표(요청 부서·내용·처리 방법·소요 시간) 모두 OK.
- 활용 목적: ① 5/22 이관 시 어떤 단체방·메일 스레드를 어느 Slack 채널로 옮길지 결정. ② Slack 운영 룰을 실제 사례 기반으로 만들기. ③ 가장 많은 요청 유형 식별 → 향후 Workflow·Form 도입 우선순위.

**Q3. 가장 많이 들어오는 요청 유형 Top 5 (체감)**
- 차승호 팀장 6년 운영 감각으로, 다른 팀에서 가장 많이 들어오는 요청 유형 Top 5는?
  - 예시 후보: COA 발행, 광고문안 검토, 표시문안 검토, 클레임 응대, 자재 견본 검토, 안전성 정기보고 자료 협조, 수출 서류 발급, 정품여부 확인, 신제품 QC 일정 문의 등
- 각 유형이 보통 어느 부서에서 누구로부터 들어오는지(부서 + 가능하면 사람)?
- 처리 표준 시간(즉시·반나절·1일·1주 등)?

## 아젠다 #10 — CT·금형 영역 후속 진단 (#4 부속 + 후속메일 4번 트윈립 묶음)

> **출처**: [[CT결과서_폴더_2026]] 2026-05-19 진단 — 191건 파일·5 ODM·45 SKU 분석 결과
> **연결**: [[차승호_1차미팅_후속메일_2026-05-19]] **4번 트윈립 금형 이슈**(오늘 퇴근 전 회신 요청)와 묶어 처리

### 배경

CT결과서 폴더 진단에서 3가지 추가 후속 의제 발견:

1. **트윈립 정체 가설** — 코스맥스 듀얼립 공유사항 PDF 본문에 "**트윈립 제형 역립 2주**" 직접 명시. 듀얼립 = 트윈립일 가능성 80%. 5/19 미팅에서 차승호가 언급한 "트윈립 금형 이슈"가 듀얼립 양면 펜 메커니즘 + 4가지 제형 이슈(크렉·오일뜨기·누액·빈공간)인지 확인 필요

2. **🚨 화성 ABS 재질 ↔ 코스맥스 가이드라인 충돌** — 화성 스테인 립 라이너(조건부적합·2025-11-03 LAKA 승인) 카트리지·바디·플러그 모두 ABS. 코스맥스 LM1팀 유분산 disclaimer 문서에서는 "ABS 재질 금지(장기간 내용물과 반응해 용기 크랙)" 명시. **두 ODM이 다른 재질 가이드라인을 운영 중**. LAKA가 ODM별 다른 기준을 인지·관리하는지

3. **본딩 글로우 립스틱 용기 공급사 전환** (대진 → 연우, 2025-04) — 한국콜마 CT 결과서 시계열로 봤을 때 용기 공급사가 명확히 전환됨. 전환 사유(품질·가격·납기·금형 문제 등) 미확인. 신규 용기 공급사(연우) CT 결과 안정도까지 확보됨(V2, 2025-05)

### 본부장 입장

#10 Q1(트윈립)은 후속메일 4번 회신으로 거의 확정될 예정. 회신 받으면 본 아젠다는 Q2·Q3만 남아 분량 줄어듦. 분량에 따라 #11로 분리할 필요는 없을 듯.

### 차승호 팀장 확인 요청 (3문항)

**Q1. 트윈립 정체 확정** (후속메일 4번과 묶음)
- 5/19 미팅에서 짚어주신 "트윈립 제품 금형 이슈" = **코스맥스 듀얼립** (색조틴트 + 글로스 양면 펜, 2026-02-10 CT 진행) 맞는지?
- 듀얼립 공유사항 PDF에 나온 4가지 이슈(내용물 크렉 / 오일뜨기·색소띠 / 역립 보관 누액 / 충전 시 빈공간) + 양면 펜 메커니즘 금형이 통합된 이슈인지, 별개인지?
- 후속메일 4번에서 정리해주실 메일에 위 가설 검증 1줄 포함 부탁

**Q2. 화성 ABS 재질 ↔ 코스맥스 가이드라인 충돌의 LAKA 관리 방식**
- 화성 스테인 립 라이너(2025-11-03 조건부적합 → 고객사 승인)에서 카트리지·바디·플러그·심집 등이 **ABS·POM** 재질로 구성됨
- 그러나 코스맥스 LM1팀 유분산 disclaimer 7페이지 문서에서는 "ABS는 내용물과 반응해 장기간에 걸쳐 용기 크랙 유발 → 사용 금지" + "솔대 POM은 포름알데히드 2000ppm 통과 필수, 해외 POM 금지" 명시
- 이 가이드라인 차이를 LAKA가 인지·관리하고 있는지? ODM별 다른 재질 기준을 통합 관리하는 마스터(재질·내용물·노출 기간 매트릭스)가 있는지?
- 특히 EU/UK/US 수출 시 — 해외 POM 금지 가이드라인이 LAKA SKU에 적용되어 있는지(특히 책판 R&R #21 외 트랙)?

**Q3. 본딩 글로우 립스틱 용기 공급사 전환 사유 (대진 → 연우, 2025-04)**
- 한국콜마 CT 결과서 4건 시계열:
  - 2023-03 V1 (14029, 대진?)
  - 2025-04 V4 (대진 검토 결과)
  - 2025-04 V1 추가홋수 (CTC250310116, 연우)
  - 2025-05 V2 안정도 (연우)
- 대진 → 연우 전환 사유 — 품질 / 가격 / 납기 / 금형 문제 / 회사 사정 중 무엇인가?
- 향후 다른 SKU(예: 프루티 글램 틴트의 일창·장업·코코스팩 3개 용기 공급사)에서도 비슷한 전환이 발생할 수 있는지? 전환 결정 거버넌스가 R&R 어디에 명시되어 있는지
