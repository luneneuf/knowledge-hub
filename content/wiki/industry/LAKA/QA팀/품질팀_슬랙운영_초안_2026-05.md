## 1. 배경 및 목적

- 5/22까지 기존 Teams·카카오톡 단체방 이관 마감, 6월부터 내부 소통은 Slack으로 일원화
- 품질팀은 22개 R&R 업무를 영업·구매·상품개발·디자인·물류·마케팅 등 다수 부서와 협업하므로 채널 구조를 명확히 설계해두지 않으면 정보 분산 및 누락 위험이 큼
- 동시에 SCM 주간보고에서 확인되듯 위해정보·회수·중대 클레임 등 **에스컬레이션이 필요한 이슈가 1~5월 중 4건 이상 발생** (수단레드 잠정 판매중지, 대만 수딩비건립오일 회수, 화성 Audit 개선지시, 퍼펙트트윈립 단상자 혼입) — 채널 운영 자체보다 **신호 전달 룰**이 더 중요

## 3. 운영 규칙

### 가. 전사 임시 운영기준(5/17) 그대로 준수

표시명·연락처·프로필 사진, DM 지양, 스레드 적극 활용, Slack ≠ 전자결재, 외부 커뮤니케이션은 이메일/유선 유지, 파일 제목에 날짜·버전.

### 나. 품질팀 특화 — 정형 보고 템플릿 3종

**① 신제품 QC 결과** (`#qa-신제품-출시검증`)
```
[QC 적합/부적합] 제품명 / 용량·기능성 / 초도품 제조번호 / 제조사 / 입고일자
판정결과: 적합 | 부적합
비고: 특이사항·관리품 배포 여부
```
※ 현 SCM 주간업무보고에서 사용 중인 양식 그대로 채널 메시지화

**② 제품 클레임 접수** (`#qa-클레임-품질이슈`)
영업부서 기안 접수 시 헤드 메시지 + 처리 단계별 진행을 **동일 스레드 내 업데이트**. 클레임 처리 8단계 ([[품질관리_시스템_체계확립_260518_텍스트추출]] Slide 3) 각 단계가 스레드 댓글로 가시화.

**③ 위해정보·회수 시그널 (긴급)** (`#qa-클레임-품질이슈`)
식약처 공문, 외부 안전성 이슈, 해외 통관 거부 등 발생 즉시 헤드 메시지 + 본부장·대표 @멘션.

### 다. 에스컬레이션 룰

| 사안 | 게시 채널 | 즉시 @멘션 |
|------|-----------|------------|
| 신제품 QC 적합 | #qa-신제품-출시검증 | (유관부서 멘션 불필요) |
| 신제품 QC 부적합 | #qa-신제품-출시검증 | 차승호 + BM PM |
| 클레임 일반 | #qa-클레임-품질이슈 | 영업 담당 + 차승호 |
| **클레임 중대 / 회수 가능성** | #qa-클레임-품질이슈 | **천승범 + 이지철** |
| **식약처 위해정보 공문** | #qa-클레임-품질이슈 | **천승범 + 이지철** |
| ODM Audit 부적합 | #qa-제조업체-audit | 천승범 + 구매팀 |
| 개정법규 자사 제품 적용 필요 | #qa-인허가-법규 | 천승범 + 유관부서 |

### 라. 외부(ODM·식약처·KCA) 커뮤니케이션 원칙

- 외부 채널 = 이메일/유선 유지 (전사 운영기준 5번 정합)
- Slack은 **외부 커뮤니케이션 결과 공유 및 내부 조율용**으로만 사용
- 외부에서 받은 공문·이메일·시험성적서는 공유폴더 업로드 + 채널에 링크 게시 (현 R&R 표 운영 방식 유지)

### 마. 공유폴더 vs Slack 역할 분담

| 정보 유형 | 정식 보관 | Slack 활용 |
|----------|----------|-----------|
| 제품표준서·시험성적서·QA 서류 | 공유폴더 (현행) | 신규 업로드 시 채널 알림 |
| 클레임 접수증·처리결과서·개선지시서 | 공유폴더 (현행) | 진행 상황 가시화 |
| 시장출하 리스트·부적합 처리 현황 | 공유폴더 (현행) | **주간 요약 채널 게시** |
| 일상 의견 교환·질의응답 | (Slack 전용) | 스레드 활용 |

→ Slack 도입이 **공유폴더 체계를 대체하지 않음**. 결재·정식 보관은 현 운영 그대로.

## 5. 향후 확장 검토 사항

- **글로벌 컴플라이언스 채널 신설 필요성**: 현 5개 채널은 한국 책임판매업자 의무 중심. [[세포라]]·[[부츠]] Vendor Compliance, MoCRA·UKCR 갱신 사이클이 본격화되면 `#qa-글로벌-컴플라이언스` 별도 운영 필요 — [[품질관리_RR_2026-05]] 갭 분석 참조. SCM 4월 보고서에서 **6월 [[세포라]]·[[부츠]] B2B 발주 진행 중** 확인됨 → 늦어도 6월 내 신설 권장
- **자동 알림 연동**: 식약처 의약품안전나라, [[대한화장품협회]], 환경공단 EPR 공지 → 채널 자동 알림(RSS/스크래퍼) 도입 검토 (개정법규 모니터링 주 1회 이상 작업 효율화)
- **SCM 부문 채널 구조와의 정렬**: 신제품 QC, 부적합, ODM Audit은 SCM 구매팀과 R&R 겹침 → SCM 측 채널 구조 확정 후 중복 채널 통폐합 협의 필요
