> **LAKA 매핑**: 이 페이지는 [[LAKA]]의 화장품 책임판매업자 입장에서 원료목록보고 절차·양식·LAKA 제품 카테고리 매핑을 정리한다.

## 1. LAKA의 보고 의무 위치

[[LAKA]]는 **화장품 책임판매업자**다 (맞춤형화장품판매업자가 아님). 따라서 적용되는 조항·기한·양식이 맞춤형 매뉴얼과 다르다.

| 항목 | **책임판매업자 (LAKA)** | 맞춤형화장품판매업자 (LAKA 해당 X) |
|------|------------------------|-----------------------------------|
| 근거법 | 화장품법 제5조 ④항 | 화장품법 제5조 ⑥항 |
| **보고 기한** | **화장품 유통·판매 전** (제품별 최초 + 변경/단종 시) | 연 1회 (매년 2월말) |
| 보고 유형 | 최초보고 / 변경보고 / 단종보고 | 연 1회 (변경분 포함) |
| 보고 양식 | 별지 제5호, **제6호** | 별지 제7호, 제8호 |
| 제출 기관 | **국내 제조: 대한화장품협회 / 수입: 한국의약품수출입협회** | 대한화장품협회 |
| 보고 사이트 | https://kcia.or.kr/report/main/ | www.kcia.or.kr/report |
| 과태료 | 50만원 | 50만원 |

→ LAKA는 [[한국콜마]]·코스맥스·화성코스메틱·코스메카·씨앤씨 등 국내 ODM 위탁 제조이므로 **국내 제조 화장품** 보고 경로([[대한화장품협회]])를 사용한다. 수입화장품은 해당 없음.

## 2. 보고 사이트 및 ID

- **사이트**: https://kcia.or.kr/report/main/
- **ID**: 생산실적 및 원료목록 보고 전용 ID (협회 회원사 일반 ID와 별개)
- **로그인 정보**: 책임판매업등록번호 + 등록연월일 (책임판매업 등록필증 기재)

## 3. 보고 흐름 (책임판매업자)

```
보고회사 및 담당자 정보 작성
   ↓
원료목록 작성 — 엑셀 업로드 또는 수기 등록
   ↓
원료검사 (표준명·배합금지 자동 확인)
   ↓ 표준명 미등록 / 금지원료 → 수정 필수
제출
   ↓
결과조회 (식약처 제출 서식 인쇄 가능)
```

### 3종 보고 유형

| 유형 | 사용 시점 | LAKA 적용 사례 |
|------|-----------|----------------|
| **최초보고** | 제품 유통·판매 전 | 신제품 출시 시 (예: SCM 1월 2주차 보고의 8건 입고 예정 제품 — 프루티글램틴트 컬러추가 #181~200, 데빌립 #811~812 등) |
| **변경보고** | 전성분 변경 또는 제조업자 변경 시 | 수딩프렙립마스크 토출구 변경(2026.01~03), 프루티글램틴트 #114~130 [[한국콜마]]→코스맥스 이원화(2026.03~) 등 |
| **단종보고** | 보고 제품 단종 시 | 부진재고 C등급 단종 — 팝핑벌룬틴트 본품·미니, 젤링누드글로스 미니, 스킨4코렉터 (2026.03 대표 컨펌, 26년 8월까지 판매추이 확인 후 처리) |

→ SCM 주간보고 흐름과 직접 연결되는 업무. [[2026-03-W1_SCM주간업무보고]] 부진재고 운영방안 등 LAKA 사업 의사결정이 곧 원료목록보고 트리거.

## 4. 보고 양식 (별지 제6호)

### 필수 컬럼 8개 (기능성 제품은 +2개)

| 컬럼 | 내용 | LAKA 작성 예 |
|------|------|--------------|
| 일련번호 | 아라비아 숫자, 순차 | 1, 2, 3... |
| **책임판매업자** | 등록필증 상호 | 주식회사 라카코스메틱스 |
| **원료 성분명** | 표준화된 한글, **한 셀에 한 성분**, 혼합원료는 개별 분리 | 정제수 / 글리세린 / 시트릭애씨드 ... |
| **제품명** | [별지 제2호/제4호 서식] 동일 | 프루티글램틴트 P102, 본딩글로우립스틱 등 |
| **유형표시** | 가1~파3 (아래 LAKA 매핑 참조) | 사6, 사7, 카1 등 |
| 기능성 유형 (해당 시) | F1~F14 | F3 (자외선차단) 등 |
| 기능성 품목기준코드 (해당 시) | 심사품목: 품목기준코드 / 심사제외: 보고일련번호 | — |
| **제조업자** | **화장품 제품 제조업자** (원료 제조업자 아님) | [[한국콜마]], 코스맥스, 화성코스메틱, 코스메카, 씨앤씨 인터내셔널 등 |
| 용도 | 수출전용=E, 국내+수출=빈칸 | E (틱톡샵 UK 전용 등) 또는 빈칸 |

### 혼합물 작성 규칙 (중요)

- **잘못된 예**: `정제수 & 소듐하이알루로네이트 & 라피노오스 & 산자나무추출물`
- **올바른 예**: 각 행 분리 — `정제수` / `소듐하이알루로네이트` / `라피노오스` / `산자나무추출물`

**"/" 사용 예외**: **단일성분으로 간주하는 복수원료**(코폴리머 등)는 슬래시로 연결
- 예: `아크릴레이트/C1-2석시네이트/하이드록시아크릴레이트코폴리머`

### 제출 시 4가지 케이스

1. 수정사항 없음 → 제출 가능
2. 단서조항 배합금지 성분 → 확인 후 제출 가능
3. **순수 배합금지 성분** → 수정 필수, 제출 불가
4. **표준명 미등록 성분** → [[화장품 성분사전]](kcia.or.kr/cid)에서 확인 또는 표준화 신청 필요

## 6. LAKA 운영 시 실무 포인트

### 가. 신제품 출시 시점과 보고 타이밍

[[품질관리_RR_2026-05]] 5번 업무는 **상시(일일)** 주기로 김건 책임이 운영한다. 보고 시점 룰:

1. 신제품 처방 확정 → ODM 발주 → 데이터 아웃 → CT → 입고 → **유통·판매 전 원료목록 최초보고 완료** → 유통 개시
2. SCM 보고서 기준 발주 → 입고는 보통 2~3개월. 이 사이에 보고 처리 필요
3. 변경보고는 전성분 변경 또는 제조업자 변경 발생 시 — 예: 프루티글램틴트 #114~130 [[한국콜마]]→코스맥스 이원화([[2026-01-W4_SCM주간업무보고]])는 **제조업자 변경 → 변경보고 필수**

### 나. ODM별 제조업자 상호 (별지 제6호 기재용)

| ODM | LAKA 비중 | 보고 시 제조업자 상호 (식약처 등록 국문) |
|-----|---------|------------------------------------------|
| [[한국콜마]] | 매월 매입 1위, 5% 지분 보유 | 한국콜마(주) — 정확 명칭은 식약처 등록기준 확인 |
| 코스맥스 | 맥시글레이어틴트 등 | (주)코스맥스 |
| 화성코스메틱 | 스테인립라이너 등 | (주)화성코스메틱 — [[2026-03-W4_SCM주간업무보고]] Audit A등급(96.8점) |
| 코스메카 (코스메카코리아) | 데빌립 등 | (주)코스메카코리아 |
| 씨앤씨 인터내셔널 | 선실드글로이치크 등 | (주)씨앤씨인터내셔널 |

→ 정확한 식약처 등록명은 [원료목록보고 사이트](https://kcia.or.kr/report/main/) 또는 [의약품안전나라](https://nedrug.mfds.go.kr/index)에서 확인 가능.

### 다. 표준명 미등록 원료 대응

[[화장품 성분사전]](kcia.or.kr/cid)에서 검색해도 표준 명칭이 없는 경우:
- **성분명 표준화 신청** → 성분명표준화위원회(월 2회) 심의
- 보완·반려 시 세 달 내 3회 무료 재신청 가능
