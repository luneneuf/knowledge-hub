## 1. 핵심 규칙 한 줄

**향료 안에 들어있는 25종 알레르기 유발성분 중 임계치 초과 함유 시, "향료"로만 표시 불가 → 해당 성분명을 추가 기재 + [[원료목록_사전보고_책임판매업자|원료목록보고]]에도 포함**

| 제품 유형 | 임계치 (그 이상 시 표시 의무) |
|-----------|------------------------------|
| **사용 후 씻어내는** (샴푸·린스·바디워시·클렌징 등) | **0.01% 초과** |
| **사용 후 씻어내지 않는 (leave-on)** (립·아이·페이스·바디 모든 잔류 제품) | **0.001% 초과** |

→ **LAKA 주력 카테고리(사6 립스틱·립라이너, 사7 립글로스·립밤, 라3 아이섀도, 사3 파운데이션 등)는 모두 leave-on 0.001% 엄격 기준 적용**.

### 산출 예시 (지침 ① 원문)
바디로션 250g에 리모넨 0.05g 포함 → 0.05 / 250 × 100 = **0.02%** → 0.001% 초과 → 표시 대상.

→ LAKA 립 제품(예: 프루티글램틴트 4.5g)에서 리모넨 0.5mg(0.0005g) 함유 시 → 0.0005 / 4.5 × 100 = **0.011%** → 표시 대상. 향료 콘셉트 제품은 ppm 단위로 임계치 도달.

## 3. 표시 방법 — 3가지 가능 / 2가지 불가

### ✅ 가능한 표기 (전성분 표시 방법 적용 권장)

기존: `A, B, C, D, 향료` → 리모넨·리날룰 추가 함유 시:

| 안 | 예시 |
|----|------|
| **1안** | A, B, C, D, **향료, 리모넨, 리날룰** |
| **2안** | A, B, C, D, **리모넨, 향료, 리날룰** |
| **3안** | A, B, **리모넨**, C, D, **향료**, **리날룰** (함량 순) |

### ❌ 불가 표기 (소비자 오해·오인 우려)

| 안 | 예시 | 불가 이유 |
|----|------|----------|
| **4안** | A, B, C, D, **향료(리모넨, 리날룰)** | 향료의 하위 성분으로 인식, 별도 성분으로 보지 않음 |
| **5안** | A, B, C, D, 향료, **리모넨***, **리날룰*** (\*알레르기 유발성분) | 25종만 알레르기를 유발하는 것으로 오인 |

→ **LAKA 표시문안 검토 운영 룰** ([[품질관리_RR_2026-05]] #4): 4안·5안 표기로 ODM에서 보내온 라벨 시안은 **즉시 반려 + 재요청**.

## 4. 별도 표시·주의사항 기재 금지

- 25종 성분에 대해 **"알레르기 유발성분임"을 별도 표시 → 부적절** (25종만 알레르기 유발하는 것으로 오인)
- **"사용 시의 주의사항" 란에 기재 X** ('전성분 표시제'의 확대로 보아야 함)

→ **LAKA의 마케팅·라벨 작성 시 주의 포인트**: "이 제품은 알레르기 유발성분(리모넨·리날룰)을 포함합니다" 같은 별도 안내 문구 추가는 오히려 부적절. 단순히 전성분 라인에 함량 순으로 배열.

## 5. LAKA 운영 영향

### 가. 영향 받는 제품 카테고리

| 유형 | LAKA 카테고리 | 임계치 | 영향도 |
|------|--------------|--------|--------|
| 사6 립스틱·립라이너·립틴트 | 프루티글램틴트, 맥시글레이어틴트, 본딩글로우립스틱, 벌키매트립스틱, 데빌립, 팝핑벌룬틴트, 스테인립라이너, 퍼펙트트윈립 | **0.001%** | **상** |
| 사7 립글로스·립밤 | 프루티립글로셔너, 젤링누드글로스, 소울비건립밤, 수딩프렙립마스크, 수딩비건립오일, 글로우립컨디셔너, 스파이시 립 플럼퍼 | **0.001%** | **상** |
| 사1 볼연지 / 사2 파우더 / 사3 파운데이션 / 사4 베이스 | 본딩커버쿠션, 러브실크블러쉬, 선실드글로이치크, 드림빔하이라이터, 베이스 | **0.001%** | **상** |
| 라3 아이섀도 / 라1 아이브로 / 라4 마스카라 | 모노아이섀도우, 포에버6아이팔레트, 픽시브로우카라, 와일드브로우셰이퍼 | **0.001%** | **상** |
| 카1~카10 기초화장 (leave-on) | 라카스킨, 헬시세럼밤, 본딩선크림 | **0.001%** | **상** |
| 다1·카10 클렌징·메이크업 리무버 (rinse-off) | 워시풀틴트리무버, 클렌징라인 | **0.01%** | 중 |
| **비매품** (사용기한 도래 전 유통 시) | 비매품 카테고리 전체 (틴트레디와이프 등) | 동일 적용 | 중 |

→ **LAKA 약 30개 SKU의 95% 이상이 leave-on → 0.001% 엄격 기준 적용**.

### 나. 천연 추출물·비건 라인 — 지침 ⑥ 직접 적용

**식물 추출물·에센셜 오일이 착향 목적으로 사용되었거나, 착향제 특성이 있는 경우 → 알레르기 유발성분 표시 의무**

LAKA의 비건/식물 추출 라인:
- **수딩비건립오일** — [[2026-02-W2_SCM주간업무보고]] Slide 5 수단레드 잠정 판매중지 대상이었던 제품 (#너리싱옐로·#카밍퍼플 등). 식물 추출 색소·향 사용 가능성 높음 → **알레르기 유발성분 함유 여부 ODM 확인 필수**
- **소울비건립밤** — 비건 콘셉트, 식물성 오일 다량 사용
- **헬시세럼밤** — "세럼" 콘셉트, 향 첨가 시 천연 향료 가능성

→ ODM에 **알레르기 유발성분 25종 함량 시험성적서** 별도 요청 필요 ([[원료목록_사전보고_FAQ_책임판매업자]] 증빙자료 항목).

### 다. 글로벌 채널과의 정합성 — UK SI 2024/1334 80종 향료 알레르겐과 별개

[[가상_QA리드_4주플랜_시뮬레이션]]이 짚은 **UK 향료 알레르겐 80종 확대 채택**(SI 2024/1334 후속)과 한국 25종은 **별개 리스트**. 단 25종은 80종의 부분집합 — UK 대응이 한국 대응을 자동으로 커버.

| 시장 | 알레르기 향료 표시 의무 | 기준 |
|------|------------------------|------|
| **한국** | 25종 | 임계치 0.001%/0.01% |
| **EU/UK** | 80종 (SI 2024/1334) | 임계치 0.001%/0.01% (동일) |
| **미국 (FDA)** | 별도 의무 없음 | — |
| **일본** | 별도 의무 없음 (PMD법 기준 적용) | — |

→ LAKA가 [[부츠]]·세포라 UK 입점 시 80종 기준으로 라벨 준비 → 한국 25종도 자동 충족. 역방향(한국 25종만 준비)은 UK 부적합 위험.

### 라. 원료목록보고와의 연결 ([[원료목록_사전보고_책임판매업자]] 직접 영향)

- **표시한 알레르기 유발성분은 [[원료목록_사전보고_책임판매업자|원료목록보고]]에도 포함 필수** (지침 원료목록 ①)
- 시행일 이전 유통 제품의 표시·기재 변경 시 → **변경보고 트리거** ([[원료목록_사전보고_FAQ_책임판매업자]] Q26과 정합 — "원료의 처방 변경"으로 해석)
- 증빙자료 보관: 제조증명서·제품 표준서 + ODM 시험성적서·원료규격서

### 마. 소용량 (10mL 초과 50mL) 화장품 표시 — 면적 부족 시 생략 가능

LAKA의 미니 제품:
- 프루티글램틴트 미니 1.9g × 2ea (3.8g) — **10mL/g 이하 → 표시 면적 부족 사유 생략 가능, 단 홈페이지에 표시 의무**
- 맥시글레이어틴트 미니 2.1g — 동일
- 젤링누드글로스 미니, 팝핑벌룬틴트 미니 — 동일
- 메가와리 기획세트·올리브영 기획세트 등 미니 위주 SKU 다수

→ **온라인 상 (laka.co.kr 등) 표시 의무는 면적 부족과 무관하게 항상 적용**. 책임판매업자 홈페이지 + 올리브영·세포라·부츠 등 입점 채널 모두 점검.

## 6. 실행 체크리스트

### 즉시 확인 (1~2주)

- [ ] **ODM 5사([[한국콜마]]·코스맥스·화성코스메틱·코스메카·씨앤씨) 처방서**에 25종 함량 명시 여부 점검
- [ ] **시트러스·라벤더·로즈·바닐라 콘셉트 SKU** 우선 — 리모넨·리날룰·시트로넬올·제라니올·쿠마린·시트랄 함량 확인
- [ ] **비건/식물 추출 라인** — 수딩비건립오일·소울비건립밤·헬시세럼밤 등 식물 추출물에 함유된 알레르기 유발성분 확인
- [ ] 현재 유통 중 SKU 라벨 표기 점검 — 4안·5안 부적절 표기 사용 여부

### 중기 (1~3개월)

- [ ] 25종 함량 데이터 누적 → 25종 매트릭스 작성 (SKU × 25종 × 함량 × 표시 여부)
- [ ] [[원료목록_사전보고_책임판매업자|원료목록보고]] — 알레르기 유발성분 누락된 보고 점검 + 필요 시 변경보고
- [ ] **온라인 채널** (laka.co.kr, 올리브영, 큐텐 등) 전성분 표시 동기화
- [ ] UK [[부츠]] 입점 SKU는 80종 기준으로 일괄 라벨 재검토

### 장기 (글로벌 정렬)

- [ ] UK SI 2024/1334 80종 대응 ([[가상_QA리드_4주플랜_시뮬레이션]] UK 즉시 리스크 항목과 통합)
- [ ] 마케팅 카피 게이트에 25/80종 표시 룰 포함 ([[품질팀_슬랙운영_초안_2026-05]] `#qa-신제품-출시검증` 채널 활용)
