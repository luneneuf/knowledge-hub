# 미국 MoCRA — Modernization of Cosmetics Regulation Act 2022

> 2022년 12월 29일 서명 발효. **1938년 이후 84년 만의 최초 대규모 화장품 규제 개혁**.  
> 연방 식품·의약·화장품법(FD&C Act) Chapter VI에 화장품 신규 조항 추가.

---

## MoCRA 개요

| 항목 | 내용 |
|------|------|
| 제정 연도 | 2022년 12월 29일 |
| 근거 법령 | 연방 FD&C Act Chapter VI |
| 규제 기관 | FDA (미국 식품의약국) |
| 핵심 방향 | FDA의 화장품 시설·성분·이상사례 감독 권한 강화 |

### 기존 규제의 문제점
- FDA의 화장품 성분·시설 감독 권한 부재
- 의무적 이상반응 보고 체계 없음
- GMP 기준 미확립
- 제품 리콜 강제 권한 없음

---

## 5대 핵심 의무

### 1. 시설 등록 (Facility Registration)

| 항목 | 내용 |
|------|------|
| 대상 | 미국 내 유통을 위해 화장품을 제조·가공하는 국내외 모든 시설 |
| 갱신 주기 | **2년마다** |
| 해외 시설 | 미국 내 에이전트 지정 필수 |
| 면제 | 최종 소매 시설, 소규모 농업 시설, 연구·개발 전용 시설 |
| 기존 시설 의무화 | 2023년 12월 29일 |
| 신규 시설 | 영업 개시 후 **60일 이내** |

### 2. 제품 리스팅 (Product Listing)

| 항목 | 내용 |
|------|------|
| 제출 정보 | 제품명, 브랜드명, 라벨 성분 목록, 용도·범주, 리스팅 번호 |
| 제출 주체 | 책임자(RP) 또는 제조업자 |
| 업데이트 | **연 1회** |
| 기존 제품 의무화 | 2023년 12월 29일 |
| 신제품 | 출시 후 **120일 이내** |

### 3. SAE 보고 (Serious Adverse Event Reporting)

별도 섹션 참조 → 아래 상세 내용.

### 4. cGMP (우수 제조 기준)

| 항목 | 내용 |
|------|------|
| 규정 수립 기한 | FDA가 2026년 12월 29일까지 |
| 참고 기준 | ISO 22716 등 국제 기준과 일관성 |
| 대기업 의무화 | 2025년 12월 29일 |
| 소기업 의무화 | 2027년 12월 29일 |

### 5. 라벨링 요건

| 항목 | 요건 | 시행일 |
|------|------|--------|
| 소비자 이상반응 보고용 연락처 | 국내 주소·전화번호·이메일 중 하나 이상 의무 기재 | 2024년 12월 29일 |
| 향료 알러겐 라벨링 | FDA 고시 목록에 따른 향료 알러겐 개별 표시 | FDA 목록 고시 후 18개월 |

## SAE 보고 상세

### 정의 — Serious Adverse Event (SAE)

화장품 사용과 관련하여 다음 중 하나에 해당하는 건강 문제:

| 유형 | 설명 |
|------|------|
| 사망 | 화장품 사용과 연관된 사망 |
| 생명 위협 | 즉각적인 생명 위협 상태 |
| 입원 | 입원 또는 기존 입원 기간 연장 |
| 영구적 장애 | 신체 구조·기능의 지속적·심각한 손상 |
| 선천성 기형 | 태아 또는 신생아의 이상 |
| 의료 개입 필요 | 화학적 화상, 심각한 알레르기 반응, 심각한 변형 등 |

> 일반 피부 트러블, 경미한 자극은 SAE 해당 없음.

### 보고 주체 — Responsible Person (RP)

> "The manufacturer, packer, or distributor of a cosmetic product **whose name appears on the label**"

- RP 운영 지원은 외주 가능하나, **규제 책임은 RP에게 귀속**
- 한국 브랜드의 경우 미국 내 유통·판매를 담당하는 법인이 RP

### 보고 절차

| 항목 | 내용 |
|------|------|
| 최초 보고 기한 | SAE 수령 후 **15영업일** 이내 |
| 서식 | MedWatch Form 3500A |
| 제출처 | 이메일: CosmeticAERS@fda.hhs.gov 또는 우편 |
| 첨부 | 제품 라벨 사본 |

### 후속 보고

- 초기 보고 후 **1년** 이내 신규 의료 정보 수령 시 → 수령 후 **15영업일** 이내 추가 보고

### 기록 보관

| 주체 | 보관 기간 |
|------|----------|
| 일반 기업 | **6년** |
| 소기업 (소규모 유통업체) | **3년** |

> SAE 보고 의무는 **소기업에도 동일하게 적용** (등록·GMP와 달리 면제 없음).

---

## 소기업(Small Business) 우대사항

| 구분 | 기준 | 혜택 |
|------|------|------|
| Very Small Business | 연간 총 매출 **\$1,000,000 미만** | 시설 등록·cGMP 일부 면제 |
| Small Business | — | 기록 보존 3년 (일반 6년), 이행 기간 연장 |

> 단, 이상사례 보고·라벨링·안전성 입증은 소기업도 면제 없음.

## FDA Prior Notice 제도

화장품은 식품처럼 별도 Prior Notice 의무 제출 항목은 아니나, 통관 과정에서 FDA PREDICT 시스템을 통해 자동 스크리닝된다.

```
수출자(한국) → CBP 전자 신고 (Customs Entry)
      ↓
FDA PREDICT 시스템 자동 스크리닝
      ↓
├─ [통과] → 통관 허가
└─ [플래그] → 물리적 검사 또는 서류 심사 → 억류(Detention) 가능
```

### 통관 억류 주요 원인

| 원인 | 설명 |
|------|------|
| 라벨 부적합 | 영문 미표기, 성분 미표기 |
| 의약품 클레임 | "콜라겐 생성 촉진", "주름 감소" 등 의약 효능 표현 |
| 금지 색소 | FDA 미승인 색소 사용 |
| 시설 미등록 | 제조사 FDA 등록 누락 |
| 금지 성분 | 수은 화합물, 클로로폼 등 |

## 한국 브랜드 체크리스트

- [ ] 한국 ODM 제조사(코스맥스·한국콜마) FDA 시설 등록 확인
- [ ] 제품 FDA Cosmetics Direct 등록 완료
- [ ] 영문 라벨 FDA 기준 충족 검토 (RP 연락처, INCI명, fl oz/ml 이중 표기)
- [ ] 성분 중 미국 금지 성분 여부 확인
- [ ] 선스크린 성분 포함 여부 → 별도 OTC 의약품 절차 확인
- [ ] 이상사례 보고(SAE) 시스템 구축 (15영업일 이내 보고)
- [ ] 기록 보관 체계: 6년 (소기업 3년)
