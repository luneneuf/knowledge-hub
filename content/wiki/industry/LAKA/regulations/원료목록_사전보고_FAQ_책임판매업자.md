## 1. LAKA에 가장 중요한 FAQ 7개

### ⭐ Q16. 동일 제품을 여러 제조업자에게 위탁 — 제조업자별로 각각 보고

**FAQ 원문**: "동일한 제품을 여러 제조업자에게 위탁하여 제조한 경우 제조업자별로 각각 입력하여 보고해야 합니다."

**LAKA 직접 적용 사례**:
- [[2026-01-W4_SCM주간업무보고]] Slide 2: 프루티글램틴트 **#114~130 [[한국콜마]] ↔ 코스맥스 이원화** (총 6.3억 절감 기대)
- 결과: **동일 SKU 17개를 콜마용·코스맥스용으로 각각 변경보고 필요** → 17 × 2 = 34건 보고
- 단순 원가절감 효과 외에 [[품질관리_RR_2026-05]] 김건 책임의 보고 작업량 증가

**참고**: "포장 또는 표시만의 공정을 한 제조업자는 보고 대상 아님" — 인쇄·라벨링·박싱만 위탁한 업체는 보고 불필요.

---

### ⭐ Q26. 변경보고 트리거 — 제조업자 변경 OR 처방(formula) 변경

**FAQ 원문**: "제품의 제조업자나 원료의 처방(formula)이 변경된 경우에 한해 변경보고해야 하며, 해당 제품을 유통·판매하기 전에 보고해야 합니다."

**LAKA 적용 — 1~5월 SCM 보고에서 발견되는 변경보고 트리거**:

| 케이스 | 트리거 유형 | 출처 |
|--------|-------------|------|
| 프루티글램틴트 #114~130 [[한국콜마]]→코스맥스 이원화 | **제조업자 변경** | [[2026-01-W4_SCM주간업무보고]] |
| 수딩프렙립마스크 토출구 변경 용기(2/중순~3/초) | 처방 X 패키지 — **변경보고 불요** (Q26 범위 밖) | [[2026-01-W2_SCM주간업무보고]] |
| 수딩프렙립마스크 원가절감 — 한국콜마 동일, 용기·단상자 업체 변경 | 처방 X 자재 업체 변경 — **변경보고 불요** | [[2026-01-W4_SCM주간업무보고]] |
| 단상자 업체 세종기획 → 에이스팩 | 자재 업체 변경 — **변경보고 불요** | [[2026-01-W4_SCM주간업무보고]] |
| 퍼펙트트윈립 솔대 캡 형상 개선 | 부자재 금형 변경 — **변경보고 불요** | [[2026-04-W5_SCM주간업무보고]] |
| 신제품 컬러 추가 (#181~200, #811~812 등) | 신 SKU — **최초보고** 트리거 | [[2026-01-W2_SCM주간업무보고]] |

→ **핵심 룰**: 책임판매업자·원료 성분명·제품명·유형표시·**제조업자**·용도 변경만 변경보고 대상. **패키지·자재·금형 변경은 변경보고 불요.**

---

### ⭐ Q27. 함량만 변경 — 변경보고 불요

**FAQ 원문**: "원료는 동일한데 함량만 달라지는 경우는 변경보고를 하지 않아도 됩니다."

→ 원료목록보고는 **원료 종류**만 추적. 함량은 별도 비공개 정보로 취급. LAKA의 처방 미세 조정 시 보고 부담 없음.

---

### Q9. 사전보고 = 최초 1회 + 변경 시만, 매번 생산 시 X

**FAQ 원문**: "최초로 제조 또는 수입되는 품목에 대해 보고('최초 보고')하고, 추후 변경사항 발생 시 유통·판매 전에 보고. 변경사항이 없는 경우 다시 보고하지 않아도 됩니다."

→ LAKA 같은 회사 (SCM 주간보고 기준 신제품 월 5~10건, 재발주 다수)에 적용:
- **재발주는 변경보고 불요** (예: 프루티글램틴트 본품 재발주 월 9~10억 입고 — 변경보고 X)
- **신 SKU·신 컬러 추가는 최초보고 필수** (예: 프루티글램틴트 #181~200, 데빌립 #811~812)

---

### Q12. 샘플(견본품)도 최초보고 필수

**FAQ 원문**: "샘플용(견본품)인 경우에도 3월 14일 이후 최초로 제조하여 유통판매하는 제품의 경우에는 반드시 유통판매전에 원료목록보고를 해야 합니다."

→ LAKA 비매품·증정품 (1월 5일 재고표상 "비매품(키링·파우치·쇼핑백·툴 등)" 카테고리 별도 / 맥시글레이어틴트 미니 증정품 1.9억 등 / 틴트레디와이프 비매품)은 **유통되는 시점**에 보고 필요.

단, **사내 R&D용 시험품**은 유통 X → 보고 불요. **외부 PR·인플루언서 발송 견본은 유통 O → 보고 필요**.

---

### Q23. 순수 배합금지 원료 — 행정처분 단계

| 차수 | 처분 |
|------|------|
| 1차 | 제조 또는 판매업무 정지 **3개월** |
| 2차 | 제조 또는 판매업무 정지 **6개월** |
| 3차 | 제조 또는 판매업무 정지 **12개월** |
| 4차 | **등록 취소** |

→ [[가상_QA리드_4주플랜_시뮬레이션]]이 짚은 미국 색소 batch certification 갭은 미국 통관 문제. **한국 책임판매업 자체 박탈은 별표1 사용 불가 원료 사용**이 핵심 리스크. **3개월 판매 정지만 발생해도 LAKA 연 매출 100억 이상 충격.**

LAKA의 경우 **[[2026-02-W2_SCM주간업무보고]] 수단레드 잠정 판매중지 사건**(2/3 공문 접수, 2/10 불검출 해제)이 가장 유사한 시그널. 결과적으로 별표1 등재 가능성이 있는 색소가 포함되어 있었다면 본격 행정처분 가능성 있었음.

---

### Q22 vs Q23. 단서조항 배합금지 vs 순수 배합금지

| 안내 문구 | 의미 | 조치 |
|----------|------|------|
| "단서조항 확인이 필요한 배합금지 원료입니다" | 조건부 금지 (예: 디알칸올아마이드류는 2급 아민 함량 5% 초과 시만 금지) | **별도 수정 불필요**, 참고 안내 |
| "순수 배합금지 원료입니다" | 단서조항 없는 절대 금지 | **사용 불가**, 처방 변경 필수 |

→ ODM 처방 검토 시 두 문구를 명확히 구분해야 함. 단서조항 메시지를 "그냥 무시"하면 안 되지만, "사용 불가"로 오인해서 제품 출시를 중단할 필요도 없음.

## 2. 절차·시스템 관련 FAQ

### Q1, Q2, Q3 — 사이트와 ID

| 항목 | 내용 |
|------|------|
| 보고 사이트 | https://kcia.or.kr/report/main/ |
| 보고 방식 | **온라인 전용** (오프라인·메일 보고 X) |
| ID | **원료목록보고 전용 ID** (협회 회원사 ID와 별개) |
| ID 분실 | 로그인 화면 > 아이디/비밀번호 찾기 > 담당자 이름·이메일 |

→ [[2026-03-W1_SCM주간업무보고]] Slide 9 "2025년 화장품 생산실적 보고 완료"는 같은 사이트에서 진행한 별도 절차. ID 공유.

### Q4. 신규 책임판매업 등록 회사 — 회원가입 절차
"이용신청" 클릭 → 회원가입 → 보고 시작.

### Q10. 수입 화장품만 유통·판매하는 자 — 보고 면제
한국의약품수출입협회의 **표준통관예정보고**가 원료목록보고를 대체. LAKA는 해당 없음 (국내 ODM 위탁 제조).

### Q11. 위탁제조 시 제조업자가 보고? — 아니요
**책임판매업자만 보고 대상.** ODM([[한국콜마]]·코스맥스 등)이 직접 보고하지 않음. LAKA 김건 책임이 처리.

### Q17, Q18. 표준명 확인 경로

1. 협회 보고 사이트 (kcia.or.kr/report/main/) > 공지사항 > "원료목록보고 표준화 명칭 목록 안내"
2. 협회 성분사전 (kcia.or.kr/cid/) > "표준화 명칭 목록 다운로드" 또는 "화장품 성분명 검색"
3. 미검색 시 **성분명표준화위원회 신청** (월 2회 개최)

### Q19. 구명칭·신명칭 변환

**케이스 1 — 일대일 매칭**: 시스템 자동 변환 (예: 가는잎그늘사초뿌리추출물 → 산거울뿌리추출물)

**케이스 2 — 통칭이 세분화**: 변경할 신명칭 선택 필수
| 구명칭 | 신명칭 |
|--------|--------|
| 감초추출물 | 스페인감초뿌리추출물 / 감초뿌리추출물 / 창과감초뿌리추출물 |
| 황련추출물 | 콥티스뿌리추출물 / 황련뿌리줄기추출물 / 황련뿌리추출물 |

→ **단상자 등 포장재 구명칭 표기**: 별도 유예기간 없음. 기존 포장재 재고 소진 후 변경된 명칭 기재 권장.

### Q20. 수출용·국내용 동시 판매 시 용도 기재
**용도 공란** (한 번만 보고). **수출 전용은 "E"**.

→ LAKA 적용: [[2026-03-W4_SCM주간업무보고]]의 (KR)·(JP)·(KR/US) 표기 SKU 중 **국내+해외 동시 판매는 공란**, **수출 전용은 E**. 단 일본·미국 전용 SKU(예: 스테인립라이너 일본용 8SKU)는 E.

### Q15. 기능성화장품 품목기준코드 확인

| 구분 | 확인 경로 |
|------|-----------|
| 심사 제품 | 식약처 전자민원창구(https://nedrug.mfds.go.kr/pbp/CCBDB01) > 기능성화장품제품정보(심사) > 업체명 검색 > "품목일련번호" OR 심사 결과 통지서 우하단 "품목기준코드" |
| 보고 제품 (심사 제외) | 식약처 전자민원창구(https://nedrug.mfds.go.kr/pbp/CCBDC01) > 기능성화장품제품정보(보고) > 업체명 검색 > "보고일련번호" OR 심사 제외 품목 보고서 하단 (구) "번호" |

→ [[품질관리_RR_2026-05]] 16번 업무(기능성화장품 심사 제외품목 보고, 김건 책임)와 직결. LAKA 자외선차단 기능성 제품(선실드글로이치크·본딩선크림 등) 보고 시 필요.

## 3. LAKA 운영 체크리스트 (FAQ 종합)

### 매월 정기 점검 (김건 책임)

- [ ] SCM 주간보고에서 **신 SKU 입고** 발생 → 최초보고 처리 여부 확인
- [ ] SCM 주간보고에서 **제조업자 변경** 발생 → 변경보고 처리 여부 확인 (Q16 단일 SKU 다중 제조업자 케이스 점검 포함)
- [ ] SCM 주간보고에서 **처방 변경** 발생 → 변경보고 처리 여부 확인 (함량 변경은 제외)
- [ ] 단종 결정 SKU → 단종보고 (예: C등급 부진재고 단종 ─ [[2026-03-W1_SCM주간업무보고]])

### 신제품 출시 사이클 게이트

1. 처방 확정 시 → ODM에 표준 성분명 사용 가능 여부 확인 요청
2. 데이터 아웃 단계 → **표준명 미등록 원료 사전 점검** (성분사전 검색)
3. 미등록 발견 시 → 성분명표준화위원회 신청 (월 2회) — **출시 일정에 영향**
4. CT 통과 → 입고 → **유통·판매 전 최초보고 완료**
5. 유통 시작

→ [[2026-03-W1_SCM주간업무보고]] 코아시스(현대백화점) 입점 추진 시 **올리브영 상위 매출 제외 신규 SKU**의 유통 채널 확대 = 새 채널은 별도 보고 불요(이미 최초보고 완료된 제품)

### 변경보고 발생 시 작업 흐름

```
SCM 보고서에서 변경 트리거 인지
   ↓
변경 유형 분류 (제조업자 변경 / 처방 변경 / 둘 다 / 함량만)
   ↓ 함량만 → 보고 불요
   ↓ 그 외 → 진행
유통·판매 시작 전 변경보고 완료
   ↓ 단일 SKU·단일 ODM → 1건
   ↓ 단일 SKU·복수 ODM (예: 콜마+코스맥스) → ODM별 각 1건
결과조회로 확인
```
