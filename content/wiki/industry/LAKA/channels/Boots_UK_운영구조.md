## 공급자 규제 준수 요건

영국 화장품 규제 전면 준수 필수.

### CPSR (Cosmetic Product Safety Report)
- 자격 있는 안전 평가자(독성학·약학·의학·피부과학 대학원 이상) 서명
- Part A: 안전성 정보 (성분, 안정성, 사용 패턴 등)
- Part B: 안전성 평가 (안전 결론, 평가자 서명)

### PIF (Product Information File)
- **10년간 보관 의무** (최종 배치 출시일부터)
- 포함: CPSR, 제조 방법, GMP 준수 선언, 클레임 근거, SCPN 신고 확인서

### SCPN (Submit Cosmetic Product Notification)
- 모든 화장품 UK 시장 출시 전 신고 의무
- RP가 신고 주체 (영국 내 법인만 가능)

### 기타 테스트 요건

| 테스트 | 기준 |
|--------|------|
| 안정성 시험 | ISO 표준 기반, 유통기한 및 PAO 근거 |
| 방부제 효능 시험 | 수분 함유 제품 의무 (ISO 11930) |
| 피부 자극성 | 민감성 소비자 보호 목적 |
| GMP 준수 | ISO 22716 권장 |
| 동물 실험 | 완제품·성분 동물 실험 금지 (UK 화장품법) |

## Make-up Exchange Promise (개봉 메이크업 교환 정책)

Boots는 영국 주요 뷰티 리테일러 중 **최초로 개봉한 메이크업 제품도 교환 가능**한 정책을 도입.

| 항목 | 내용 |
|------|------|
| 정책명 | Make-up Exchange Promise |
| 적용 범위 | 개봉 메이크업 제품 |
| 기간 | **35일 이내** |
| 교환 | 가능 (결함 없어도) |
| 환불 | **불가** (교환만 가능) |
| 온라인 구매 | **해당 없음** (오프라인 매장만) |

---

## CS 정책 (35일 반품, Advantage Card 연계)

| 항목 | 내용 |
|------|------|
| 반품 기간 | **35일 이내** |
| 미개봉 제품 | 영수증 지참 시 환불 또는 교환 |
| 개봉 화장품 | 원칙적으로 반품 불가 (결함 있는 경우 예외) |
| 개봉 메이크업 | 35일 이내 교환 가능 (Make-up Exchange Promise) |
| 환불 처리 기간 | **14 업무일** 이내 |
| Advantage Card | 포인트 적립, 반품 시 구매 이력 확인 가능 |

### 브랜드사 관점 시사점
- Make-up Exchange Promise로 소비자 리스크 낮아져 구매 촉진 → 교환 대비 재고 계획 필요
- 교환 제품이 다른 브랜드로 이동할 가능성 (교환 대상 제한 없음)
- 반품된 화장품 재판매 불가 → 폐기 처리 비용 브랜드 부담 가능성
