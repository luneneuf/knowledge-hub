## 역사

- **1849**: 영국 노팅엄에서 존 부츠(John Boot)가 약초·약품 판매 약국 창업.
- **19세기 말~20세기 초**: 아들 제시 부츠(Jesse Boot)가 사업을 이어받아 저렴한 약품 대량 공급 체인으로 확장. "영국 국민 드럭스토어" 이미지 구축.
- **20세기**: 영국 전역 체인화. 자체 브랜드(No7, Soap & Glory 등) + 처방약 조제 결합 복합 모델.
- **2000년대**: 유럽 내 도매·리테일 합병으로 Alliance Boots 형성.
- **2014**: 미국 Walgreens와 통합 → Walgreens Boots Alliance(WBA) 편입.
- **2025**: Sycamore Partners(미국 사모펀드) + Pessina 일가에 인수. WBA에서 분리, 독립 비상장 프라이빗 컴퍼니로 재편. "영국 기반 리테일에 집중하는 전략적 리셋."

## 지배구조 (2025~)

```
Sycamore Partners (미국 사모펀드, 지배지분)
+ Stefano Pessina 일가 (전량 재투자, 공동 지배)
    → Alliance Boots Holdings Ltd (지주사, 75%+ 의결권)
        → The Boots Company PLC
```

Sycamore와 Pessina 일가가 [[세포라]]·[[울타뷰티]] 지분을 전략적으로 보유했다는 공시·보도 없음 — 세 채널은 교차 지분 없는 독립 유통사.

## 채널 특성

- **드럭스토어+약국**: 처방전 조제·헬스케어와 뷰티가 결합된 일상형 채널. [[세포라]]·[[울타뷰티]]와 가장 다른 포지션.
- **헬스&뷰티 허브**: 스킨·바디·헤어·헬스케어 브랜드에 특히 중요. 성분 신뢰성·약사 추천 채널로서의 권위.
- **영국·유럽 커버리지**: K-뷰티 브랜드의 "유럽 첫 오프라인 거점"으로 활용.
- **Boots Advantage Card**: 영국 최대 로열티 프로그램 중 하나.

## 공급자 규제 준수 요건

영국 화장품 규제 전면 준수 필수.

### CPSR (Cosmetic Product Safety Report)
- 자격 있는 안전 평가자(독성학·약학·의학·피부과학 대학원 이상) 서명
- Part A: 안전성 정보 (성분, 안정성, 사용 패턴 등)
- Part B: 안전성 평가 (안전 결론, 평가자 서명)

### PIF (Product Information File)
- **10년간 보관 의무** (최종 배치 출시일부터)
- 포함: CPSR, 제조 방법, GMP 준수 선언, 클레임 근거, SCPN 신고 확인서

### SCPN (Submit Cosmetic Product Notification)
- 모든 화장품 UK 시장 출시 전 신고 의무
- RP가 신고 주체 (영국 내 법인만 가능)

### 기타 테스트 요건

| 테스트 | 기준 |
|--------|------|
| 안정성 시험 | ISO 표준 기반, 유통기한 및 PAO 근거 |
| 방부제 효능 시험 | 수분 함유 제품 의무 (ISO 11930) |
| 피부 자극성 | 민감성 소비자 보호 목적 |
| GMP 준수 | ISO 22716 권장 |
| 동물 실험 | 완제품·성분 동물 실험 금지 (UK 화장품법) |

## Make-up Exchange Promise (개봉 메이크업 교환 정책)

Boots는 영국 주요 뷰티 리테일러 중 **최초로 개봉한 메이크업 제품도 교환 가능**한 정책을 도입.

| 항목 | 내용 |
|------|------|
| 정책명 | Make-up Exchange Promise |
| 적용 범위 | 개봉 메이크업 제품 |
| 기간 | **35일 이내** |
| 교환 | 가능 (결함 없어도) |
| 환불 | **불가** (교환만 가능) |
| 온라인 구매 | **해당 없음** (오프라인 매장만) |

---

## CS 정책 (35일 반품, Advantage Card 연계)

| 항목 | 내용 |
|------|------|
| 반품 기간 | **35일 이내** |
| 미개봉 제품 | 영수증 지참 시 환불 또는 교환 |
| 개봉 화장품 | 원칙적으로 반품 불가 (결함 있는 경우 예외) |
| 개봉 메이크업 | 35일 이내 교환 가능 (Make-up Exchange Promise) |
| 환불 처리 기간 | **14 업무일** 이내 |
| Advantage Card | 포인트 적립, 반품 시 구매 이력 확인 가능 |

### 브랜드사 관점 시사점
- Make-up Exchange Promise로 소비자 리스크 낮아져 구매 촉진 → 교환 대비 재고 계획 필요
- 교환 제품이 다른 브랜드로 이동할 가능성 (교환 대상 제한 없음)
- 반품된 화장품 재판매 불가 → 폐기 처리 비용 브랜드 부담 가능성
