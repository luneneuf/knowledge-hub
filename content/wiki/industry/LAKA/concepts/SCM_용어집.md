# SCM 용어집 — LAKA 사내 약칭·일본어 차용어

LAKA SCM 주간업무보고·물류 가이드에서 통용되는 약칭, 일본어 차용어, 사내 분류 용어를 한 곳에 모음. 외부 입사자·신규 협업자가 즉시 해석할 수 있도록 정의 + 출처 + 컨텍스트 포함.

---

## 1. 거래처·거점 약칭

### HDL
- 정식: **(주)에이치디엘코리아 (HDL Korea Inc.)** — [[HDL_에이치디엘코리아|상세 entity]]
- 역할: LAKA **김포물류센터** 운영 3PL + 국제물류 포워더
- 일평균 보관: 2,500PT (2026-04 기준)
- LAKA SCM 사슬의 **국내 물류 허브**

### JT (株式会社JT)
- LAKA 일본 법인 — [[株式会社JT|상세 entity]]
- PMDA 대응·일본 안전재고 보관 위탁
- "**JT 안전재고**" = JT가 일본 시장 대응으로 보관하는 LAKA 재고

### 어크로스비
- 한국 크로스보더 3PL — [[어크로스비|상세 entity]]
- 공식 거점: 미국·일본·중국 (UK·DE는 거점 외 라우팅)

### ODM 3사
- [[한국콜마]] · [[코스맥스]] · **코스메카** (코스메카코리아) — 생산 위탁사

## 2. 단위·캐파 용어

### PT (파렛트, Pallet)
- 보관 단위. 1PT = 1 파렛트 화물
- LAKA HDL 보관 규모: 2,500PT (일평균, 2026-04 기준)
- 절감 효과 측정에도 사용: 250PT 감소 = 월 400만원 절감

### 착인캐파
- **HDL 1일 착인 작업 처리 한계 = 7~8천개**
- 착인 = 단상자/제품 표기 정보(Lot·바코드·라벨) 검수·부착 작업 (#착인 항목 참조)
- 월말 출고 폭주 + 품목당 50개 이하 다품종 출고 시 효율 급락 (셋업 오버헤드)

### MOQ (Minimum Order Quantity)
- ODM 최소 발주 수량
- C등급(영업 니즈 낮고 1년 MOQ 커버 불가) 단종 정책의 기준선
- 코스맥스·한국콜마 MOQ는 3,000개까지 인하 ([[ODM_발주_프로세스]])

## 3. 검수·표기 용어

### 착인 (着印, ちゃくいん)
- **두 가지 의미** — 컨텍스트 주의:
  1. **명사**: 제품/단상자에 인쇄된 표기 정보 (Lot 번호, 바코드, 라벨). 일본어 着印 차용
     - 출처: [2026-03-W4](../../../raw/cosmetics/scm_weekly/2026-03-W4_SCM주간업무보고.md:145) "단상자 및 제품 **착인 정보 확인**" — 입고검사 절차 #7
     - 출처: [2026-04-W3](../../../raw/cosmetics/scm_weekly/2026-04-W3_SCM주간업무보고.md:187) "제품 바코드 / **Lot 착인 확인**" — 비전 검사기
  2. **동사형 (착인 작업)**: 단상자·제품에 라벨 부착·바코드 등록·표기 검수하는 입출고 라벨링 작업
     - 출처: 본부장 가이드 "HDL 1일 **착인캐파** 7~8천개"
- 일본어 着荷(ちゃっか, 도착 입고) 차용 가능성도 있으나 LAKA에서는 **着印**(인쇄 표기) 의미가 우선

### 단상자
- 화장품 외장 종이 박스 (제품 1개를 담는 단위 패키지)
- "단상자 혼입 = 다른 제품의 외장 박스가 섞여 들어간 불량" (예: [[퍼펙트-트윈-립]] #902호 제품에 #905호 단상자가 들어간 클레임)
- "단상자 하짝 홋수명 인쇄" = 단상자 바닥에 색상 번호(홋수명) 표기 — 혼입 시 현장 직관 식별용

### 홋수명 / 색번
- 색상 식별 번호 (예: #902, #903, #905)
- 단상자 묶음 단위 홋수명 육안 검사 = QA 표준 절차

### 비전 검사기
- 라인별 카메라 자동 검수 장비 (바코드·Lot 표기 자동 확인)
- 단상자 글자 포인트가 작거나 빛 반사 시 부적합 인식 수량 다수 발생 — 알려진 한계
- 재작업품은 반드시 비전 검사기 입구로만 투입 (우회 금지)

## 4. 재고 라벨 분류 (HDL 보관 기준)

LAKA는 HDL 창고에서 재고를 **라벨 언어 기준 4분류**로 관리. 2026-04-W5부터 카테고리 재정의됨.

| 카테고리 | 정의 (2026-04부터) | 2026-04-W5 잔액 |
|---|---|---:|
| **국문** | 국내 + 해외 + 큐텐 통합 물량 | 72.7억 |
| **일문** | JT 안전재고 + 큐텐 일본향 | 35.1억 |
| **영문** | 미주 향 | 10.3억 |
| **국/영문** | 전 채널 통합 라벨 (신규) | 5.3억 |

- 카테고리 정의 변경 이력 (2026-04-W5):
  - 일문: "JT 안전재고" → "JT + 큐텐"
  - 신규 추가: "국/영문" = 전 채널 통합 라벨
- "국/영문" 카테고리 증가(+2.51억) = **다국가 통합 라벨링 전략** 진행 시그널
- 출처: [2026-04-W5](../../../raw/cosmetics/scm_weekly/2026-04-W5_SCM주간업무보고.md:106)

## 5. SKU 등급 (단종 정책)

[2026-03-W1 §6](../../../raw/cosmetics/scm_weekly/2026-03-W1_SCM주간업무보고.md) 기준:

| 등급 | 정의 | 처리 |
|---|---|---|
| **S·A** | 정상 판매 | 일반 운영 |
| **신제품** | S/A 외 신제품 | MOQ 발주 / 누적 재고금액 조정 |
| **B** | 월평균 500개 미만 판매 | 월평균 × 3개월 수준 유지, 일시적 품절 허용 |
| **C** | 영업 니즈 낮고 1년 기준 MOQ 커버 불가 | **순차적 단종 처리 (대표님 컨펌 완료)** |

- C등급 단종 후보 사례: 팝핑벌룬틴트 본품·미니, 젤링누드글로스 미니, 스킨4코렉터 전품목
- C등급 마지막 시장 테스트: [[sociolla 소시올라|Sociolla]](인니) + POYA(대만) — 26년 8월까지 셀-아웃 추이 모니터링

## 6. 일본 시장 행사 용어

### 메가포
- **Mega Power 약칭** — Qoo10 Japan 정기 대형 프로모션 (대규모 할인·쿠폰)
- LAKA 매출 영향 큼: "2월 초 메가포 행사 3.6억 예상" ([2026-01-W4](../../../raw/cosmetics/scm_weekly/2026-01-W4_SCM주간업무보고.md))

### 메가와리
- **Mega 割(わり, 와리=할인) 약칭** — Qoo10 Japan 대형 프로모션 (월 2~3회)
- LAKA 매출 영향 매우 큼: "2월 말 메가와리 행사 7.2억 예상"
- 트랙스(아래) 행사 처리량: 80,131건, 건당 30엔 (2026-03 기준)

### 트랙스 (TRAX·Triax 추정)
- Qoo10/일본 EC 통합 운송·풀필먼트 사업자
- 메가와리·메가포 같은 대형 행사 시 LAKA 일본 물량 처리 위탁
- 마일리지 정산 구조 (예: 2,403,930엔 처리 → 마일리지 2,200만원 → 4월 비용 차감)

### 큐텐 (Qoo10)
- [[Qoo10_일본|상세 entity]] — 일본 K-뷰티 EC 1위, LAKA 립 11분기 연속 1위 채널

## 7. 채널·거래 용어

### 올리브영 B2B
- [[올리브영]] 매장 공급 = B2B 채널
- "올리브영 & B2B → 데빌립 국문 재고 3.6억" (2026-02-W2) — 통합 운영
- 어워즈 데이터 = 셀-아웃 부분 가시성 ([[Sell-in_Sell-out]])

### Sell-in / Sell-out
- 별도 페이지: [[Sell-in_Sell-out]]
- 셀-인 = LAKA → 유통사 출하 매출 (B2B 매출표 = 셀-인)
- 셀-아웃 = 유통사 → 최종 소비자 판매 (대부분 블랙박스)

### 안전재고
- 일정 수요 변동·리드타임 변동에 대비해 보유하는 추가 재고
- LAKA 사례: "JT 안전재고" (일본 시장 대응), "데빌립 11·12월 JT 발주 및 안전재고 발주분 지속 입고" ([2026-02-W2](../../../raw/cosmetics/scm_weekly/2026-02-W2_SCM주간업무보고.md))

### 페리 / 항공
- 페리: 카페리 운송 (한↔일 주력)
- 항공: 항공 수송 (긴급 출고)
- 2026-04부터 **페리/항공 출고 시 대표 사전승인 시행** — 운송비 통제

### 큐텐 / 아마존
- 큐텐 = [[Qoo10_일본]] (일본 K-뷰티 EC)
- 아마존 = Amazon Japan·US
- "2월 일본 큐텐/아마존 7.5억 예상" 식으로 묶어 표기

### 인박스
- 단상자를 묶어 담는 중간 단위 박스 (보통 묶음 단위 = 수십~수백개)
- "인박스 투입 전 외관 검사 필수" — 단상자 혼입 클레임 후 도입된 표준
