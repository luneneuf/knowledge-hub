# 공급업체 리스크 스코어링

AI가 실시간 외부 데이터를 학습해 공급업체별 리스크를 정량 점수로 산출하는 방법론. 재무 데이터·분기 자체평가 등 후행 지표 의존에서 벗어나 **실시간 사건·이벤트 기반 선행 리스크 감지**로 전환하는 것이 핵심.

## 기존 방식의 한계

| 기존 방식 | 문제 |
|---------|------|
| 재무 데이터 기반 평가 | 분기·연간 후행 지표만 반영 |
| 분기 자체평가 설문 | 대상 공급업체가 직접 작성 → 편향 가능 |
| 실사(Due Diligence) | 건당 수일~수주 소요, 확장성 없음 |

## AI 기반 스코어링 아키텍처 (Prewave 사례)

**데이터 레이어**: 50+ 언어, 일 450만 건 뉴스·SNS·정부공시·NGO 보고서·기상·지진 데이터 스캔

**탐지 레이어**: 120종 이상 이벤트 (화재, 파업, 인솔벤시, 노동권 위반, 공급 중단 등)

**스코어 레이어**: 4개 차원 비선형 가중 합산
- Peer Score (국가·산업·원자재 리스크 기준)
- Alert Score (실시간 이벤트, 시간 감쇄 가중)
- Assessment Score (규제 자체평가 자동 채점)
- External Data Score (재무 데이터 API)

**출력**: 60분 내 Critical 알림, Action Priority 5단계(No/Low/Mid/High/Critical), 24개월 추이

## 인간 개입 지점

AI는 리스크 점수와 권장 액션을 **제안**한다. 최종 결정은 사람:
- CPO 직속 중앙 리스크 팀: 가중치·임계값 정의
- 구매 담당자: 발주·계약 직전 점수 확인, 최종 공급업체 선정
- Critical 알림 발생 시: incident review 발의, on-site 감사, 안전재고 확보 여부 결정

## 적용 대상 (K-뷰티 수입 컨텍스트)

| 공급업체 유형 | 모니터링 포인트 |
|-------------|--------------|
| ODM 파트너 (코스맥스·한국콜마) | 공장 화재·QC 이슈·FDA 조사 |
| 원자재 공급사 | 원산지 지역 지정학 리스크, 자연재해 |
| 해외 물류 파트너 | 파업·항만 혼잡·선사 재무 상태 |
| 일본 유통 파트너 | 거래처 재무 건전성, 시장 규제 변화 |
