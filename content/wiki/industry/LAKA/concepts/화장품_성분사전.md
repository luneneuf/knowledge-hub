# 화장품 성분사전 (kcia.or.kr/cid)

[[대한화장품협회]] 운영, **한국 화장품 표준 성분명 데이터베이스**. 책임판매업자가 원료목록보고를 위해 입력하는 성분명은 본 사전에 등록된 표준명을 기준으로 함. URL: https://kcia.or.kr/cid

> ⚠️ **stub 페이지** (2026-05-18 신설). 사전 등록 성분 총수·검색 인터페이스·표준화 신청 양식 직접 확인 미완.

---

## 사용 시점 — [[품질관리_RR_2026-05]] #5 원료목록보고 직결

1. ODM([[한국콜마]]·코스맥스 등)로부터 전성분표 수령
2. 각 성분명을 [[원료목록_사전보고_책임판매업자|KCA 보고 시스템]]에 입력
3. 보고 시스템이 자동으로 **표준명 등록 여부** 검증
4. 미등록 시 → 본 사전에서 검색 또는 **표준화 신청**

## 4가지 결과 케이스 ([[원료목록_사전보고_책임판매업자]] 인용)

| 케이스 | 처리 |
|--------|------|
| 수정사항 없음 | 제출 가능 |
| 단서조항 배합금지 성분 | 확인 후 제출 가능 |
| 순수 배합금지 성분 | 수정 필수, 제출 불가 |
| **표준명 미등록 성분** | **본 사전 검색 → 없으면 표준화 신청** |

## 표준명 표준화 신청 절차

| 단계 | 내용 |
|------|------|
| 신청 | KCA에 성분명 표준화 신청서 제출 |
| 심의 | **성분명표준화위원회 (월 2회 개최)** |
| 결과 | 등록 / 보완 / 반려 |
| 재신청 | **3개월 내 3회 무료** (보완·반려 시) |

→ 신제품 출시 일정에 영향. **신제품 처방 확정 단계(BM·ODM 협업)에서 미리 표준명 확인** 필수.

## LAKA 운영 룰 권고

[[품질관리_RR_2026-05]] #4 표시문안 검토 + #5 원료목록보고 운영에 본 사전을 다음과 같이 결합:

1. **ODM 처방서 수령 즉시** 성분명 일괄 본 사전 검색 (Pre-flight 체크)
2. 미등록 성분 발견 시 → BM/ODM에 즉시 회신·표준화 신청 일정 협의
3. 신제품 출시 일정에 **표준화 심의 1.5~3개월** 버퍼 반영

→ Vendor 일정 vs KCA 심의 일정 충돌이 잦은 경우 [[K뷰티_ODM_QA_실패_8유형]] Type 7(라벨링 부적합)으로 발현 가능.
