# 물류비 절감 레버 (2026 Q1~Q2)

SCM 부문 SC기획팀 물류파트의 2026 상반기 물류비 절감 정책 모음. 보관비 / 일본 행사 마일리지 / 출고리스트 전사화 / 수출 운송 거버넌스 4개 축.

## 1. [[HDL_에이치디엘코리아|HDL]] 김포 보관비 절감 (2026-03)

| 항목 | Before | After | 변동 |
|---|---:|---:|---|
| 일평균 보관량 | 2,750 PT | **2,500 PT** | **−250 PT** |
| 월 보관비 | — | — | **약 −400만원** |
| **연 환산 절감** | — | — | **약 −4,800만원** |

→ 보관(공간) 효율 개선 완료. 250 PT 감축이 곧 연 4,800만원 = **PT당 연 19.2만원** 비용 단가 환산 가능.

## 2. 일본 [[메가와리]] 행사 비용 처리 (2026-03 [[트랙스]] 행사)

| 항목 | 값 |
|---|---:|
| 처리 물량 | **80,131 건** |
| 건당 발생 비용 | 30엔 |
| 총 발생 비용 | **2,403,930엔** |
| 마일리지 | **약 2,200만원** |
| 처리 방식 | **4월 비용 차감 반영** |

### 시사점

- 일본 메가와리 같은 대형 일본 행사 = 건당 30엔 × 수만건 = 수천만원 단위 비용
- 마일리지 2,200만원은 트랙스(쿠팡 모회사 일본 자회사) 시스템의 후정산 모델
- "비용 절감을 위해 현장 운영에 대한 지속적인 관리·감독 강화 예정" — 현장 통제력 강화 방향

## 3. 출고리스트 전사화 (2026-01-15 전사 메일 배포)

| 항목 | 내용 |
|---|---|
| 시작 | 2026-02 2주차 시행 |
| 도구 | **Laka_출고 리스트.xlsx** (전사 공유) |
| 기능 | 출고 일정 + 일일 캐파 확인 / 실시간 물류 작업 상태(작업중/대기/출고완료) / **송장번호 & 출고일자 실시간 확인** |

### 효과

- 부서별 일정 예측 가능성 향상
- 송장번호 자동 업데이트로 별도 문의 없이 직접 배송 추적 가능

## 4. 기타출고 프로세스 통합 (2026-02-04 전사 메일 배포)

| 항목 | 내용 |
|---|---|
| 배경 | 산발적 소통 & 중복되는 본사 출고 건 최소화 → 물류비 절감 |
| 기대 효과 | 산발적 메신저 요청 → 통합 양식으로 누락 방지 + 이력 관리 / 송장 자동 추적 / 배송지 오류 최소화 / 부서별 일정 예측 / **본사 요청 영업 샘플 한 번에 발송** |

### 시사점

- 본사 발 영업 샘플이 산발 발송되면서 비용 누수 → 통합 발송으로 회당 운송비 절감
- 메신저 산발 요청을 양식화 = 디지털 거버넌스 표준화

## 5. 수출 운송 거버넌스 변경 (2026-04-06 대표 승인)

| 항목 | 내용 |
|---|---|
| 배경 | 무분별하게 사용되는 긴급 운송 줄여 손실 비용 최소화 |
| 신규 룰 | **페리/항공 사용 시 대표이사 사전 승인 필요** |
| 도구 | 기안 작성 예시 + 운임비용 정보 가이드 배포 |

### 시사점

- 페리/항공 = 컨테이너선 대비 단가 수 배. 무분별한 긴급 운송이 손실의 핵심
- **대표 결재 = 통제 의도**: 본부장/팀장이 아닌 대표 결재 = 강한 게이트 신호
- 신제품/긴급 채널 ([[퍼펙트트윈립_단상자혼입_2026Q1]] 무상 96,384 EA 같은) 긴급 무상 보전 시 운송 비용 폭증 가능

## 횡단 시사점

1. **물류비는 SCM 부서 직접 통제 가능 영역** — ODM 단가 / B2B 발주는 대표·영업 의존이지만 물류비는 SC기획팀 물류파트 자체 KPI로 절감 가능. 직접 ROI 가시화 좋은 영역
2. **연 4,800만원 = 작은 듯 큰 효과** — [[원가절감_케이스북_2026H1]] BOM 절감 케이스 1개당 1~6억 대비 작지만, 1회성이 아닌 **연속 절감 (recurring)**. 누적 효과 큼
3. **출고 디지털화 (Excel 전사 공유) = MVP** — Laka_출고 리스트.xlsx는 엑셀 기반 단순 도구. SCM AI/SaaS([[SCM_AI_솔루션_비교]]) 도입 전 단계의 MVP. 향후 [[K뷰티_수입유통_SCM_AI_도입_패턴]] 진입 후보
4. **대표 결재 게이트는 양날** — 게이트 효과 명확, 그러나 SCM 자율성 축소. 표준 운임 기준 만들어 본부장 결재로 위임 가능한 영역 식별 필요
5. **메가와리 마일리지 2,200만원 = 일본 채널 비용 구조 학습** — 트랙스/쿠팡 일본 채널의 정산 구조 위키화 가치 ([[Qoo10_일본]] 등)
