# 코스메틱 리테일 KPI — 인덱스

코스메틱 리테일은 **재구매 주기가 짧고, 트렌드 민감도가 높으며, SKU가 방대**한 특성상 일반 소매업과 다른 KPI 중요도 체계를 갖는다. 각 지표는 아래 개별 페이지에서 정의·공식·벤치마크·LAKA 적용 시사점을 다룬다.

---

## 재고 효율 지표

| KPI | 코스메틱 벤치마크 | 페이지 |
|---|---|---|
| Sell-Through Rate | 색조 50~60% / 스킨케어 40~55% (분기) | [[STR]] |
| 재고 회전율 | 연 4~6회 (우수: 6~8회) | [[재고_회전율]] |
| GMROI | 2.5~3.0 | [[GMROI]] |
| OSA (진열 가용성) | 95~98% | [[OSA]] |
| Shrinkage Rate | 3.0~3.5% | [[Shrinkage_Rate]] |

## 매출 효율 지표

| KPI | 코스메틱 벤치마크 | 페이지 |
|---|---|---|
| 전환율 | 온라인 1.9% / 오프라인 ~27% | [[전환율]] |
| ATV / AOV | \$50~\$150 | [[ATV]] |
| Basket Size (UPT) | 2~3개 | [[바스켓_사이즈]] |
| Foot Traffic | 채널·입지별 상이 | [[Foot_Traffic]] |
| 반품률 | 5~10% | [[반품률]] |

## 고객 가치 지표

| KPI | 코스메틱 벤치마크 | 페이지 |
|---|---|---|
| 재구매율 | 스킨케어 25~40% | [[재구매율]] |

| CAC | ~\$127 (뷰티 DTC 평균) | [[CAC]] |
