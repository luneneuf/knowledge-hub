# 퍼펙트트윈립 단상자 혼입 인시던트 (2026 Q1)

[[퍼펙트-트윈-립]] 일본 초도 출하분에서 **902호 제품에 905호·903호 단상자가 혼입**된 상태로 JT(일본)·큐텐 양 채널에서 동시 발견. ODM([[코스맥스]]) 생산공정의 비전검사기 우회 라인 + 재작업 SOP 부재가 결합된 전형적 **휴먼 에러 + 설비 우회** 사례.

## 4M 원인 분석 (코스맥스 회신 기준)

| 구분 | 주요 원인 | 개선 대책 |
|---|---|---|
| **설비 (Machine)** | 우회 라인이 존재하여 비전 검사를 건너뛸 수 있는 구조 | 불량품 전량 **시건 장치 투입**, 일반 작업자 임의 투입 차단 |
| **자재 (Material)** | 재작업품과 정상품 식별표 부재 / 스크래치 등 외부 불량품 혼합 대량 적재 / 단상자 혼입 원불량 | 공정 불량 유형별 시건 장치 / 단상자 공정 투입 시 묶음단위 홋수명 검사 |
| **방법 (Method)** | 공정 흐름 표준 부재 (재작업 시 프로세스 누락) / 재작업품 **2차 비전 검사 절차 미수립** | 재작업품은 반드시 비전 검사기 재통과 규정화 |
| **작업자 (Man)** | 재작업품을 검사기 후단에 직접 투입(우회) / 단상자 홋수명 확인 미흡 | 모든 재작업품은 비전 검사기 입구 투입 교육 / 라인장 지정 / 인박스 투입 전 외관검사 필수 |

### 비전 검사기 자체 한계

- Line별 비전 검사기 2대 운용 (바코드 / Lot 착인 확인)
- **단상자 글자 포인트가 작고 빛반사로 부적합 인식 다수 발생** → 부적합품 별도 확인 없이 공정 재투입되는 휴먼 에러로 연결
- 부적합품 재처리 시 비전 검사 미실시 + 육안 검사 미흡

## 라카 측 개선 요청 사항

1. 부적합품 시건 장치로 임의 투입 차단
2. 부적합품 재작업 시 비전 검사기 재통과 + 육안 검수 절차 마련
3. 재작업품 사용 검토 및 처리 담당자 지정 후 승인
4. 인박스 투입 전 홋수명 등 제품 외관 검사 필수
5. **2채널 비전 검사기 5대 도입** (검사 오류 개선 및 휴먼 에러 방지)
6. 현장 공정 검사 개선
7. **단상자 하짝에 홋수명 인쇄** — 혼입 발생 시 현장 검수 직관 대응 가능 (예: `#904 / #904`)

## 솔대 캡 분리 동반 이슈

같은 제품군에서 별도 부자재 불량도 동반 보고됨. 단상자 혼입과는 다른 근본원인이나 클레임 단일 묶음.

- **원인**: 중캡과 솔대 Hot Melting 접착면이 평면 형상 → 솔대 접착면 고정력 약함
- **개선**: 중캡 내측면 돌기와 솔대 접착면 돌기가 맞물리는 형상으로 금형 수정 → 고정력·접착력 강화 확인
- 부자재 Issue 금형 개선 완료

## 시사점

1. **신제품 초도 일본향에 대형 클레임** — 일본 시장은 검수 엄격, JT/큐텐 양 채널에서 거의 동시 발견되어 채널 확산 차단 필요
2. **ODM 비전검사기 5대 신규 도입 강제** — 라카가 검사 설비 사양까지 지정한 사례. ODM 설비 투자 협상력 행사 가능 케이스
3. **단상자 하짝 인쇄 = 비용 거의 0의 휴먼 에러 차단** — 단상자에 홋수 인쇄만 추가하면 현장 1초 검수로 혼입 식별. 全 제품 적용 검토 가치
4. **부자재 불량(솔대 캡) + 공정 불량(혼입) 동시 발생** — 초도 양산 안정화 단계가 부실. 신제품 [[신제품_QC_초도판정_프로토콜]] 첫 양산분 양산 안정성 확인 단계 추가 필요
5. **클레임 비용 정량화** — 무상 96,384 EA = **개당 원가 \$3,575 (절감 후) 기준 약 3.4억원 손실** (참고: [[원가절감_케이스북_2026H1]])
