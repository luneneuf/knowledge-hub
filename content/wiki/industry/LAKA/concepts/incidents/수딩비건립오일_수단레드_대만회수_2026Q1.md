# 수딩비건립오일 수단레드 대만 회수 (2026 Q1)

[[수딩-비건-립-오일]] 너리싱 옐로·카밍 퍼플 등에 사용된 **캄포사** 원료에서 **수단레드(Sudan Red) 검출 의혹** 발생. 2026-02 서울식약청 잠정판매중지 → 2026-02-10 불검출 확인 후 판매재개. 동시에 대만 유통사 Amiid를 통한 글로벌 자발 회수 진행. 총 회수 비용 **약 1.5~1.8억원** 발생.

## 사건 타임라인

| 일자 | 사건 |
|---|---|
| 2026-02-03 (화) | 서울식약청으로부터 **수단레드 잠정판매중단 공문 접수** + 검출시험용 시료 요청 → 식약청 전달 완료 |
| 2026-02 | 코티티(KOTITI) 외부시험성적서 대응 문의 결과: **식약처 자체시험 결과로만 판정** 예정 답변 접수 |
| 2026-02-10 (화) | 서울식약청 **불검출 확인 + 판매중단 해제 공문 접수** → 유관부서 배포 완료 |
| 2026-02~03 | 대만 Amiid 창고 회수 작업 진행 |
| 2026-03 | 1차 회수 수량 집계 (3,100 EA) + 2차 회수 예정량 집계 (7,268 EA) |
| 2026-04 (예상) | 최종 비용 인보이스 확정 + 송금 |

## 비용 구조

총 비용: **약 1.5~1.8억원 (한화)** — 하기 항목 모두 포함:

- 수단레드 테스트 비용
- 리테일러 반품비
- 리턴 핸들링비
- 소비자 환불

### 비용 송금 경로 (특이)

```
한국콜마 → 라카 → Amiid (4월 예상)
```

- **한국콜마에서 Amiid로 직송금 불가** — 양사 미거래 / 미계약 상태
- 2026-02-23 한국콜마 메일·유선 전달 완료, 3월 회신 예정
- 라카가 중간 송금 책임을 떠안는 구조

## 규제 행정 메커니즘 학습 포인트

1. **외부 시험성적서 무효** — KOTITI 같은 공인 시험기관 성적서를 제출해도 식약처는 자체 시험 결과만으로 판정. 잠정중단 해제까지 외부 시험 대응은 시간/비용 낭비
2. **잠정판매중단 → 검출시험 → 해제 사이클** — 약 1주 내 진행 (2/3 접수 → 2/10 해제)
3. **자발 회수의 글로벌 파급** — 한국 시장에서는 해제됐어도 대만 등 수출국에서 자발 회수 별도 진행. 회수 비용은 제조사(한국콜마) 귀책이지만 결제 라인업 정리가 별도 협상 필요

## 시사점

1. **원료 공급사 리스크 = ODM 리스크** — 캄포사 원료 이슈가 한국콜마 책임으로 전가됐지만, 라카는 책임판매업자로서 1차 대응 부담. ODM 5사 + 원료 공급사 리스크 스코어링 ([[공급업체_리스크_스코어링]]) 확대 필요
2. **수단레드 = K-뷰티 글로벌 단골 이슈** — 색소 원료 오염 사례 다수 (롬앤 등 과거 이슈). 사전 검출시험 정례화 가치
3. **회수 송금 라인 사전 합의 필요** — 한국콜마-Amiid 직거래 없으면 라카가 중간 흐름. 글로벌 유통사와 ODM 사이 자금 흐름 표준 계약 필요
4. **품질팀 회수 대응 SOP 부재 추정** — 회수 수량 집계·비용 청구·식약처 대응 흐름이 case-by-case. [[클레임_처리_규정]] 외 별도 회수 SOP 필요
5. **회수 비용 1.5~1.8억은 [[원가절감_케이스북_2026H1]] 수딩프렙립마스크 절감 3.67억의 절반 가까이** — 1건 사고로 절감 노력 상쇄
