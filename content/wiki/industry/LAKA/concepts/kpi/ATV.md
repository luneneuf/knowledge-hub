# ATV / AOV — Average Transaction Value / Average Order Value

## 정의

단일 거래에서 고객이 지출하는 평균 금액. 업셀링(upselling)·크로스셀링(cross-selling) 전략 및 프로모션 효과를 측정하는 데 활용.

> **공식**: ATV = 총 매출 ÷ 총 거래 건수

- **ATV** (Average Transaction Value): 오프라인 매장 맥락
- **AOV** (Average Order Value): 이커머스·온라인 맥락
- 두 용어는 실질적으로 동일 지표

## 벤치마크

- 코스메틱 스토어 일반 ATV: **\$50~\$150** (약 7만~22만 원)
- 인플루언서 커머스 기반 뷰티셀렉션(한국) 평균 객단가: 약 **10만 2,000원** (업계 평균 5만~7만 원 대비 상회)

## ATV 향상 전략

- 제품 번들링 및 루틴 세트 구성
- 특정 금액 이상 구매 시 사은품 증정
- 업셀링: 동일 카테고리 프리미엄 제품 제안
- 크로스셀링: 연관 카테고리 제품 추천 (립 → 블러셔)
