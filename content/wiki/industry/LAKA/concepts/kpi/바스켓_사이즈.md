# Basket Size / UPT — Units Per Transaction

## 정의

한 번의 거래에서 고객이 구매하는 **평균 품목 수**. [[ATV]]가 금액 기준이라면 Basket Size는 수량 기준. 코스메틱은 세럼+크림+선크림처럼 루틴(routine) 기반 구매가 많아 이 지표가 특히 중요하다.

> **공식**: Basket Size = 총 판매 수량 ÷ 총 거래 건수  
> 예시: 5,000개 / 2,000건 = **2.5**

## 코스메틱 특성

스킨케어는 단계별 루틴 제품 동시 구매가 빈번해 색조 대비 Basket Size가 높게 나타나는 경향. 히어로 SKU 프랜차이즈화(동일 라인의 복수 제품)가 Basket Size를 자연스럽게 끌어올린다.

→ [[히어로_SKU_프랜차이즈화]] 참고

## Basket Size 향상 수단

- 제품 번들링 ("스킨케어 루틴 3종 세트")
- 샘플 증정으로 신제품 체험 유도 후 재구매 전환
- 진열 시 연관 제품 인접 배치
- "이 제품과 함께 구매" 크로스셀링
