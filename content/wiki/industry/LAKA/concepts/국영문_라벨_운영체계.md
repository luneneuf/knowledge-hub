# 국/영문 라벨 운영체계 (4분류 A~D)

2026-01-15 [[이지철|이지철 대표]] 보고 완료된 **SKU별 라벨 운영 4분류 정책**. SC기획팀 S&OP파트 주관. 국가별 라벨 분리 운영의 작업비/리드타임/재고 부담을 줄이기 위해 SKU 운영 전략에 따라 4가지 라벨 모델로 표준화.

## 배경

### 국가 코드 운영 현황

| 구분 | 일본 | 한국 | 미국 | 유럽 | 기타 |
|---|---|---|---|---|---|
| 국가 코드 | 운영 | 운영 | 일부 운영 | 미운영 | 미운영 |
| 변경 전 | 각 국가별 [[화장품법_정기보고_캘린더|화장품법]] 맞춰 **개별 라벨 부착** | | | | |
| 변경 후 | **변경 없음** (일본 별도 운영 유지) | | | | |

→ 일본·한국은 국가코드 별도 운영, 미주는 일부, 유럽·기타는 미운영 상태. 국가별 라벨 분리 운영이 SKU 수 증가 + 작업비 누적의 원인.

## 4분류 정책

| 분류 | 적용 대상 | 용기 | 단상자 | 비고 |
|---|---|---|---|---|
| **A** | 기존제품 **0~20 SKU** | 국/영문 변경 | 국/영문 변경 | 개별 단상자 운영 |
| **B** | 기존제품 **50~100 SKU** (판매 부진) | **주주 작업** | **주주 작업** | 외주 작업. 예: [[모노-아이섀도우]] |
| **C** | 기존제품 **50~100 SKU** (전략 품목) | 영문 라벨 부착 | 영문 라벨 부착 | **공용 단상자 운영**. 예: [[프루티-글램-틴트]] 본품/미니 101~150 |
| **D** | **신제품** | 국/영문 기입 | 국/영문 기입 | 2~3천개 MOQ 커버 가능 시 → **국가별 코드 생성** |

## 효과 (국·영문 동시 기입할 경우)

| 측면 | 효과 |
|---|---|
| **수요** | 유연한 재고 할당 & 재고 부담 감소 |
| **구매** | 작업비 절감 & 생산 리드타임 단축 |

## "주주 작업" 용어

- **주주(株主) 작업** = 외주 라벨 부착 작업 (외주 라벨링 업체 위탁)
- B등급(판매 부진) SKU는 신규 인쇄 단상자/용기 발주 대신 기존 재고에 외주 라벨 추가 부착
- 단가 절감 + 재고 잔량 활용 동시 달성

## 시사점

1. **공용 단상자 운영 = 재고 할당 유연화의 결정적 레버** — [[LAKA_재고_시계열_2026H1]] 2026-04부터 등장한 **"국/영문" 공용 SKU 분류**가 본 정책의 결과물. 4월 2.79억 → 4월말 5.3억 급증
2. **A/B/C 분류는 SKU 수 + 판매 단계 동시 고려** — [[부진재고_등급제_운영]] S/A/B/C 등급과 별개 차원. 부진재고 등급 × 라벨 운영 분류 매트릭스 구성 가능
3. **D등급 MOQ 2~3천개 기준** — 신제품 첫 1년 발주가 2~3천개 이상 가능하면 국가별 코드 분리. 그 미만이면 공용 코드로 시작. **소량 글로벌 진입 전략**의 라벨 처리 기준
4. **"주주 작업" 외주 의존** — 본 보고에 외주 업체 명시 없음. [[원파워]]·[[세종기획]] 등 라벨 업체와 무관한 별도 라벨링 외주 가능성 — 후속 확인 필요
5. **유럽·기타국가 미운영 = 향후 확장 시 4분류 재설계 필요** — 폴란드(Gabona), 네덜란드(NOTINO·JCOR), 체코 등 유럽 진입 확대 시 EU 라벨 규정 반영 필수
