# 김포 물류센터 수입검사 프로세스

ODM 입고분에 대한 [[HDL_에이치디엘코리아|HDL 김포 물류센터]] 內 라카 품질팀 수입검사 표준 10단계. 2026-03-W4 SCM 주간보고에 처음 정리·공유.

## 10단계 흐름

| 단계 | 작업 | 핵심 체크 포인트 |
|---|---|---|
| **1** | 제품 물류센터 이동 | ODM → HDL 입고 처리 |
| **2** | 랜덤 샘플링 후 박스 정보 및 수량 확인 | 박스 라벨 / 발주 수량 일치 |
| **3** | 지정된 수입검사실 장소로 이동 | 검사 환경 분리 |
| **4** | 라벨 상태 확인 및 완제품 랜덤 샘플링 | 단상자 외관, 라벨 부착 상태 |
| **5** | 바코드 스캔 및 **GTIN 식별 검사** | 글로벌 표준 식별번호 일치 |
| **6** | 제품 구성 및 외관 확인 | 단상자 內 구성품 누락 여부 |
| **7** | 단상자 및 제품 착인 정보 확인 | 제조번호 / 제조일 / 사용기한 |
| **8** | 제품 성형 상태 및 용기 기능 검사 | 캡 결합·솔대·디스펜서 등 |
| **9** | 제품 발색 및 사용감 검사 | 컬러 일관성 (홋수 매칭) |
| **10** | 검사 종료 후 팔레트 재 적재 | 적합/부적합 구분 |

## 단계별 결합 인시던트 매핑

- **단계 4 (라벨 상태)** — [[국영문_라벨_운영체계]] 4단계 분류 (A~D) 별 라벨 부착 검수
- **단계 5 (GTIN 바코드)** — [[퍼펙트트윈립_단상자혼입_2026Q1]]에서 비전검사기가 단상자 작은 폰트·빛반사로 부적합 인식 다수 발생 → 인입 단계에서 동일 이슈 발생 가능
- **단계 6~7 (구성/착인)** — 단상자 혼입 검출 최후 보루
- **단계 8 (성형/기능)** — 퍼펙트트윈립 솔대 캡 분리 같은 부자재 불량 검출

## 인적 구성 / 운영

- 검사자: [[LAKA]] SCM 부문 품질팀
- 장소: HDL 김포 물류센터 內 수입검사실
- 회계실사 사례 (참고): 2026-01-05 [[HDL_에이치디엘코리아|HDL 김포]]에서 SC기획팀 3인 + 재무팀 박경준팀장 + 삼덕법인 회계사 2인 실사. **특이사항 없음 완료**

## 시사점

1. **GTIN 식별 검사가 5단계에 명시 = 라카가 글로벌 표준 도입했음** — 글로벌 B2B (세포라/부츠 등) 진입 시 필수 식별 체계
2. **10단계는 ODM 완성품 입고용** — 원료 검사·반제품 검사 SOP 별도 필요 가능성
3. **단상자 검사 단계가 4·6·7로 분산** — 단계별 책임자가 다르면 [[퍼펙트트윈립_단상자혼입_2026Q1]] 같은 휴먼 에러가 ODM에서 통과해도 HDL에서 잡힐 가능성. 단계별 검출률 측정 가치
4. **검사실 환경 통제 (3단계)** — 색상 검사(9단계)는 조명 영향 큼. 표준 조도/색온도 명세 필요
5. **검사 완료 후 적합/부적합 구분 적재만 명시** — 부적합 처리 단계는 별도 SOP 필요 (재작업 / 반품 / 폐기)
