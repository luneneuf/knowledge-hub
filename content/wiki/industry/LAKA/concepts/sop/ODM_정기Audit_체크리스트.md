# ODM 정기 Audit 체크리스트

라카 SCM 품질팀이 ODM 제조소에 방문하여 진행하는 **현장 평가** SOP. [[화성코스메틱]] 2026-03-19 평가 케이스를 표준 양식으로 정리.

| 방문일자 | YYYY.MM.DD (요일) |
| 평가 점수 | XX.X점 |

| 개선결과서 전달 요청기한 | 보통 방문일 + 2주 (예: 3/19 → 4/3) |

|---|---|

| 평가 점수 | **96.8점** |
| 등급 | **A등급** |
| 판정 | **거래가능** |

### 개선 요청 사항 (제조소 현장 운영 점검 시)

1. **원료 창고 적합 구역 內 적합 라벨 미부착 확인**
2. **원료 시험 대기 장소 外에 시험중 원료 보관**

> A등급 96.8점이어도 원료 보관 영역에서 2건 지적. 원료 트래킹 거버넌스가 ODM 공통 약점.

## 시사점

1. **A등급 96.8점에도 지적 2건 발생** — A등급이 무지적을 의미하지 않음. 등급 산정 룰 사내 명문화 필요
2. **원료 영역 약점은 ODM 공통 가능성** — 원료 적합 라벨/보관 영역은 [[K뷰티_ODM_QA_실패_8유형]] 中 빈번 패턴. 5사 모두 점검 필요
3. **개선결과서 2주 회수 기한 = 라카 표준** — 양식화 가치
4. **5사 Audit 기록 비대칭** — 본 보고에서 화성 1건만 노출. 나머지 4사 Audit 이력이 정형 기록 없는 게 더 큰 리스크. [[K뷰티_ODM_QA_실패_8유형]] 분석과 연계 검토
5. **점수→등급 산정 룰 미공개** — 96.8 = A의 경계가 어디인가. 사내 표준 수립 필요
