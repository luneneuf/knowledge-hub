# K뷰티 립 CS 벤치마크

> LAKA·rom&nd·BBIA 세포라 입점 K-뷰티 립 틴트 브랜드의 CS 클레임·대응 비교.  
> 기준: 세포라(미국), Amazon, Hwahae, gopicky.com 리뷰 데이터.

---

## 비교 대상 브랜드

| 브랜드 | 대표 제품 | 세포라 입점 | 가격대 |
|--------|----------|-----------|--------|
| **LAKA** | Fruity Glam Tint, Bonding Glow Lipstick | 미국 세포라 | \$20 내외 |
| **rom&nd** | Juicy Lasting Tint (JLT) | 미국 세포라 | \$12~14 |
| **BBIA** | Last Velvet Lip Tint, Glow Tint | Amazon, OLIVE YOUNG | \$10~15 |

---

## CS 클레임 비교표

| 브랜드 | 주요 CS 클레임 | 심각도 | 최근 이슈 |
|--------|-------------|--------|----------|
| **LAKA** | 지속력 부족, 도포 정밀도 | 낮음 | 특이 이슈 없음 |
| **rom&nd JLT** | 구순염(cheilitis), 입술 건조, 가려움 | **높음** | 2025년 5월 클레임 300배 급증 → Project ZERO 선언 |
| **BBIA Velvet** | 입술 건조, 각질 부각, 크레용 냄새 | 중간 | 지속적 건조 클레임 |
| **BBIA Glow** | 지속력, 입술 안쪽 발색 불균일 | 낮음~중간 | — |

## rom&nd Project ZERO — 2025년 사건 요약

### 사건 경과

```
소비자 CS 접수 (전화·이메일)
       ↓
세포라 리뷰 페이지 1성 리뷰 급증
       ↓
틱톡·인스타그램 바이럴 ("rom&nd 입술 다 탔다")
       ↓
세포라 바이어 브랜드사에 공식 문의
       ↓
rom&nd 공식 입장 발표 + Project ZERO 발표
       ↓
언론 보도 (K-beauty 전문 미디어)
```

### 원인 분석

| 항목 | 내용 |
|------|------|
| 발생 시기 | 2025년 5월 |
| 주요 증상 | 구순염, 입술 건조, 입술 트러블 |
| 클레임 규모 | 이전 대비 약 300배 급증 |
| 주요 의심 성분 | 빠른 증발 용제(알코올 계열), 염료 Red 202/201 |
| 3rd party 검사 | 염료 자체 자극성 미확인 |
| 브랜드 분석 | 빠른 증발 용제(flash-off)가 건조 원인 가능성 |

### rom&nd 대응 — 효과적 CS 대응 사례

- CS 급증 즉시 원인 분석 착수
- 제3자 검사 의뢰 → 투명하게 결과 공개
- 포뮬러 개선 프로젝트명(Project ZERO) 부여해 신뢰 회복 시도
- 소비자·채널 바이어에게 진행 상황 지속 커뮤니케이션

## LAKA 강점 분석

경쟁사 CS 분석을 바탕으로 한 LAKA의 상대적 강점:

| 강점 | 근거 |
|------|------|
| 낮은 건조 클레임 | 젤-워터 텍스처, 보습 중심 포뮬러 |
| 자연스러운 마무리 | "입술 결 부각 없음" 다수 언급 |
| 알레르기 클레임 없음 | rom&nd 2025년 사태 대비 우위 |
| 브랜드 이미지 안정성 | 주요 CS 이슈 없음 |

## K-beauty 공통 CS 리스크 포인트

| 리스크 | 설명 |
|--------|------|
| 민감성 소비자 증가 | 서구 시장 소비자의 알레르기·피부 민감성 인식이 한국보다 높음 |
| 염료 규제 차이 | 한국 허용 염료가 미국·EU에서 제한될 수 있음 |
| 성분 클레임 모니터링 | 출시 후 6개월 이내 리뷰 패턴 집중 관찰 필요 |
| CS 대응 속도 | 미국·유럽 소비자는 브랜드 공식 대응 기대, SNS 대응 속도 중요 |
| 바이어 관계 | CS 클레임 급증은 바이어와의 계약 재검토 트리거 |

## 경쟁사 리뷰 활용 방법 (LAKA 운영 관점)

1. 세포라 리뷰 모니터링: rom&nd·BBIA 클레임 패턴을 자사 제품과 비교
2. 성분 사전 검증: 경쟁사 클레임 유발 성분(염료, 용제)을 자사 포뮬러에서 검토
3. 바이어 소통 자료: 경쟁사 대비 LAKA CS 안정성 데이터 바이어에 제공
4. 마케팅 포인트: "건조함 없는 K-beauty 립 틴트" 차별화 메시지 활용
5. MoCRA 체계: SAE 보고 시스템 사전 구축으로 규제 대응 준비
