## 주요 관찰

1. **2025년 매출의 절반 가까이를 4개 거래처가 차지**: JSW(태국) + AMID(대만) + TNC(호주) + 크레도(유럽) = 3,606M / 6,362M ≈ 56.7%. 채널 집중 리스크.
2. **2026년 들어 상위 4개가 모두 급감**(JSW -83% / AMID -97% / TNC -93% / 크레도 -98%). 거래처 이슈인지, 재고 사이클 이슈인지, B2B → 직접유통 전환인지 확인 필요.
3. **신규 빅 라인 등장**: Sociolla(인니) 800·Sephora US 284·Far East(홍콩) 86·W Cosmetics(호주/네덜란드) 81. 합계 1,251M — '26년 B2B 매출 2,052M의 61%를 신규 라인이 메우는 구조.
4. **Skincupid (영국)의 '26년 122M**은 사실상 [[부츠]] 채널 공급 매출. 자체몰 라인(25 → 14)과 Boots 라인(123 → 122) 분리 기재.
5. **글로벌 플랫폼 그룹 약화**: [[실리콘투]] 271 → 0 / [[크레이버_UMMA]] 82 → 0 / 예스아시아 329 → 37. 직매입형 글로벌 디스트리뷰터 전반에서 '26 매출 빠짐.
6. **TNC 사명 변경 기재**: "56 통상 → 웨이포인트 글로벌". 사명 변경 이력 + 표상 호주 분류 확인 필요(웨이포인트 = 한국 회사일 가능성).
