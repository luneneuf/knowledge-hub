## 개요
중소기업의 성과관리 시스템 변화 트렌드를 정리한 일반 리서치. [[LAKA]]에 특화된 내용이 아닌, 업계 전반의 흐름과 도구를 조망한다. 이후 [[src_LAKA_성과관리_리서치_의뢰서|LAKA 특화 의뢰서]]와 [[src_LAKA_성과관리_케이스별_제도설계|케이스별 제도설계]]의 이론적 기반이 되었다.

## 핵심 내용

### 성과관리 5단계 변화
1. **MBO(목표관리)** — 전통적 연간 목표 설정 + 평가
2. **탈등급(No Rating)** — 등급 폐지, 정성 피드백 중심
3. **OKR** — 목표(Objective) + 핵심결과(Key Results) 기반
4. **상시 피드백** — 실시간 인정·코칭 체계
5. **AI 기반** — 편향 감지, 예측 분석

### MBO vs OKR 비교
| 항목 | MBO | OKR |
|------|-----|-----|
| 목표 설정 | 하향식 | 상향식 + 하향식 혼합 |
| 주기 | 연간 | 분기 |
| 공개 범위 | 관리자-직원 간 | 전사 공개 |
| 성과 연동 | 보상 직결 | 보상과 분리 권고 |
| 적합 조직 | 안정적·계층적 | 빠른 성장·변화 |

### 주요 사례
- **Adobe**: 2012년 연간 리뷰 폐지 → Check-in 도입 → 이직률 30% 감소

### 국내 HR SaaS 도구
| 도구 | 특징 |
|------|------|
| 레몬베이스 | OKR + 상시 피드백 + 1on1 |
| 플렉스 | 근태·급여 + 목표관리 통합 |
| INHR+ | 직무 역량 기반 평가 |
| 클랩(CLAP) | 실시간 피드백 + 인정 문화 |

### 200~500명 기업 3단계 로드맵
1. **기반 정비** (0~6개월): 직무 정의, 평가 기준 수립
2. **파일럿** (6~12개월): 일부 부서 시범 운영
3. **전사 확대** (12~18개월): 피드백 반영 후 전사 적용

### 핵심 메시지
> 제도 도입보다 **문화 전환이 먼저**다. 도구를 바꾸기 전에 "평가에 대한 인식"을 바꿔야 한다.

## 이직 관점 시사점
- LAKA처럼 공식 HR 체계가 없는 90명 규모 조직에 직접 적용 가능한 지식
- BSC/MBO 도입 경험을 OKR·상시피드백과 연결하는 스토리 구성 가능
