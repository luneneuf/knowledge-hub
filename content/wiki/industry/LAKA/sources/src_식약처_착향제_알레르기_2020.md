## 핵심 주장 한 줄

**향료에 들어있는 25종 알레르기 유발성분이 leave-on 0.001% / rinse-off 0.01% 초과로 함유 시, "향료"로만 표시 불가 — 해당 성분명 추가 기재 + 원료목록 보고에도 포함 의무.**

## 문서 구성 (총 13개 세부 항목)

### A. 표시·기재 관련 세부 지침 (8개)

① 0.01% / 0.001% 산출 방법 (성분량 ÷ 제품 내용량 × 100)
② 사용 후 씻어내는 / 씻어내지 않는 제품 구분
③ 표기 순서 — 1·2·3안 가능, **4안(`향료(리모넨, 리날룰)`)·5안(별표 표시) 불가**
④ "알레르기 유발성분임" 별도 표시 금지 / "사용 시의 주의사항"란 기재 대상 아님
⑤ 소용량(10mL 초과 50mL) 표시 면적 부족 시 생략 가능, 단 **홈페이지에 표시 의무**
⑥ 천연오일·식물 추출물 — 착향 목적·특성 시 알레르기 성분 표시 의무
⑦ **2019 부자재로 2020 제조한 화장품 → 사용기한까지 유통 가능 + 오버레이블링 권장**
⑧ **온라인 판매처 표시 의무 + 기존 부자재 유통품과 온라인 차이 시 안내 문구 권장**

### B. 원료목록 보고 관련 세부 지침 (3개)

① 표시한 알레르기 유발성분 → 원료목록 보고에 포함 의무
② 시행일 이전 유통 제품 표시 변경 시 → 원료목록 변경 보고 적절
③ **증빙자료** — 제조증명서·제품표준서 / 또는 제조사 신뢰성 자료(시험성적서·원료규격서)

### C. 참고1 — 법령 전문

- 화장품법 시행규칙 [별표4] 마목 (착향제 = "향료" 표시, 단 알레르기 성분 있을 시 명칭 기재)
- 부칙 〈총리령 제1516호, 2018.12.31.〉 §5조 1년 경과조치
- 알레르기 유발성분 25종 + CAS 번호 표 (별표2)

## 25종 성분 + CAS

[[향료_알레르기유발성분_표시#2. 알레르기 유발성분 25종 (CAS 번호 포함)|wiki 페이지에 통합 정리됨]]. 본 src 페이지에선 중복 게재 생략.

## LAKA 적용 핵심 (실무는 wiki 페이지 참조)

- LAKA SKU **95%+ leave-on** → 0.001% 엄격 기준 일률 적용
- 시트러스·라벤더·로즈·바닐라 콘셉트 SKU = 25종 중 8개 "상" 노출
- ODM([[한국콜마]]·코스맥스·화성·코스메카·씨앤씨) 처방서에 25종 함량 명시 의무
- 표시 4·5안 시안은 ODM에서 받으면 즉시 반려 룰 ([[품질관리_RR_2026-05]] #4)
- 비건/식물 추출 라인(수딩비건립오일·소울비건립밤·헬시세럼밤) — 지침 A⑥ 직접 적용
