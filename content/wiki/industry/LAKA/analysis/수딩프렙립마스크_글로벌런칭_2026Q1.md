# 수딩프렙립마스크 글로벌 런칭 케이스 (2026 Q1)

[[수딩-프렙-립-마스크]] **신제품 글로벌 동시 런칭** (한·미·일·영). 라카가 단일 SKU에 가장 큰 단가 절감(-34%) + 가장 빠른 발주 페이스 + 글로벌 4개국 동시 런칭을 베팅한 **2026 H1 최대 신제품 베팅 케이스**. 3월 판매 1·2위를 차지하며 외형상 성공으로 보이지만, 발주 vs 판매 정합도를 보면 **수요 "예측"이 아닌 영업 "약속" 기반 push 발주 + 채널 분산 대응** 패턴이 확인됨. [[부진재고_등급제_운영]] · [[신제품_재고운영_AsIs_ToBe]] 정책 도입의 **직접 트리거 케이스** 가설.

## 2. 수요 예측 방법론 — 영업 "약속" 기반 push 발주

### 핵심 슬라이드 문구 (1-W2)

> 수딩프렙립마스크 — 기존용기 사용 총 **10만개 1/29까지 입고 목표**
> (**틱톡샵 7만개 + 아마존 3만개**)
> 토출구 변경 용기 2월 중순부터 분납, **3월 초까지 총 20만개 입고 목표**

→ 형식상 "수요 예측"이 아닌 **채널별 약속 수량의 단순 합산**. 데이터 0건 시점 (런칭 D-30) 의사결정.

### 방법론 4단계 (역추적)

| 단계 | 방법 | 정량 단서 |
|---|---|---|
| 1. 채널 분할 | 영업/마케팅이 사전 채널별 약속 수량 제출 | "틱톡샵 7만 + 아마존 3만" 같이 채널 단위 분해 명시 |
| 2. 총량 합산 | 채널 약속의 단순 합산 | 미국 단독 50만+ EA (1월 단월 발주) |
| 3. 시점 분할 | 입고 일정으로 분할 | 1월말 10만 + 3월초 20만 누적 가시화 |
| 4. 마진 조정 | **신규 용기 추가 개발 + 사출금형 증작 검토** | "잘 팔릴 것" 정성적 확신 → 추가 베팅 |

### 클로드웹 S&OP 인수인계 자료의 Issue 1 패턴과 일치

| 외부 자료 Issue 1 | 본 케이스 |
|---|---|
| FCST가 수요 예측이 아닌 영업 목표로 작동 | 채널별 영업 약속의 합산 |
| 전사 달성률 24~29% (3개월 연속) | 미국 50만+ 발주 vs 3월 판매 5.4만 = **소진율 11%** |
| 80~100% 달성 SKU는 4~5% | 본 SKU는 외형상 1위지만 정량적으로는 미달 |

### ⚠️ 외부 자료의 "코드 오류" 분석 정정

## 4. 라카가 자체 인지한 위기 신호 4건

### 신호 1 — 영국 틱톡샵으로 재고 분산 시작 (2026-03-26)

3-W4 슬라이드 4 (재고금액 3/26 기준) "전주 대비 증감율 특이 건":
> **수딩프렙립마스크 → 틱톡샵 UK 재고 (2.8억)**

→ **미국 발주분의 일부를 영국 채널로 전환** 시작. 단일 국가 소진 어려움 자체 인식 신호.

### 신호 2 — 추가 발주 중단 (3월 매입 -2.5억)

4-W3 slide1:
> 한국콜마 — 수딩프렙립마스크 2억, **전월비 -2.5억**

→ 3월부터 추가 발주 거의 안 함. **1~2월 발주가 과다**임을 자체 인식.

### 신호 3 — 부진재고 등급제 대표 승인 (2026-02-23)

[[부진재고_등급제_운영]] (S/A/B/C 등급제 + 단종 정책)이 본 SKU 1~2월 발주 결과 데이터를 보고 한 결정일 가능성. 시점 정합 (1월 발주 완료 → 2월말 정책 수립).

### 신호 4 — 신제품 운영 As-is/To-be 전환 (3-W4)

[[신제품_재고운영_AsIs_ToBe]] As-is = "수요(영업니즈) 기반 일괄 입고" → To-be = "최소 수량 + 분할 입고". **As-is의 대표 케이스가 본 SKU**일 가능성 高. 정책 전환 모멘텀의 직접 트리거.

## 7. 횡단 시사점

### 1. "수요 예측" 정의 자체의 재정립 필요

- 라카는 신제품 발주를 **"영업·마케팅 약속 합산 → ODM 발주"** 패턴으로 운영
- 이는 데이터 기반 예측 (통계 베이스라인 + 영업 조정)이 아닌 **push 모델**
- 외부 자료 Issue 1 "FCST가 수요 예측이 아닌 영업 목표로 작동"의 정확한 메커니즘

### 2. 글로벌 동시 런칭 = CAPA·재고·물류 동시 부담

- 한·미·일·영 4개국 동시 런칭 = ODM CAPA 단기 집중 + 4개국 물류 + 4가지 라벨 운영
- [[국영문_라벨_운영체계]] D분류 정책 (신제품 국/영문 기입)의 운영 부담
- [[생산_CAPA_제약]] 부자재 [[코코스팩]] 튜브 카테고리 단일 의존 — 사고 시 4개국 동시 결품

### 3. 단가 절감 = push 발주의 합리화

- 본 SKU -34% / -3.67억 절감이 라카 內 단일 최대
- **마진 확보가 "이 정도면 안 팔려도 손실 적다"는 논리로 push 발주 정당화**
- 그러나 마진 확보 ≠ 시장 수요. 마진 X 수요량 모델 필요

### 4. 영업 약속 정확도 사후 검증 부재

- 1월 약속 "틱톡샵 7만 + 아마존 3만" vs 3월 실판매 5.4만 = **약속 정확도 54%**
- 사후 약속-실판매 갭 모니터링 없음 (외부 자료 Issue 1 "달성률 결과 피드백 부재"와 동일)
- 다음 신제품 발주 시 동일 패턴 반복 가능성 高

### 5. 채널 분산 = 위기 대응 첫 카드

- 본 SKU의 영국 틱톡샵 분산 (2.8억)이 [[신제품_재고운영_AsIs_ToBe]] "JT 미출고 재고 타채널 전환 (2.5억)" 정책의 선례 가능성
- 채널 분산은 미봉책 — 영국에서도 안 팔리면 결국 [[부진재고_등급제_운영]] 부진재고 TF로
