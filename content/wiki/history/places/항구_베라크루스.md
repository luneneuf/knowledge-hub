# 항구 — 베라크루스 (Veracruz)

## 기본 정보

| 항목 | 내용 |
|------|------|
| 위치 | 멕시코 베라크루스 주 동해안, 멕시코 만 |
| 설립 | 1519년 5월 18일, 에르난 코르테스 |
| 설립명 | Villa Rica de la Vera Cruz |
| 역할 | 신스페인(누에바 에스파냐) 최대 대서양 무역항 |

## 역사 개요

### 설립 배경

1518년 스페인 탐험가 후안 데 그리할바가 앞바다의 산 후안 데 울루아 섬을 처음 기록했다. 이듬해인 1519년 코르테스가 상설 항구·도시를 세웠다. 아즈텍 정복을 위한 군사·정치 전초기지로 출발한 것이 수세기 상업 허브로 진화했다는 점이 이 항구의 핵심 아이러니다.

### 식민지 시대 무역 구조

**수출 (아메리카 → 유럽)**
- 은(silver), 금, [[교역품_쇼콜라틀|카카오/초콜릿]], [[교역품_바닐라|바닐라]], 코치닐 염료, 고추
- 은과 코치닐이 신스페인 재정의 핵심 수입원

**수입 (유럽 → 아메리카)**
- 방직품, 금속제품, 와인, 올리브유, 종교 미술품, 이민자, 아프리카계 노예

**독점 구조**
- 16세기 중반~17세기: 신스페인 사실상 유일한 대서양 무역항
- 스페인 보물 함대(Indias Fleet) 정기 집결지 — 매년·격년으로 은·금·코치닐 선적
- 1560~1700년대: "스페인 제국 6대 항구" 중 하나로 평가됨

**콘술라도 설립**
- 1795년: 베라크루스 상인단이 공식 콘술라도(consulado, 상인 길드) 지위 획득
- 항구 상업 엘리트로서 도시 행정·세금·항만 정책에 실질적 영향력 행사
- 1778년 "자유무역령" 이후 물동량이 크게 증가

### 카카오·초콜릿의 유럽 진출 관문

베라크루스는 신대륙 카카오가 구대륙으로 건너간 물리적 게이트였다. 단, 흔히 회자되는 "1528년 코르테스가 카카오를 스페인 궁정에 헌상했다"는 서술은 직접 증거가 없는 후대 정리다.

**검증된 타임라인**

| 연도 | 사건 |
|------|------|
| 1544 | 문헌상 최초 — 도미니코회 수사들이 인솔한 과테말라 마야 귀족 사절단이 스페인 궁정에서 카카오 음료를 시연 |
| 1585 | 베라크루스 → 세비야 첫 공식 상업 선적 기록 |
| 1615 | 스페인 공주 안나가 프랑스 루이 13세와 결혼, 초콜릿 문화가 프랑스 궁정으로 전파 |
| 17세기 중엽 | 파리·런던에 초콜릿 하우스 등장, 유럽 상류층 음료로 정착 |

**"스페인 독점" 구조**
- 국가 기밀로 카카오 정보를 은폐한 것이 아니라 궁정·수도원 중심 소비가 약 1세기 가까이 이어진 결과
- 대량 무역으로 풀린 시점부터는 빠르게 유럽 전역으로 확산

**아즈텍의 카카오 소비**
- 베르날 디아스 델 카스티요 *Historia verdadera*: 황제 목테수마의 식탁에 거품이 이는 카카오 음료 항아리 50여 개가 들어오는 것을 목격했고 황제가 그중 일부를 마셨다고 기록 — "황제가 매일 50잔을 마셨다"는 통속적 표현은 이 장면이 변형된 것
- 카카오 콩이 동전 역할을 한 것은 메소아메리카 경제사 연구로 확립된 사실이나, "100알 = 노예 1명" 같은 환율은 2차 자료에서 반복되는 수치로 1차 문헌상 확정이 어렵고 지역·시기별 편차가 크다

### 방어 체계 — 산 후안 데 울루아 요새

바다 한가운데 섬 위에 요새를 올려 항구 입구를 통제하는 구조. 해적·사략선·영국·네덜란드·프랑스 함대의 반복적인 기습에 대응하기 위해 구축됐다.

### 침공의 역사

이 항구는 신스페인·멕시코의 생명선이었기에, 외세가 멕시코를 굴복시키려 할 때마다 여기를 먼저 쳤다.

| 연도 | 사건 |
|------|------|
| 1568 | **산 후안 데 울루아 해전.** 존 호킨스가 이끌고 사촌 [[프란시스_드레이크|프란시스 드레이크]]가 주디스호 함장으로 참전한 잉글랜드 선단이 폭풍 회피차 입항. 휴전 협정을 맺었으나 스페인이 기습해 호킨스 함대 5척 손실, 생존자 70~80명만 귀환. 드레이크가 평생 스페인 복수심을 품게 된 사건. 28년 뒤 두 사촌은 같은 1595~96 카리브 원정 중 몇 달 시차로 모두 사망 |
| 16~18세기 | 해적·영국·네덜란드·프랑스 반복 기습 |
| 1838 | **파티시에 전쟁(Pastry War).** 프랑스인 제과점 주인 르몽텔의 60만 페소 배상 요구를 멕시코가 거절하자 프랑스 함대가 베라크루스를 포격·점령. 산타 아나가 저항 중 다리 부상으로 절단 |
| 1847 | 미·멕시코 전쟁, 미국 함대 20일간 포위·포격 후 점령 → 멕시코시티 점령으로 이어짐 |
| 1861 | **3국 공동 점령.** 영국·프랑스·스페인 연합이 외채 불이행을 명목으로 진주. 영국·스페인 철수 후 프랑스는 막시밀리안 황제 꼭두각시 정권 수립으로 전환 |
| 1914 | **미국 해군·해병대 점령(약 7개월, 4월~11월).** 빌미는 탐피코 사건(미국 선원 체포)이었으나 실제 목적은 독일제 무기 선적 차단. 미군 22명·멕시코인 약 170명 사망. 11월 14일 반환 |
| 1924.02.12 | 멕시코 내정 불안으로 일시 임시 수도 지정 |

### 쇠퇴와 경쟁

- 18세기 후반~: 스페인 제국 제도 개편, 라틴아메리카 독립전쟁 여파 → 독점적 지위 약화
- 멕시코 독립 후: 만사니요·라사로 카르데나스(태평양)·탐피코·알타미라(멕시코만)와 경쟁 → "절대적 관문"에서 "주요 항구 중 하나"로

### 아프로-멕시코 문화의 형성

유럽–아프리카–아메리카 삼각무역의 관문이었기에, 아프리카계 노예가 대량 유입되며 아프로-메스티소-스페인 문화 혼종이 폭발적으로 일어났다. 소노 하로초(Veracruz 전통 음악), 해산물 요리, 라틴·아프리카 리듬이 섞인 베라크루스만의 문화가 형성됐다.

### 현대

**항만 현황 (2025년 기준)**
- 컨테이너 처리량: 약 150만 TEU
- 연간 처리능력: 9,500만 톤
- 방파제 길이: 4.5km
- 자동차 전용 터미널: 연간 최대 100만 대 수출입 가능
- 멕시코 상위 5대 항구 중 하나

**치안 문제**
- 2008~2012년: 로스 세타스 등 마약 카르텔 격전지
- 항만 통제권을 둘러싼 갱단 전쟁으로 납치·실종·살해 빈발
- 500년 역사의 국가 무역 관문이 마약 밀수 루트 거점이 된 비극적 반전

| 흑요석 곤봉 | 무기·특산품 | 아즈텍 마쿠아우이틀(macuahuitl). 베르날 디아스 증언상 말의 목을 거의 잘라낼 정도의 위력이었고 관절·틈새·가죽 보호구에 특히 효과적. "스페인 갑옷을 통째로 잘랐다"식 서술은 과장. 정복 이후 전장에서 빠르게 사라지고 의례·수집품으로만 잔존 |

### 미확인 사항

게임 내 역할 상세(시나리오 시작점 여부, 관련 발견물·퀘스트, 대표 항해사, 시대개방 트리거 등)는 미확인. 추후 업데이트 필요.

## 역사 vs 게임 연계 앵글

- 베라크루스(게임 항구) → 카디스 → 유럽 식탁으로 이어지는 루트가 [[교역품_바닐라|바닐라]]·[[교역품_쇼콜라틀|카카오]] 역사 루트와 정확히 일치
- "스페인 보물 함대 집결지"라는 역사적 배경이 게임 내 은 호송선단 루트와 연결 가능
- 산 후안 데 울루아 요새 → 게임 내 해안 방어 시설로 등장할 가능성
