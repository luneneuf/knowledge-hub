# I. 가이드라인 개요

## 1.1 본 가이드라인의 목적 및 적용 범위

### 📋 목적

본 가이드라인은 「인공지능 발전과 신뢰 기반 조성 등에 관한 기본법」(이하 "인공지능기본법")의 시행(2026년 1월 22일)에 따라, 우리 회사가 게임 개발·퍼블리싱·운영 과정에서 준수해야 할 법적 의무를 명확히 하고, 각 부서의 실무 담당자가 즉시 활용할 수 있는 구체적인 대응 방안을 제시하기 위해 마련되었습니다.

### 🎯 핵심 원칙

1. **법률이 요구하는 것만 정확히 준수** - 과잉 준수로 인한 불필요한 비용 방지
2. **실무 중심의 실행 가능한 가이드** - 추상적 원칙이 아닌 구체적 행동 지침
3. **부서별 맞춤형 접근** - 개발/운영/CS 각 부서의 업무 특성 반영
4. **지속적 업데이트** - 대통령령 및 하위 규정 제정 시 즉시 반영

### 📁 적용 범위

**✅ 본 가이드라인이 다루는 범위**
- 게임 내 AI 기능 개발 및 적용 (NPC, 콘텐츠 생성, 추천 시스템 등)
- 게임 운영 도구 및 CS 도구의 AI 활용
- 외부 AI 솔루션 도입 및 계약 검토
- 이용자 대상 고지 및 표시 의무 이행
- 글로벌 서비스 및 해외 파트너사 관리

**❌ 본 가이드라인이 다루지 않는 범위**
- **HR/채용 시스템의 AI 활용** → 별도 가이드라인 참조 (인사팀 관할)
- 일반적인 개인정보보호법 준수 사항 → 기존 개인정보보호 가이드라인 참조
- AI 기술 개발 방법론 → 기술 문서 참조

### 👥 대상 부서 및 담당자

| 부서 | 주요 검토 섹션 | 우선순위 |

# II. 핵심 법적 의무 요약

## 2.1 즉시 준수 사항 (2026.1.22 시행일 기준)

### ⭐ 제31조: 투명성 확보 의무 (최우선 준수 대상)

이 조항은 게임사에 **가장 직접적이고 즉각적인 영향**을 미치는 핵심 의무입니다.

---

#### 📢 의무 1: 생성형 AI 사전 고지 (제31조 제1항)

**법조문**
> "인공지능사업자는 고영향 인공지능이나 생성형 인공지능을 이용한 제품 또는 서비스를 제공하려는 경우 제품 또는 서비스가 해당 인공지능에 기반하여 운용된다는 사실을 이용자에게 사전에 고지하여야 한다."

**핵심 요건**

| 구분 | 내용 |
|------|------|
| 대상 AI | 생성형 인공지능 또는 고영향 인공지능 |
| 고지 대상 | 이용자 (게임 유저) |
| 고지 시점 | 사전 (서비스 제공 전) |
| 고지 내용 | 해당 AI에 기반하여 운용된다는 사실 |
| 위반 시 제재 | 과태료 최대 3천만원 |

**게임사 적용 사례**

**✅ 고지가 필요한 경우**
- AI 챗봇 NPC와 대화하는 기능을 제공하는 경우
- AI가 실시간으로 퀘스트 텍스트를 생성하는 경우
- AI가 캐릭터 이미지를 생성하여 제공하는 경우
- AI 음성 합성으로 NPC 대사를 제공하는 경우
- AI 기반 콘텐츠 추천 시스템을 운영하는 경우 (생성형 AI 사용 시)

**❌ 고지가 불필요한 경우**
- 개발 과정에서만 AI를 사용하고 최종 결과물을 고정된 형태로 제공하는 경우
- 내부 운영 도구로만 AI를 사용하고 이용자는 AI와 직접 상호작용하지 않는 경우
- 전통적인 게임 AI (룰 기반, 유한상태기계 등 비생성형)

**실무 고지 방법**

| 고지 위치 | 적용 시점 | 예시 문구 |
|----------|----------|----------|
| 게임 온보딩 | 최초 실행 시 | "이 게임은 일부 기능에서 AI 기술을 사용합니다" |
| 기능 진입 시 | AI 기능 최초 사용 전 | "이 NPC는 AI 기술로 대화를 생성합니다" |
| 게임 설정 메뉴 | 상시 확인 가능 | "AI 사용 안내" 메뉴 항목 |
| 이용약관 | 회원가입 시 | "AI 기반 서비스 제공 고지" 조항 |

#### 🏷️ 의무 2: 생성형 AI 결과물 표시 (제31조 제2항)

**법조문**
> "인공지능사업자는 생성형 인공지능 또는 이를 이용한 제품 또는 서비스를 제공하는 경우 그 결과물이 생성형 인공지능에 의하여 생성되었다는 사실을 표시하여야 한다."

**핵심 요건**

| 구분 | 내용 |
|------|------|
| 대상 | 생성형 AI로 만든 결과물 |
| 표시 대상 | 결과물 자체 |
| 표시 내용 | AI에 의해 생성되었다는 사실 |
| 표시 방법 | 대통령령으로 정함 (현재 미확정) |
| 위반 시 제재 | 제40조 시정명령 대상 (불이행 시 과태료 3천만원) |

**게임사 적용 사례**

**콘텐츠 유형별 표시 방법**

| 결과물 유형 | 표시 방법 예시 | 우선순위 |
|------------|---------------|---------|
| 텍스트 | "AI 생성" 라벨<br>메타데이터 표시<br>폰트 색상 구분 | ⭐⭐⭐ |
| 이미지 | 워터마크<br>코너 아이콘<br>이미지 설명란 표기 | ⭐⭐⭐ |
| 음성 | 재생 전 안내 메시지<br>자막에 표시<br>음성 플레이어 UI 표시 | ⭐⭐ |
| 영상 | 영상 시작/종료 시 표시<br>상단/하단 바 표시<br>비디오 메타데이터 | ⭐⭐ |

**표시 문구 예시**

**간결형 (권장)**
- "AI 생성"
- "AI Generated"
- 🤖 아이콘

#### 🎭 의무 3: 딥페이크급 콘텐츠 명확 고지 (제31조 제3항)

| 포토리얼리스틱 배경 | 중간 ⚠️⚠️ | 실제 장소 재현 시 주의 |
| 실사 광고 영상 | 높음 ⚠️⚠️⚠️ | 마케팅 소재는 본 법 외 광고법 검토 |

| 추상적/판타지 콘텐츠 | 낮음 ✅ | 제3항 미적용 가능성 높음 |

## 2.2 조건부 적용 사항 (대상 해당 시)

### 🔒 제32조: 대규모 AI 안전성 확보 의무

**법조문**
> "인공지능사업자는 학습에 사용된 누적 연산량이 대통령령으로 정하는 기준 이상인 인공지능시스템의 안전성을 확보하기 위하여 다음 각 호의 사항을 이행하여야 한다."

**적용 대상 판단**

| 항목 | 내용 |
|------|------|
| 기준 | 학습 누적 연산량이 대통령령 기준 이상 |
| 현황 | 대통령령 미제정 (기준 미확정) |
| 예상 기준 | 10^25 FLOPs 이상 (EU AI Act 참조 시) |
| 게임사 해당성 | 대부분 해당 안 됨 (외부 API 이용 시) |

**대부분의 게임사는 해당 안 되는 이유**

게임사는 일반적으로 다음과 같이 AI를 활용합니다:

**외부 API 이용 (대부분)**
- OpenAI GPT API 사용
- Google Gemini API 사용
- Stability AI API 사용

→ 이 경우 **API 제공자(OpenAI, Google 등)**가 제32조의 1차 의무자이며, 게임사는 "이용사업자"로서 직접 의무 없음

**자체 소규모 모델 (일부)**
- 캐릭터 행동 예측 모델
- 게임 밸런싱 모델
- 추천 알고리즘

→ 이러한 모델은 연산량이 대통령령 기준에 미달할 가능성 높음

**⚠️ 해당되는 예외적 경우**

다음의 경우 제32조 적용 가능성이 있습니다:

- 자체 대규모 언어 모델(LLM)을 처음부터 학습하는 경우
- 계열사 또는 협력사와 공동으로 파운데이션 모델을 개발하는 경우
- 대규모 멀티모달 AI 시스템을 자체 구축하는 경우

**의무 내용 (해당 시)**

1. AI 수명주기 전반에 걸친 위험 식별·평가·완화
2. 안전사고 모니터링 및 대응 위험관리체계 구축
3. 이행 결과를 과기정통부장관에게 제출

**게임사 대응 방안**

| 상황 | 대응 |
|------|------|
| 외부 API만 사용 | 계약서에 제32조 준수 보증 조항 포함 |
| 자체 모델 개발 | 연산량 추정 후 기준 초과 시 위험관리체계 구축 |
| 미확정 상태 | 대통령령 제정 모니터링 (2025년 중 예상) |

### 🌏 제36조: 국내대리인 지정 의무 (해외 사업자)

**법조문**
> "국내에 주소 또는 영업소가 없는 인공지능사업자로서 이용자 수, 매출액 등이 대통령령으로 정하는 기준에 해당하는 자는 국내대리인을 서면으로 지정하고, 이를 과학기술정보통신부장관에게 신고하여야 한다."

**적용 대상 판단**

| 요건 | 내용 |
|------|------|
| 1. 국내 사업장 없음 | 국내에 주소 또는 영업소가 없는 사업자 |
| 2. 규모 기준 충족 | 이용자 수, 매출액 등이 대통령령 기준 이상 |
| 현황 | 대통령령 미제정 (기준 미확정) |
| 위반 시 제재 | 과태료 최대 3천만원 |

**게임사 관련 시나리오**

| 상황 | 국내대리인 필요 여부 | 비고 |
|------|-------------------|------|
| 한국 법인이 서비스 | ❌ 불필요 | 국내 사업장 있음 |
| 해외 본사가 직접 서비스 | ⚠️ 규모 따라 필요 | 대통령령 기준 확인 필요 |
| 해외 개발사와 퍼블리싱 계약 | ⚠️ 경우에 따라 필요 | 서비스 주체 확인 필요 |
| 해외 AI API 사용 | ❌ 대부분 불필요 | API 제공자가 지정 |

**퍼블리싱 계약 시 주의사항**

해외 개발사가 AI 기능을 포함한 게임을 한국에 퍼블리싱하는 경우:

**책임 소재 확인**
- 누가 "인공지능사업자"인가? (개발사 vs 퍼블리셔)
- AI 기능의 운영 주체는 누구인가?
- 한국 이용자 데이터는 누가 처리하는가?

**계약서 검토 사항**
- 국내대리인 지정 의무 이행 책임자 명시
- 비용 분담
- 법령 위반 시 손해배상 조항

**대응 방안**

| 단계 | 조치 사항 | 시기 |
|------|----------|------|
| 1단계 | 대통령령 제정 모니터링 | 2025년 중 |
| 2단계 | 우리 서비스 규모 확인 (이용자 수, 매출액) | 대통령령 제정 후 |
| 3단계 | 기준 충족 시 국내대리인 지정 준비 | 시행일 3개월 전 |
| 4단계 | 과기정통부 신고 | 시행일 전 |

**국내대리인 후보**
- 한국 지사 (있는 경우)
- 법무법인 (외부 위탁)
- 퍼블리싱 파트너 (계약 조건에 따라)

## 2.3 하지 않아도 되는 것 (과잉 준수 방지)

### ❌ 과도한 해석 사례와 올바른 이해

많은 기업들이 법률을 지나치게 넓게 해석하여 불필요한 비용과 업무 부담을 발생시키는 경우가 있습니다. 다음은 **하지 않아도 되는 것**들입니다.

---

### 1. 마케팅 소재에 AI 표시 (본 법 적용 대상 아님)

**잘못된 해석**
- "홍보 영상을 AI로 만들었으니 영상에 'AI 생성' 표시를 해야 한다"
- "광고 이미지를 Midjourney로 만들었으니 워터마크를 넣어야 한다"

**올바른 이해**

| 항목 | 설명 |
|------|------|
| 법적 근거 | 제31조는 "이용자에게 제공하는 서비스"가 대상 |
| 마케팅 소재 | 사업자의 광고 수단이지 이용자에게 제공하는 서비스 아님 |
| 적용 법률 | 표시·광고의 공정화에 관한 법률 등 별도 법령 검토 필요 |

**실무 권고**
- 인공지능기본법상 의무는 아님
- 다만, 광고 윤리 차원에서 자율적 표시 검토 가능
- 소비자 오인 방지 목적의 자율 규제

---

### 2. 내부 운영 도구만 사용하는 경우 이용자 고지

**잘못된 해석**
- "GM이 CS 답변 작성에 ChatGPT를 보조 도구로 쓰니까 유저에게 고지해야 한다"
- "개발팀이 버그 분석에 AI를 쓰니까 공지해야 한다"

**올바른 이해**

| 항목 | 설명 |
|------|------|
| 고지 대상 | 이용자가 "AI 기반 서비스"를 제공받는 경우 |
| 백오피스 도구 | 내부 업무 효율화 도구는 이용자 대상 서비스 아님 |
| 판단 기준 | 이용자가 AI와 직접 상호작용하거나 AI 결과물을 받는가? |

**구분 기준**

| 상황 | 고지 필요 | 이유 |
|------|----------|------|
| GM이 답변 초안을 AI로 작성 → 검토 후 발송 | ❌ 불필요 | 최종 책임은 사람(GM) |
| CS 챗봇이 이용자에게 직접 답변 | ✅ 필요 | 이용자가 AI와 상호작용 |
| 개발팀이 버그 분석에 AI 사용 | ❌ 불필요 | 내부 도구 사용 |
| AI가 게임 내 추천 콘텐츠 생성 | ✅ 필요 | 이용자가 AI 결과물 수령 |

---

### 3. 일반 게임 AI에 "고영향 AI" 의무 적용

**잘못된 해석**
- "게임 추천 알고리즘이 유저 권리에 영향을 주니 고영향 AI다"
- "계정 제재 판단에 AI를 쓰니 고영향 AI 의무를 지켜야 한다"
- "안티치트 시스템이 생체정보를 쓰니 고영향 AI다"

**올바른 이해**

**고영향 AI 법정 영역 (제2조 제4호)**

게임 산업과 직접 관련 없는 영역들입니다:

- 에너지 공급
- 먹는물 생산
- 보건의료 제공
- 의료기기 개발
- 원자력 시설 관리
- **범죄 수사·체포 업무**를 위한 생체정보 분석 (← 게임 안티치트 해당 안 됨)
- **채용, 대출 심사 등** 권리·의무 중대 영향 (← 게임 플레이 해당 안 됨)
- 교통수단 운영
- 공공서비스 제공 결정
- 학생 평가

**게임 관련 오해 해소**

| 오해 | 정정 |
|------|------|
| 게임 추천 알고리즘은 고영향 AI | ❌ 법정 영역에 해당 안 됨 |
| 계정 제재는 권리·의무 중대 영향 | ❌ 채용·대출과 같은 법률관계 아님 |
| 안티치트는 생체정보 분석 영역 | ❌ "범죄 수사·체포 업무" 아님 |

**⚠️ 주의: HR 채용은 예외**

사내 인사·채용 시스템에서 AI를 사용하는 경우는 **명백히 고영향 AI**에 해당합니다. (본 가이드라인 제외 대상)

---

### 4. 모든 AI 기능에 영향평가 실시

**잘못된 해석**
- "AI를 쓰는 모든 기능마다 영향평가를 해야 한다"

**올바른 이해**

| 항목 | 설명 |
|------|------|
| 영향평가 대상 | 고영향 AI만 해당 (제35조) |
| 의무 강도 | "노력 의무" (강행 의무 아님) |
| 게임 일반 기능 | 대부분 영향평가 대상 아님 |

**영향평가 필요 여부**

| AI 기능 | 고영향 해당 | 영향평가 필요 |
|---------|-----------|-------------|
| NPC 대화 생성 | ❌ | ❌ |
| 이미지 생성 | ❌ | ❌ |
| 콘텐츠 추천 | ❌ | ❌ |
| 사내 채용 시스템 | ✅ | ✅ (권고) |

---

### 5. 모든 외부 AI 사용에 대해 국내대리인 요구

**잘못된 해석**
- "OpenAI API를 쓰니까 OpenAI가 국내대리인을 지정해야 한다고 요구해야 한다"

**올바른 이해**

| 항목 | 설명 |
|------|------|
| 국내대리인 대상 | 이용자 수·매출액이 대통령령 기준 이상인 사업자만 |
| 현황 | 기준 미확정 |
| 게임사 역할 | API 제공자의 국내대리인 지정 여부 확인은 권고 사항이지 의무 아님 |

**실무 권고**
- 대형 AI 플랫폼(OpenAI, Google 등)은 자체적으로 한국 법 준수 예상
- 계약 시 "관련 법령 준수" 일반 조항으로 충분
- 중소 AI 벤더의 경우 계약 검토 시 확인 권고

---

### 📌 과잉 준수 방지 원칙

**법률이 요구하는 것**
1. 생성형 AI 사용 시 이용자에게 고지 및 결과물 표시
2. 대규모 AI 개발 시 안전성 확보 (대부분 해당 안 됨)
3. 해외 사업자는 일정 규모 이상 시 국내대리인 지정 (기준 미확정)

**법률이 요구하지 않는 것**
1. 마케팅 소재 AI 표시
2. 내부 도구 사용 시 이용자 고지
3. 일반 게임 AI에 고영향 AI 의무
4. 모든 기능에 영향평가
5. 모든 외부 벤더에게 국내대리인 요구
